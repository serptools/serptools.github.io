What are OML files and how to open them

Are you having problems opening an OML file or are you simply curious about its contents? We're here to explain the properties of these files and provide you with software that can open or handle your OML files.

What is an OML file?

An .OML file is an OutSystems Modeling Language file.

We have not yet analyzed in detail what these files contain and what they are used for. We're working on it.

How to open OML files

Important: Different programs may use files with the OML file extension for different purposes, so unless you are sure which format your OML file is, you may need to try a few different programs.
