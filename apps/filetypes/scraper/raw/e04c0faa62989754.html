What is an AQ file?

An AQ file is a database file created by Incline Software Ancestral Quest, a family tree genealogy application. It contains genealogical data that outlines a family tree, which includes the names, birthplaces, and descriptions of family members and their relationships with each other.

More Information

Ancestral Quest allows you to record, organize, and visualize your family tree. When you save your information in the software by selecting File → New..., Ancestral Quest creates an AQ file, by default, to store the information. You can save, close, and re-open the AQ file to further edit your family tree.

You can also back up your AQ database as an .AQZ file to save space by selecting File → Backup.... Or you can export the information in an AQ database to the .GED genealogical database format by selecting File → Export....

How to open an AQ file

You can only open AQ files with Ancestral Quest. To open an AQ file with the software, select File → Open....
