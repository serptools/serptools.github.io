What are MCF files and how to open them

Are you having problems opening a MCF file or are you simply curious about its contents? We're here to explain the properties of these files and provide you with software that can open or handle your MCF files.

What is a MCF file?

MCF files have multiple uses, and MasterCook Cookbook is one of them. Read more about the other uses further down the page.

MasterCook Cookbook File

The .mcf file extension is most commonly used by the MasterCook recipe software programs. Versions 2 and later of this software use the .mcf file extension. The MCF files store the software's recipes, with each MCF file containing information about the recipe ingredients and cooking instructions.
