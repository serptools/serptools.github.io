Microsoft Excel Spreadsheet Template

What is an XLTX file?

An XLTX file is a template created by Microsoft Excel, a spreadsheet application included in the Microsoft Office suite. It contains predefined settings and layout information used to create multiple Excel spreadsheets (typically .XLSX files) with similar formatting and properties.

More Information

Excel is the most widely-used spreadsheet application in the world. It comes with every edition of the Microsoft Office suite and is designed for home, academic, and business environments.

The software includes many pre-created XLTX files that allow you to get a head start on creating spreadsheets, such as budget planners, calendars, and invoices. You can also create custom templates from your spreadsheets. To save a spreadsheet as an XLTX file with Microsoft Excel, select File →Save As, then choose "Excel Template" from the format dropdown menu.

NOTE: XLTX files are saved in the Open XML file format Microsoft introduced in Microsoft Office 2007. The template type replaced .XLT files, which are stored in a binary format.

How to open an XLTX file

You can open XLTX files in Microsoft Excel (Windows, Mac, iOS, Android). To do so, select File → Open from the program's menu bar. After you do, you can either edit the template the XLTX file contains, or you can use the XLTX file to create a new spreadsheet.

Note that, to open XLTX files in Microsoft Excel 2003 and earlier versions, you must install the Microsoft Office Compatibility Pack. If you have not installed the compatibility pack, you will not be able to open your XLTX file in Excel 2003 and earlier versions.
