LcdStudio Playlist File

What is an XPL file?

File used by LcdStudio, a program that allows users to create programs that render computer performance data on an entire monitor screen; contains a list of .XDS design files to be played in an LcdStudio application; used to display dynamic visualizations from live computer performance data.

Little Snitch Configuration File

Developer Objective Development
Popularity
3.5  |  4 Votes
 

An XPL file may also be a configuration file used by Little Snitch, a macOS tool used to track incoming and outgoing data transmissions by applications installed on a computer. It contains rules, profiles, and preferences that dictate the connections allowed between installed applications and the Internet. XPL files are primarily used to back up a collection of rules that can be restored in Little Snitch at a later time.

More Information

You will most likely only encounter an XPL configuration file if you use Little Snitch in macOS to monitor your applications' Internet connections. To create an XPL file with Little Snitch Configuration, select File → Create Backup.... To add a new rule to a configuration, select File → New Rule..., then save the XPL configuration file again.

XPL files are saved in a proprietary format and can only be opened with Little Snitch. To open an XPL file with Little Snitch Configuration follow these steps:

  1. Select File → Restore from Backup...
  2. Click Select Backup File...
  3. Choose the XPL file to restore and click Next
  4. Click Import...
