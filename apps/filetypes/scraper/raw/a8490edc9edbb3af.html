Windows Boot Configuration Data File

Developer Microsoft
Popularity
3.3  |  17 Votes
 

What is a BCD file?

Windows boot configuration format introduced with Windows Vista; replaces the older boot.ini format used by the NTLDR program; supports underlying firmware abstraction so that multiple systems can be supported.

More Information

BCD initially supported PC/AT BIOS and EFI systems but is extensible and can support other firmwares as well.

NOTE: BCD files use the Windows registry (.REG) format and therefore can be edited with the Windows Registry Editor.

RealView Debugger Board Chip Definition File

Developer ARM
Popularity
3.2  |  10 Votes
 

File used by RealView Debugger (RVD), an ARM processor debugging application included with ARM Development Studio; describes the registers and memory map for a target integrated circuit; used during the design's testing and debugging process; saved in a plain text format.

More Information

NOTE: ARM processors are commonly used in mobile devices and for creating SoC (System on a chip) designs.

Super Mario Maker 2 Course Data

Developer Nintendo
Popularity
1.3  |  3 Votes
 

A BCD file may also be part of a game level created by Super Mario Maker 2, a Super Mario level creator available for the Nintendo Switch. It contains binary data specifying the level's layout and some associated metadata. BCD files are stored with accompanying DAT and BTL files, which contain other data associated with the level.

More Information

Super Mario Maker 2 is the sequel to the original Super Mario Maker. Just like in Super Mario Maker, Super Mario Maker 2 players can create and play their own Super Mario levels. Players can also publish their levels online so others can play them.

Unlike Super Mario Maker levels (.CDT files), Super Mario Maker 2 levels are stored as a package of three files. The level's BCD file contains most of its data, such as the placement of terrain, objects, and enemies. Super Mario Maker 2 players and modders use BCD files, and their associated DAT and BTL files, to examine and modify Super Mario Maker 2 levels, and distribute levels offline.

Common BCD Filenames

course_data_###.bcd - The name assigned to Super Mario Maker 2 course data files.

How to open a BCD file

You can open a BCD file with TheGreatRambler's online Super Mario Maker 2 Level Viewer, to see the layout of the level the file contains. You can also use SMM2 Level Viewer (Windows) to view a Super Mario Maker 2 level. However, SMM2 Level Viewer requires users to submit a level's code to view that level, rather than open the level's BCD file.
