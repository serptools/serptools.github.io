Scream Tracker 2 Module

Developer Future Crew
Popularity
3.8  |  15 Votes
 

What is an STM file?

An STM file is an audio tracking module file saved in the Scream Tracker Module (STM) format, a proprietary format used by versions 1 and 2 of Scream Tracker. It contains tracks, patterns, and samples used for instruments played back in the songs.

More Information

Future Crew developed the first version of Scream Tracker in the late-1980s to modify the popular .MOD Amiga Module format introduced in 1987. Future Crew then released version 2 of Scream Tracker in 1990.

The STM format is the default format that Scream Tracker 1 and 2 use to save music. However, future Crew eventually replaced the STM format with the Scream Tracker 3 Module (S3M) format when it released Scream Tracker 3 in 1994.

Even though Future Crew discontinued Scream Tracker in the 1990s, you may still encounter old STM files created by the original application. You may also acquire STM files created by music enthusiasts using a third-party application that supports the STM format.

How to open an STM file

You can open an STM file with many audio applications, such as OpenMPT (Windows), Nullsoft Winamp (Windows), MikMod (multiplatform), and Schism Tracker (multiplatform). For example, you can open an STM file with OpenMPT by selecting File → Open....

How to convert an STM file

OpenMPT can convert STM files to the following formats:

  • .MID - MIDI File
  • .IT - Impulse Tracker Module
  • .WAV - WAVE Audio File
  • .FLAC - Free Lossless Audio Codec File
  • .AU - Audacity Audio File
  • .OPUS - Opus Audio File
  • .OGG - Ogg Vorbis Audio File
  • .RAW - Raw Audio Data
  • .MP3 - MP3 Audio File

Exchange Streaming Media File

Developer Microsoft
Popularity
3.4  |  19 Votes
 

Data file saved by Microsoft Exchange Server in conjunction with .EDB files; used by the Exchange information store, which is comprised of:

More Information

  1. Priv1.edb
  2. Priv1.stm
  3. Pub1.edb
  4. Pub1.stm

STM files store streamed native Internet content. New messages are saved to the STM file on the mail server until they are accessed by a user's mail client. When messages are accessed by a MAPI client, they are transferred to the EDB file and converted to the native Exchange format.

The Powder Toy Stamp File

Developer hardWIRED
Popularity
3.5  |  6 Votes
 

An STM file contains a stamp created by The Powder Toy, a free 2D physics simulation sandbox game. It stores a stamp, which is a customized area selected and saved in The Powder Toy during gameplay. STM files are considered "mini-saves" that can be used to make up a larger simulation save in the game, which is stored in the .CPS file.

More Information

The Powder Toy allows you to play with different substances to simulate the effects of different property interactions. The game comes with a variety of liquids, gases, electronic components, and building materials that you can use to build terrains, explosives, buildings, and more. During gameplay you can set a specified area to be saved and loaded at a different time. The elements or objects in the specified area are included in the save. These saved areas are called "stamps" and are stored in the STM file.

To create an STM file, type "S" during gameplay and draw the area you want to save as a stamp. To load an STM file, type "K" during gameplay and choose the stamp you want to load.

SSI Web Page

Developer N/A
Popularity

Web page that contains server side includes (SSI); references functions on the server that generates content for the Web page; processed by the server when accessed, which outputs HTML to client's Web browser.
