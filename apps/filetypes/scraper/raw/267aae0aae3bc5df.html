Pipe System Viewer File

Developer Engineered Software
Popularity
4.2  |  36 Votes
 

What is a PSV file?

A PSV file is a computer-assisted design (CAD) file created by Engineered Software PIPE-FLO Professional, an application used to design liquid piping systems. It contains a read-only model of a piping system. PSV files could previously be opened in PIPE-FLO Viewer, but that application is now discontinued.

More Information

PIPE-FLO Professional is a fluid system modeling and calculation program that fluid engineers use to design and test prototype piping systems. In the early 2000s, Engineered Software, the company that produces PIPE-FLO Professional, created the PSV file format and PIPE-FLO Viewer application to allow engineers to share project drafts with clients, vendors, and other engineers.

Those who opened a PSV file in PIPE-FLO Viewer could see not just the piping system's design, but also the system's test results and other associated information. They could not, however, edit the piping system, as PSV files were read-only.

How to open a PSV file

PSV files were meant to be opened in PIPE-FLO Viewer, a free application that Engineered Software made available on their website. Unfortunately, PIPE-FLO Viewer is now discontinued.

PlayStation 2 Save File

Developer Sony
Popularity
4.1  |  28 Votes
 

PlayStation 2 (PS2) game file created using a PlayStation 3 (PS3); saved when a PS2 game is played on a PS3 system; can be transferred from the PS3's hard drive to a memory card, which can be used to copy the PSV file to a computer's hard drive.

More Information

You can manipulate PSV files using PSV Exporter and PS2 Save Builder. First, you can extract files from a PSV file using PSV Exporter. Then, you can import the extracted files into PS2 Save Builder for conversion into another format, such as the Code Breaker .CBS format.

Pluralsight Video File

Developer Pluralsight
Popularity
4.1  |  15 Votes
 

A PSV file may also be a video used by Pluralsight Offline Player, a video player used to play training videos when offline. It contains a video of a training course downloaded from the Pluralsight website to be viewed offline.

More Information

Pluralsight is a technology learning platform that helps businesses assess the technical abilities of teams and teach business objectives. It provides users with a wide array of educational videos that can be viewed online. However, if a user needs to view the videos offline, they can download the videos stored in PSV files and view them with the Pluralsight Offline Player. Users can view downloaded videos in Windows and macOS using Pluralsight Offline Player and in Android and iOS using the Pluralsight app.

Pipe-Separated Values File

Developer N/A
Popularity
3.8  |  13 Votes
 

File used by various applications for data storage and transfer; separates data fields by the pipe "|" character similarly to how .CSV files use the comma "," character; useful when data fields contain embedded commas or tabs, such as with a text description field.

More Information

PSV files can be imported into a Microsoft Excel spreadsheet if you define the data delimiter as the pipe character. Microsoft Access can also import PSV files into data tables. Additionally, programmers sometimes use the sed Unix command line utility to parse PSV files for input data or for scripted data exchange applications.
