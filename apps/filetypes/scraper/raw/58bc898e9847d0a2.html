What are CST files and how to open them

Are you having problems opening a CST file or are you simply curious about its contents? We're here to explain the properties of these files and provide you with software that can open or handle your CST files.

What is a CST file?

CST files have multiple uses, and ComicStudio Story is one of them. Read more about the other uses further down the page.

ComicStudio Story

ComicStudio is a Japanese drawing program mainly used by illustrators, manga artists, and animation studios. It saves the story-line of a comic in files using file extension CST.

How to open CST files

    All known file formats using extension .CST

    While ComicStudio Story is a popular type of CST-file, we know of 6 different uses of the .CST file extension. Different software may use files with the same extension for different types of data.
