PaintShop Pro Image

Developer Corel
Popularity
3.5  |  33 Votes
 

What is a PSP file?

A PSP file is a bitmap image saved in the native Paint Shop Pro (PSP) format. It may include layers, guides, metadata, and other objects within the image. Corel replaced the PSP format with the .PSPIMAGE format when it released version 8 of Paint Shop Pro (now PaintShop Pro).

More Information

PaintShop Pro, often called "PSP," is an image editor that provides similar capabilities to Adobe Photoshop. For example, it allows you to edit images and digital photos with various tools and includes beginner-friendly and advanced workspaces.

Before being replaced by the PSPIMAGE format when Corel released version 8 of Paint Shop Pro, the PSP format was the native format for saving images with the application. After saving an image as a PSP file, you can close it and re-open it to continue editing it.

How to create a PSP file with PaintShop Pro

While Corel replaced the PSP format with the PSPIMAGE format, PaintShop Pro can still save images as PSP files. To create a PSP file with PaintShop Pro:

  1. Select File → New.
  2. Edit the image.
  3. Select File → Save or Save As.

NOTE: Jasc Software developed PaintShop Pro before Corel acquired it in 2004.

How to open a PSP file

You can open a PSP file with Corel PaintShop Pro (Windows) by selecting File → Open. You can also open PSP files with other image editors, such as the 32-bit version of CorelDRAW Graphics Suite (Windows and macOS) and ACD Systems Canvas (Windows).

How to convert a PSP file

Various applications, such as Corel PaintShop Pro and CorelDRAW Graphics Suite, can convert PSP files to other formats.

For example, PaintShop Pro can convert a PSP file to various image file types, including the following formats:

  • .PSPIMAGE - PaintShop Pro Image
  • .JPG - JPEG Image
  • .PNG - Portable Network Graphic
  • .BMP - Bitmap Image
  • .EMF - Enhanced Windows Metafile
  • .EPS - Encapsulated PostScript File
  • .TIF - Tagged Image File
  • .GIF - Graphical Interchange Format File
  • .TGA - Targa Graphic
  • .WMF - Windows Metafile
  • .PSD - Windows Metafile

PL/SQL Server Page

Developer Oracle
Popularity
3.4  |  15 Votes
 

Web page that contains code written in the Procedural Language/Structured Query Language (PL/SQL); includes SQL database commands that use Oracle's proprietary PL/SQL syntax; used for dynamically accessing database information when generating Web page content.

More Information

PSP pages can be viewed in a Web browser. However the user will only see the content generated by the server, not the actual PL/SQL code.

Pixel Studio Project

Developer Hippo Games
Popularity
4.0  |  2 Votes
 

A PSP file may also be a project file created by Hippo Games Pixel Studio, a multiplatform pixel art editor. It contains a layer-based pixel art image. Pixel Studio allows users to save PSP files locally, to cloud storage, or to the Hippo Games Pixel Network.

More Information

Pixel Studio allows pixel artists and video game developers to create pixel-based sprites and animations. The program is available for Windows, macOS, Android, and iOS, in both free and paid versions.

In all versions of Pixel Studio, artists can save the images and animations they've created as PSP files. The PSP format retains all of an image's layers, frames, and edit history. This allows Pixel Studio users to continue editing each of their image or animation's layers and frames, even after exiting Pixel Studio.

How to open a PSP file

You can open a PSP file with Hippo Games Pixel Studio (multiplatform). To do so:

  1. Select the three lines menu button that appears in the app's bottom-right corner.
  2. Select Open.
  3. Navigate to and open your PSP file.

How to convert a PSP file

You can use the paid version of Pixel Studio to convert the image or animation a PSP file contains to the following formats:

  • .AI - Adobe Illustrator Artwork
  • .EPS - Encapsulated PostScript
  • .EXR - OpenEXR Image
  • .HEIC - High Efficiency Image
  • .PDF - Portable Document Format
  • .SVG - Scalable Vector Graphic
  • .WEBP - WebP Image
  • .PSD - Photoshop Document
