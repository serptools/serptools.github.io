Pocket Excel File

Developer Microsoft
Popularity
2.8  |  9 Votes
 

What is a PXL file?

File created by Pocket Excel, a spreadsheet program used on Pocket PC mobile devices; contains data stored as a grid of rows and columns; can be converted into an Excel .XLS file.

More Information

PXL files can also be exported to XLS files in Pocket Excel by using the "Word 97/2000" type in the Save As... menu. Older versions of Microsoft ActiveSync can also convert PXL files to XLS files.

Printer Command Language XL Document

Developer HP
Popularity
2.3  |  3 Votes
 

A PXL file may also be a Printer Command Language document saved in the PCL 6 Enhanced (PXL) format. It contains instructions an HP printer uses to print a document. The instructions a PXL file contains are written in a hybrid binary and plain text format.

More Information

In the 1980s, Hewlett-Packard developed the first version of the Printer Command Language (PCL). HP printers use the commands and other data that .PCL files contain to print documents.

In 1995, HP developed version six of PCL. The Enhanced version of PCL 6 is commonly referred to as PCL XL or PXL. PXL instructions are sometimes saved in PXL files. You can send a PXL file to an HP printer to print the document it describes.

NOTE: PCL XL is HP's preferred printer language for enterprise and managed printers.

How to open a PXL file

You can open a PXL file with Ghostscript GhostPCL (multiplatform) or CoolUtils PCL Viewer (Windows) to view the document it describes.

Users familiar with the command line can use the following commands to send a PXL file to a compatible HP printer (after navigating to the directory that contains their PXL file):

  • Windows: print YourFile.pxl
  • Unix: lpr YourFile.pxl

Package and Extension Library Application

Developer Apple
Popularity
1.5  |  2 Votes
 

A PXL file may also be an iOS 1 software package. It is used to install an application on an iPhone running iOS 1. Some iOS 1 package managers allow users to manually install PXL files on their iPhones.

More Information

The original version of iOS used software packages saved in the PXL format to install applications. (Newer versions of iOS instead use .IPA files.) Typically, users never needed to manually manage or install these files. However, users who had jailbroken their iPhones could use iOS 1 package managers, such as iBrickr and Breezy, to manually install PXL files.

In the past, iBrickr's website maintained a repository of PXL files that users could install. This archive is still available at the Internet Archive.

How to open a PXL file

You can use iBrickr (Windows) or Breezy (Mac) to install the app a PXL file contains on an iPhone running iOS 1.
