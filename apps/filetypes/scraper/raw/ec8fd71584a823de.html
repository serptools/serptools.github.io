Quicken Data File

Developer Intuit
Popularity
3.4  |  46 Votes
 

What is a QDF file?

A QDF file is the primary data file created by the Windows version of Intuit Quicken, a personal finance management application. It contains all of the user's information, including accounts, transactions, portfolios, and balance information.

More Information

Quicken is a personal finance application for managing and tracking multiple financial accounts. Before the 2010 version of Quicken, the software saved user data in a series of files, all sharing the same filename but with different extensions (e.g., QDF, .IDX, .QEL, and .QPH). Intuit introduced the single-file version of the QDF format in the Windows 2010 version of Quicken to store all the data previously stored by the separate files.

While the Windows version saves user data as QDF files, the Mac version of Quicken saves user data as a .quicken file, followed by the version of Quicken that saved it. For example, the 2017 version of Quicken saves user data as .QUICKEN2017 files.

Additionally, if you are using the Windows version of Quicken, you can back up QDF files as .QDF-BACKUP files. The Mac version of Quicken saves QDF backups as quicken.backup files.

How to open a QDF file

You can open QDF files with the Windows version of Intuit Quicken in Windows and Intuit Quicken for Mac. To open a QDF file, select File → Open

How to convert a QDF file

The Windows version of Intuit Quicken can convert QDF files to various file types, including the following formats:

  • .QXF - Quicken Essentials for Mac Exchange File
  • .QIF - Quicken Interchange Format File
  • .XLSX - Microsoft Excel Spreadsheet
  • .PDF - Portable Document Format

Quadro Design File

Developer Quadro
Popularity
3.4  |  5 Votes
 

A QDF file contains a design used by Quadro, a program that allows users to design and build playsets, karts, and more. It stores a design, which includes components, such as tubes, panels, and connectors that make up the structure. QDF files are used to create a large variety of objects, such as playsets, pools, karts, and locomotives.

More Information

To open a QDF file in Quadro, simply select File → Open.... To save a QDF file in Quadro, select File → Save or Save as....

The QDF file is used to save a model built by a user in Quadro that can be closed and reopened for further editing. After completing the design, the user prints the design and builds it with the necessary Quadro components.

NOTE: You can create QDF files from scratch or modify one of the QDF templates available for download from the model database on the Quadro website.

Snapform File

Developer Ringler Informatik
Popularity
1.9  |  10 Votes
 

Document or electronic form created with Snapform Designer; saves data in a vector-based format that can be zoomed and printed without losing quality; may also contain form fields that can be filled in by the user; similar to the .PDF format, but uses proprietary Ringler Snapform formatting.

More Information

Snapform is available for Windows, Macintosh, and Linux platforms.
