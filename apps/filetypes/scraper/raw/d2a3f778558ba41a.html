Caltech Intermediate Form Integrated Circuit Design

Caltech Intermediate Form file extension or .cif file is a text file format used to render descriptive instructions for integrated circuits, most commonly known as a chip. The chip is so minuscule that applying any creation or alteration using unaided human capacity is challenging. CIF addresses this problem and is now widely used to describe a chip. CIF serves as a 'microscope' that helps users see through the chip.

CIF is generated to provide integration of various research tools into a standard database structure. CIF also provides a limited set of graphic primitives that are very useful for describing the 2D shapes on the different layers of a chip. It also allows hierarchical description, making the representation brief but comprehensive.
