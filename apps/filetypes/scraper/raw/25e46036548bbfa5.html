iGO Landmark File

Developer NNG
Popularity
3.8  |  40 Votes
 

What is a 3DL file?

GPS data file used by the iGO navigation software; contains points of interest, called landmarks; includes the location and associated metadata; saved to the \​iGO\​content\​building\​ directory of the navigation device's storage card.

More Information

3DL files are usually saved alongside a .3DC building file.

LightConverse 3D Model File

Developer Light Converse
Popularity

High-quality textured 3D model format used by LightConverse, a 3D lighting visualization and control software program; represents a three-dimensional scene optimized for real-time rendering with multiple light sources; contains mesh data, material attributes, and bitmap references.

Sierra LandDesigner 3D File

Developer Sierra Entertainment
Popularity
3.5  |  11 Votes
 

CAD file created by Sierra LandDesigner 3D, an older software used for generating garden and landscaping designs; may include layouts for various kinds of plants, trees, flowers, vegetables, groundcovers, and hardscapes; can be navigated as a 3D walkthrough in the software.

More Information

Several 3DL files are included with the LandDesigner 3D software and provide templates for landscape planning.

NOTE: Sierra LandDesigner was last released for Windows 95/98 and has since been discontinued.

3D Lookup Table File

Developer N/A
Popularity
3.0  |  6 Votes
 

A 3DL file contains 3D lookup table (3D LUT) data, which is used by various video and image editors to modify the color grade of videos. It stores a table of RGB color values that are indexed by sets of input RGB color values for adjusting color effects, such as shadows, contrast, and saturation. 3DL files are commonly used in the production of movies to specify the coloring of the final release print of a movie.

More Information

3DL files may be created by various programs, such as SpectraCal CalMAN, 3D LUT Creator, or Picture Instruments Look Converter. The 3DL file can then be imported into an image or video editor, such as Adobe Photoshop, Adobe Premiere Pro, Adobe After Effects, Andersson Technologies SynthEyes, and Assimilate Scratch. Once imported, the data in the 3DL file is loaded by the image or video editor and the color settings are applied.
