Itsy Package

Developer N/A
Popularity
3.2  |  9 Votes
 

What is an IPK file?

An IPK file is a compressed archive primarily used for installing software on handheld devices running Linux-based operating systems, such as WebOS and select embedded systems. It is based on the Debian package (.DEB) format and contains the required files and installation instructions for setting up an application or system update on a compatible device.

More Information

IPK archives typically include:

  • Program binaries the application requires to run.
  • Configuration files (.CONF or .INI files) that store settings for the application or system.
  • Library files (.SO files) required by the application to run.
  • Documentation files (.README, .HTML, or .TXT files) that provide instructions for users or developers.
  • Images or icons (.PNG and .JPG files) for graphics displayed in applications, such as splash screens or icons.
  • Install scripts that automate the installation process.
  • Other metadata, such as translations if the software supports multiple languages.

While the archives are typically used for distributing software and updates to handheld devices, select routers, IoT devices, set-top boxes (e.g., media players and Smart TVs), and development boards (e.g., Raspberry Pi and BeagleBone) may also use them. Non-Linux systems rarely use IPK archives.

NOTE: The "Itsy Package" name originates from the Itsy Package Management System, designed for lightweight and embedded Linux systems.

How to open an IPK file

You can open IPK files with the dpkg package management tool for the Debian operating system (a Linux distribution). You can also open IPK files with the 7-Zip compression utility in Windows.

webOS TV Application

Developer LG Electronics
Popularity
3.3  |  4 Votes
 

An IPK file may also be an application that can be installed and run on an LG webOS smart TV. Developers create IPK files using the webOS TV IDE. To run the app an IPK file contains, enable Developer Mode on your webOS TV and use the webOS TV IDE or command-line interface to install the app.

More Information

WebOS is a smart TV operating system used by LG televisions. Developers who create webOS apps do so using the webOS TV SDK, which contains a development kit, a command-line interface, and a webOS emulator.

After a developer finishes creating their webOS application, they package the app as an IPK file. They can then install the IPK file on a webOS TV with Developer Mode active, to continue testing their app, or upload the IPK file to the LG Smart TV Content Store, for distribution to other smart TVs.

How to open an IPK file

IPK files are meant to be installed and run on LG webOS smart TVs. After enabling Developer Mode on your LG webOS smart TV, you can use the webOS IDE or command-line interface to install your IPK file. Note that, after 50 hours, your TV will automatically leave Developer Mode and uninstall the IPK file (unless you manually extend the Developer Mode session time).

You can also use the webOS emulator, included with the webOS TV SDK, to run the app an IPK file contains.

Interactive Package

Developer Ubisoft
Popularity
2.7  |  3 Votes
 

An IPK file may also be a game data archive used by a video game developed with Ubisoft's UbiArt game engine. It contains a subset of the game's assets, such as images, videos, 3D models, and animation data. Games that use IPK files include Rayman Origins, Child of Light, and multiple games in the Just Dance series.

More Information

UbiArt is a proprietary game engine that UbiSoft created and began using internally in the mid-2000s. Games developed using the UbiArt game engine contain several proprietary-format files, including IPK files.

IPK files contain multiple related game assets, such as textures, shaders, and animation files used to show and animate in-game objects. Each IPK file is compressed using either Zlib or LZMA compression. Some of the files that IPK archives contain may themselves be compressed.

In the late 2010s, UbiSoft began phasing out the UbiArt game engine. However, UbiSoft still uses the engine to develop the Just Dance series of games.

How to open an IPK file

Modders who want to extract the game data that IPK files contain can do so using a variety of player-created tools. For example, QuickBMS (Windows) users have created many game-specific .BMS scripts that allow them to extract data from IPK files. You can also use the Ubiart Archive Tools utility to extract (and create) IPK files.
