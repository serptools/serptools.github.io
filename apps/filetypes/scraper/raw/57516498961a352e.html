Material Exchange Format

Developer SMPTE
Popularity
3.7  |  67 Votes
 

What is an MXF file?

An MXF file is a video saved in the Material Exchange file format developed by The Society of Motion Picture and Television Engineers (SMPTE). It stores video and audio data and metadata that describes that data. MXF files often store television advertisements, television archives, and movies meant to be delivered to theaters.

More Information

The SMPTE created the MXF format to streamline video exchange between platforms, including video cameras, digital editing equipment, and servers. The format adopts a subset of the Advanced Authoring Format (AAF), which standardized the interchange of video between multiple vendors and platforms. Like any other media container format, the appropriate codec must be available to play the media an MXF file contains.

How to open an MXF file

You can open an MXF file with various video editors and media players, such as Adobe Premiere Pro (multiplatform), Avid Media Composer (multiplatform), Apple QuickTime Player (Mac), and VideoLAN VLC media player (multiplatform). To open an MXF file with VLC media player, select File → Open File....

How to convert an MXF file

Various media applications, including Adobe Premiere Pro, Avid Media Composer, and VideoLAN VLC media player, can convert MXF files to other formats.

For example, Premiere Pro can convert MXF files to many file types, including the following formats:

  • .MP4 - MPEG-4 Video
  • .MOV - Apple QuickTime Movie
  • .M4V - MPEG-4 iTunes Video
  • .M2V - MPEG-2 Video
  • .JPG - JPEG Image
  • .PNG - Portable Network Graphic
  • .DPX - Digital Picture Exchange
  • .GIF - Graphical Interchange Format
  • .EXR - OpenEXR Image
  • .WAV - WAVE Audio
  • .MP3 - MP3 Audio
  • .AIF - Audio Interchange File Format
  • .XML - Final Cut Pro XML

Maxis Font File

Developer Electronic Arts
Popularity
3.1  |  7 Votes
 

Font file used by Maxis SimCity 4, a life simulation game; stored in the \​Fonts\​ directory of the game installation and contains a font such as Arial, Courier, or another font; used for displaying game text.
