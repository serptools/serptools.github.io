CorelDRAW File

Developer Corel
Popularity
3.8  |  606 Votes
 
NEW: Open your .CDR file online with File Helper.View Online

What is a CDR file?

A CDR file is a drawing created with CorelDRAW, a vector illustration and page layout design application included with CorelDRAW Graphics Suite. It typically contains vector image data, such as lines, paths, and colored shapes, but may also include text, tables, and digital effects. The CDR format is commonly used for documents, such as brochures, tabloids, letters, advertisements, postcards, webpages, and web banners.

More Information

The CorelDRAW Graphics Suite includes several apps, such as CorelDRAW and PHOTO-PAINT. Graphic designers and digital photographers commonly use the suite to create and edit artwork.

CDR is the primary format associated with CorelDRAW. When you save a graphic or document with CorelDRAW, the application saves it with a .cdr extension default.

If you want to save a CorelDRAW file as a template to replicate design and content, select File → Save as Template. CorelDRAW creates a .CDRT file when saving a template, which you can use as a starting point for creating new CDR files.

How to open a CDR file

You can open a CDR file with the CorelDRAW application included with CorelDRAW Graphics Suite (Windows and macOS). To open a CDR file with CorelDRAW, select File → Open.

You can also open CDR files with Corel PaintShop Pro (Windows), Adobe Illustrator (multiplatform), and Inkscape (multiplatform).

How to convert a CDR file

Various image editors, including CorelDRAW Graphics Suite and Adobe Illustrator, can convert CDR files to other image formats.

For example, CorelDRAW can convert CDR files to many formats, including the following file types:

  • .JPG - JPEG Image
  • .PNG - Portable Network Graphic
  • .BMP - Bitmap Image
  • .GIF - Graphical Interchange Format
  • .SVG - Scalable Vector Graphics File
  • .DWG - AutoCAD Drawing
  • .DXF - Drawing Exchange Format
  • .RTF - Rich Text Format
  • .DOC - Microsoft Word Document (Legacy)
  • .WPD - WordPerfect Document
  • .PDF - Portable Document Format
  • .PSD - Adobe Photoshop Document
  • .AI - Adobe Illustrator Artwork

Macintosh DVD/CD Master

Developer N/A
Popularity
3.5  |  40 Votes
 

Disc image created on a Macintosh computer using the built-in Disk Utility program; may contain multiple folders and files in a single disc image file that can be burned to a CD; similar to an .ISO file created on a Windows computer.

More Information

To create a CDR file with Apple Disk Utility, first, select the "Convert" option, then open an existing disc image file. Next, select "DVD/CD master" from the Image Format menu and click "Save."

NOTE: If a CDR file will not open on a Windows computer, it may be possible to open the file by changing the file extension from .cdr to .iso.

Raw CD Audio Data

Developer N/A
Popularity
3.4  |  40 Votes
 

A CDR file may also be an audio track extracted, or "ripped," from an audio CD to a computer's hard disk. It is similar to an .AIF or .WAV file, but does not contain the same header information. Many music players can play CDR files.

More Information

Users create CDR files using CD audio extraction software, such as HH Software Virtual CD. The files typically contain standardly-formatted audio that is saved without the format's typical header information. As such, music players may not recognize the files as audio files by default. However, you can still play CDR files in many music players.

How to open a CDR file

If your CDR file contains music saved in a standard format, you can open and play the file in VideoLAN VLC media player (multiplatform). You can also use programs like HH Software Virtual CD (Windows), Lightning UK ImgBurn (Windows), and Roxio Toast (Mac) to burn CDR files to CDs.

How to convert a CDR file

If you can play your CDR file in VideoLAN VLC media player, you can also use the program to convert your CDR file to other audio formats, including:

  • .MP3 - MP3 Audio
  • .OGG - Ogg Vorbis Audio
  • .FLAC - Free Lossless Audio Codec
  • .WAV - WAVE Audio

Crash Data Retrieval Data File

Developer Bosch
Popularity
2.8  |  28 Votes
 

A CDR file contains Event Data Recorder (EDR) information acquired from automobiles with a Crash Data Retrieval (CDR) device. The EDR records events generated from sensors within the car and then saves them to the CDR file. Users can then transfer the CDR file to a computer and import it into Bosch CDR System Software installed on the computer.

More Information

Bosch introduced CDR technology in 2000, which crash researchers, law agencies, auto manufacturers, and government agencies, use to access EDR information from vehicles, such as cars and SUVs. By viewing the data in a CDR file, you can understand the specifics of the crash, including where the impact was on the vehicle and the force of the impact.

How to open a CDR file

You can open CDR files with Bosch CDR System Software. For example, to open a CDR file, select File → Open.
