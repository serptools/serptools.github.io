.SPP File Extension

3 file types use the .spp file extension.
  • 1.Serif PhotoPlus Picture File
  • 2.Substance Painter Project
  • 3.Altova XMLSpy Project

Serif PhotoPlus Picture File

Developer Serif
Popularity
3.6  |  21 Votes
 

What is an SPP file?

Picture saved in the native PhotoPlus image format; may include layers, layer effects, transparency, editable text, and QuickShapes; may also include mesh data, which allows parts of the image to be distorted using the Mesh Warp tool.

More Information

SPP files are similar to Photoshop (.PSD) documents, but are created with Serif PhotoPlus.

NOTE: Serif discontinued PhotoPlus and replaced it with Affinity Photo in 2015.

Substance Painter Project

Developer Allegorithmic
Popularity
3.0  |  5 Votes
 

An SPP file may also be a project created by Allegorithmic Substance Painter, a 3D texturing program for Windows, macOS, and Linux. It contains project information, which includes meshes and textures that make up the 3D model. SPP files also include metadata used to configure the project.

More Information

The SPP file is the main file type associated with Substance Painter. You can create an SPP file by selecting File → Save or Save As.... You can open an SPP file by selecting File → Open.... You can also export the SPP file to other formats, such as .BMP, .PSD, .JPEG, .HDR, .TIFF, and .GIF, in the File menu.

NOTE: Substance Painter is used for a variety of 3D animation fields, such as visual effects, feature animations, and video games.

Altova XMLSpy Project

Developer Altova
Popularity
3.0  |  1 Vote
 

An SPP file may also be a project created by Altova XMLSpy, an XML editor and IDE designed for modeling, editing, and managing XML-based applications. It stores the properties of a project in the XML standard, including paths to the locations of files included in the project, such as .XML, .XSL, .XSD, and .HTM files. SPP projects allow XMLSpy users to manage and organize multiple source files within a single workspace to streamline their workflow.

More Information

XMLSpy supports various XML-related formats, including XSLT, XQuery, and XSD, and provides tools for XML validation, transformation, and debugging. The software also features a visual schema editor, a code generator, and advanced functionalities for working with databases, JSON, and other data formats, making it helpful for developers handling data integration and web services.

You can link files and organize folders within an SPP project, which is especially helpful for managing dependencies and performing batch operations when developing. Also, SPP projects enhance collaboration by enabling you to share access to the same ready-to-use project setup with team members.

How to open an SPP file

You can open SPP files with Altova XMLSpy in Windows. To open an SPP file, select Project → Open Project.

Since SPP files store plain text in XML, you can open them with a text or XML editor. However, opening SPP files with a text or XML editor will not appear the same as opening them with XMLSpy.
