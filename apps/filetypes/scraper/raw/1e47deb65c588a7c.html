What is a TT file?

A .TT file is a Tiny Tafel Format Data file.

Presenting genealogical data with precision and clarity is essential for successful research and study. The Tiny Tafel (TT) format offers a solution for achieving this, by providing an efficient way to display surnames in a logically organized overview. The Tiny Tafel provides a Soundex-encoded version of every surname, so you can match those with different spellings.

TT files are easily accessed from programs offering genealogy tools, including editing, reports and chart creation. Its versatility makes it suitable for all scales of project, from individual family trees to large-scale academic studies. This adaptability has made the TT file format an industry standard.

Using TT files allows researchers to quickly find the key surnames associated with a family tree. A standardized structure places information into specific categories, making comparisons between different genealogies simpler, thus enabling greater insight.
