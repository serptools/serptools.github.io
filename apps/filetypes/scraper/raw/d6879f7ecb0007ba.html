Are you having problems opening an ECW file or are you simply curious about its contents? We're here to explain the properties of these files and provide you with software that can open or handle your ECW files.

What is an ECW file?

ECW files have multiple uses, and Enhanced Compressed Wavelet Image is one of them. Read more about the other uses further down the page.

Enhanced Compressed Wavelet Image File

ECW or the enhanced compressed wavelet file extension is a type of image file format compression wherein it stores information captured using satellites and aerial cameras or georeaders placed in space. This file type is known to capture aerial pictures via satellites for the development of geolocation and mapping databases.

ECW files are popular among developers and users of Geographic Information System, and thus, there are a lot of applications that can be utilized to view files with .ecw format.

Earth Resource Mapping pioneered enhanced compressed wavelet and was later owned by ERDAS (Earth Resource Data Analysis System). ECW is an ideal image format for large image files because of its ability to achieve significant compression rates that other formats have difficulty with.

ER Mapper Image Compressor is an application designed to solely support .ecw files. Since aerial images of geographic locations and areas are generally large, they would take up a large memory size. Nonetheless, the said compressor can significantly compress the image to reduce the file size without altering or losing the original image quality.
