What is a P97 file?

A P97 file is a trunk tracking database created by Starrsoft Win97. It contains a record of the channels, frequencies, and other data logged from a trunked radio system, using the PRO-97 or PRO-2055 trunk tracking scanners. P97 files can be opened only with Win97.

More Information

Radio system enthusiasts use trunk tracking scanners, like Radio Shack's PRO-97 and PRO-2055 scanners, to scan for and log activity that takes place on trunked radio systems. These systems are comprised of one or more control channels and a set of other associated frequencies. While tuned to the control channel, a user's two-way radio automatically sends and receives broadcasts from the associated (trunked) frequencies.

After logging data from a trunked radio system, enthusiasts often transfer that data to associated trunk tracking software, such as Win97. They can then analyze, edit, and export the data they've collected from their trunk tracking scanner.

How to open a P97 file

You can open a P97 file with Starrsoft Win97 (Windows). However, Win97 is discontinued. Thus, if you do not already have access to Win97, you likely will not be able to open your P97 file.

How to convert a P97 file

You can use Win97 to export the database a P97 file contains in the .CSV (Comma-Separated Values) format.
