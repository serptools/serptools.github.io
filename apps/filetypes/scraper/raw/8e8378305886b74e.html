Windows Script

Developer Microsoft
Popularity
3.8  |  29 Votes
 

What is a WS file?

A WS file contains an executable script that can be run via the Windows Script Host. The script may be comprised of various types of routines and elements, including JScript and VBScript routines and XML elements. When you double-click a WS file, the Windows Script Host will automatically run the script the file contains.

More Information

The Windows Script Host is a tool that system administrators can use to automate a variety of login and administrative actions. To do so, administrators create and run Windows Scripts, which may be saved as WS files.

WS files are plain text files. Notably, WS files can include routines written using JScript, VBScript, BASIC, Perl, Python, and many other programming languages. To create WS files, administrators use Microsoft Notepad, Microsoft Visual Studio Code, or another text editor.

NOTE: Windows Scripts are more commonly saved as .WSF files.

How to open a WS file

If you want to view or edit a WS file's contents, you can open it with Microsoft Notepad, Microsoft Visual Studio Code, or any other text editor.

In Windows, you can run the script a WS file contains by double-clicking the file or using the wscript command-line command (as described here.)

Warning

Do not double-click a WS file that you receive as an email attachment or from an untrusted source. Because Windows automatically runs WS files when users double-click them, cybercriminals sometimes use WS files to attack users' computers.

WonderSwan Game ROM

Developer N/A
Popularity
3.0  |  5 Votes
 

A WS file may also be a game ROM file used by WonderSwan emulators, such as Oswan, OpenEmu, and Higan. It contains an exact copy of the game data dumped from the original WonderSwan game cartridge.

More Information

WonderSwan emulators allow users to play WonderSwan games on a computer as if they were run on the original system. Game data from the original WonderSwan game cartridge is dumped into a WS file, which stores the game data in a format that can be played by a WonderSwan emulator.

WonderSwan is a Japanese handheld video game console created by Bandai. It was released in 1999 as a competitor to Nintendo's Game Boy Color. It was discontinued in 2003. WonderSwan game titles include Buffers Evolution, Beatmania, Final Lap 2000, Gunpey, Harobots, and Puzzle Bobble.

Whitespace Source Code

Developer Edwin Brady and Chris Morris
Popularity
3.0  |  2 Votes
 

A WS file may also be a program written in the Whitespace programming language. It contains Whitespace source code, which consists of spaces, tabs, and linefeeds (whitespace characters). WS files also often contain comments written using letters, numbers, and other non-whitespace characters.

More Information

Whitespace is a programming language developed by Edwin Brady and Chris Morris. The Whitespace interpreter ignores any non-whitespace characters, so Whitespace programs are written using only spaces, tabs, and linefeeds, which the interpreter reads as either commands or data. Commands are written using preset sequences of spaces, tabs, and linefeeds. Other data is written in binary, using spaces as 0s and tabs as 1s.

While Whitespace is not used for serious development, some developers enjoy developing within the language's limitations and creating fun Whitespace programs. These developers may save their Whitespace programs as WS files.

How to open a WS file

You can open and edit a WS file in any text or source code editor, such as Microsoft Visual Studio Code. Keep in mind that, because Whitespace programs are written using only whitespace characters, your WS file may appear empty. You can run the program a WS file contains using a Whitespace interpreter.
