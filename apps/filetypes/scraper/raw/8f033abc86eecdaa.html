Impulse Tracker Module

Developer Jeffrey Lim
Popularity

What is an IT file?

An IT file is a music module saved in the Impulse Tracker Module (IT) format. It contains a sequence of notes, or patterns, that play back sound samples stored in the file. Various audio-tracking programs may create IT files, including Impulse Tracker and OpenMPT.

More Information

Impulse Tracker is a free music tracker developed by Jeffrey Lim that was released in 1995 and is no longer developed. The music sequencing program allows you to create music by organizing notes in different channels as patterns that play samples stored in instruments (.ITI files).

In Impulse Tracker, you can save your modules as IT files, which you can share with other audio-tracker users who can import them into software that supports the module format. IT files are similar to ScreamTracker 3 modules (.S3M files) but support more data per pattern and higher-quality samples.

How to open an IT file

You can open an IT file with Impulse Tracker (Windows). However, the software is not actively developed and may not run on your operating system version.

Other applications that can open IT files include OpenMPT (Windows), Nullsoft Winamp (Windows), and Schism Tracker (multiplatform).

You can also open ITI files with Audacity with the FFMPEG v2.2.2 or later plug-in installed.

How to convert an IT file

Various programs, such as OpenMPT and Schism Tracker, can convert IT files to other file types.

For example, OpenMPT can convert IT files to the following formats:

  • .MPTM - OpenMPT Module
  • .MID - MIDI File
  • .WAV - WAVE Audio File
  • .FLAC - Free Lossless Audio Codec File
  • .AU - Audacity Audio File
  • .OPUS - Opus Audio File
  • .OGG - Ogg Vorbis Audio File
  • .RAW - Raw Audio Data
  • .MP3 - MP3 Audio File
