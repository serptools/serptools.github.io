All known file formats using extension .DM

2 known uses of the DM file extension

  • Delta Music Module
  • Sony DRM Encoded File

Sony DRM Encoded File

Files that contain the .dm file extension usually are associated with media files that have been encoded with Digital Rights Management copy-protection software.

These files ensure that an associated media file is not used without the distributor's express consent and that user's cannot copy or distribute the associated media illegally. DM files usually contain audio files used for cellphone ring tones and other cell media clips.

The digital rights management DM files are created with the Sony Ericson DRM Packager. Media developers use this utility to deliver their media content to Sony Ericson mobile devices.
