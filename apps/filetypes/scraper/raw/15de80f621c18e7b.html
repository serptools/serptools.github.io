What are IFC files and how to open them

Are you having problems opening an IFC file or are you simply curious about its contents? We're here to explain the properties of these files and provide you with software that can open or handle your IFC files.

What is an IFC file?

IFC files have multiple uses, and IFC-SPF is one of them. Read more about the other uses further down the page.

How to open IFC files

All known file formats using extension .IFC

While IFC-SPF is a popular type of IFC-file, we know of 2 different uses of the .IFC file extension. Different software may use files with the same extension for different types of data.

2 known uses of the IFC file extension

  • IFC-SPF
  • ISO-10303-21 STEP-File
