VisualBoyAdvance Saved Game

Developer VisualBoyAdvance
Popularity
3.8  |  32 Votes
 

What is an SGM file?

Saved state file created by VisualBoyAdvance, a Game Boy emulator; contains a snapshot of the game state, which can be saved at any time during gameplay; used for saving and loading in-game progress.

More Information

In VisualBoyAdvance, SGM files can occupy one of ten save slots per game. The files are named using the ROM filename prefix and the number of the save slot. For example, the .GBA ROM MyGame.gba could have save slot 1 created with the filename MyGame1.sgm.

Starpoint Gemini 2 Mod Package

Developer Little Green Men
Popularity
3.0  |  2 Votes
 

An SGM file may also be a mod package used by Starpoint Gemini 2, a space trading and combat game. It contains a mod package, which includes all the content of a mod. SGM files are used for modding various aspects of the game, including spaceships, stations, planets, weapons, game world regions, mercenaries, and equipment.

More Information

SGM files are created from SGS mod definition files, which are used to define all the parts of a mod. You can use the game's mod manager to pack all the files in a mod into the SGM mod package in order to publish and share it with other Starpoint Gemini gamers.

NOTE: SGM files are commonly located in the "Mods" folder of the Starpoint Gemini installation directory.
