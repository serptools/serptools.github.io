ART File

What are ART files and how to open them

Are you having problems opening an ART file or are you simply curious about its contents? We're here to explain the properties of these files and provide you with software that can open or handle your ART files.

What is an ART file?

ART files have multiple uses, and AOL ART Bitmap is one of them. Read more about the other uses further down the page.

AOL ART Bitmap File

Graphcis format using Johnson-Grace compression.

How to open ART files

Important: Different programs may use files with the ART file extension for different purposes, so unless you are sure which format your ART file is, you may need to try a few different programs.

While we have not verified the apps ourselves yet, our users have suggested ten different ART openers which you will find listed below.

All known file formats using extension .ART

While AOL ART Bitmap File is a popular type of ART-file, we know of 9 different uses of the .ART file extension. Different software may use files with the same extension for different types of data.
