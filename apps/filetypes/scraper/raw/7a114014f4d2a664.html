Tibo Jigs@w Puzzle Save File

Developer Tibo Software
Popularity
3.1  |  8 Votes
 

What is a QSV file?

A QSV file contains a saved puzzle created by Jigs@w, a program used for creating and playing jigsaw puzzles. It stores the state of a puzzle when it was saved, which includes the placement of pieces and the number of pieces that are connected or disconnected.

More Information

Jigs@w puzzles may be built from ordinary .JPG, .TIF, .GIF, .PNG, and .BMP files as well as many other image formats. The puzzles are typically saved as .PZL files, but they may be compressed into .PZP zip files for sharing over the Internet.

iQIYI Video File

Developer iQIYI
Popularity
2.8  |  4 Votes
 

A QSV file may also be a video file used by iQIYI, an online Chinese online video platform. It stores compressed video data in the proprietary iQIYI multimedia container format. QSV files can only be played with the iQIYI video platform.

More Information

You will most likely only encounter a QSV video file if you download a video from iQIYI and save it as a QSV file. You may also receive a QSV file from a co-worker, friend, or family member. However, QSV files typically only store videos in Chinese, which makes them uncommon in areas outside of China.

Since QSV files are encrypted and saved in the proprietary iQIYI format, they can only be opened and viewed with iQIYI. However, you can convert QSV files to the more widely supported .MP4 format with Swift Video Converter.
