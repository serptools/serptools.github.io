What is an SVGZ file?

An SVGZ file is a Scalable Vector Graphics (.SVG) file compressed with gzip compression. It contains graphics data in XML, which includes the positioning of lines, text, curves, colors, and shapes that make up a two-dimensional graphic. SVGZ files also support layers, transparency, gradients, animations, and filters.

More Information

Adobe developed the SVGZ format to reduce the size of SVG files, which is helpful when transferring SVG graphics online. SVGZ files are typically 50 to 80 percent smaller in size than uncompressed SVG files.

Graphics professionals often use SVGZ files to transfer SVG graphics to share with other professionals. Also, web designers utilize SVGZ files to share SVG graphics with other designers, which are widely supported by web browsers.

How to open an SVGZ file

You can open SVGZ files with various image-editing programs. Some applications that support SVGZ files include Adobe Illustrator (Windows and macOS), Corel PaintShop Pro (Windows), Inkscape (multiplatform), and SVG and SVGZ Viewer (Android).
