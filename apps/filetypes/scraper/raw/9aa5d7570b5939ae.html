What are AUP files and how to open them

Are you having problems opening an AUP file or are you simply curious about its contents? We're here to explain the properties of these files and provide you with software that can open or handle your AUP files.

What is an AUP file?

AUP files have multiple uses, and Audacity Project is one of them. Read more about the other uses further down the page.

Audacity Project

Audacity is a multi-platform audio recording and editing software program. The Audacity software application allows users to edit and optimize audio data. Using the program, a user can break larger files into smaller volumes so that the software can open and save the associated project files more quickly.

When an audio project is created using the Audacity application, the file is saved with the .aup file extension.
