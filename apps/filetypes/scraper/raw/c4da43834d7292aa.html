.MPT File Extension

Microsoft Project Template

Developer Microsoft
Popularity
3.3  |  11 Votes
 

What is an MPT file?

An MPT file is a template created by Microsoft Project, a project management application. It contains a pre-defined structure, including a project timeline, tasks, and monetary budget. Project users utilize MPT files as a starting point for creating .MPP projects.

More Information

Microsoft introduced Project in 1984 and has since released subsequent versions. The software allows you to manage projects by developing schedules, tracking progress, assigning tasks, and analyzing budgets.

How to create an MPT template with Microsoft Project

To save a project as a template in Microsoft Project, follow these steps:

  1. With the project already open, click File → Export.
  2. Click Save Project as File, then choose Project Template.
  3. Name the template and click Save.

To create a project from a template in Microsoft Project, follow these steps:

  1. Click File → New.
  2. Choose the template you want to use by searching your templates or online templates.
  3. Click Create in the preview dialog box of the template you selected.

How to open an MPT file

You can open and edit an MPT template with Microsoft Project in Windows. You can also open an MPT file with third-party applications, such as MOOS Project Viewer and MPXJ.

How to convert an MPT file

Various applications, including Microsoft Project and MPXJ, can convert MPT files to other formats.

For example, Microsoft Project can convert MPT files to many file types, including the following formats when you select File → Save As:

  • .MPP - Microsoft Project File
  • .PDF - Portable Document Format
  • .XLSX - Microsoft Excel Spreadsheet
  • .CSV - Comma-Separated Values File
  • .XML - XML File
  • .XPS - XML Paper Specification File
