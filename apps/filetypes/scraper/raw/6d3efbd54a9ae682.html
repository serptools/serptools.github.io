What is a SHTM file?

The SHTM file extension indicates to your device which app can open the file. However, different programs may use the SHTM file type for different types of data.

While we do not yet describe the SHTM file format and its common uses, we do know which programs are known to open these files, as we receive dozens of suggestions from users like yourself every day about specific file types and which programs they use to open them.
