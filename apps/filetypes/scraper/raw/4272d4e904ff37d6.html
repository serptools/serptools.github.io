Cal3D Binary Mesh File

Developer Cal3D
Popularity
3.3  |  15 Votes
 

What is a CMF file?

Three-dimensional mesh file created using Cal3D, a character animation library authored in C++; stores the 3D geometry of a character using polygons made up of nodes and edges; used for structuring the character.

More Information

Cal3D meshes can also be stored in an XML form using the .XMF extension.

Creative Music Format

Developer Creative Labs
Popularity
3.6  |  5 Votes
 

Audio file format developed by Creative Labs; designed to be used with Creative Sound Blaster sound cards and Creative software; can also be played by some third-party programs with supporting plug-ins installed.

More Information

CMF files store music data as a series of notes, similar to a standard MIDI (.MID) file. However, they also contain instruments, which are used for playing back different sounds.

NOTE: Sound Blaster Instrument (.SBI) files are similar to CMF files, but only contain instrument data.

Connected Backup File

Developer Autonomy
Popularity
2.5  |  4 Votes
 

Backup file created by Connected Backup, a program used for creating secure local and remote backups of computer data; stores backed up files in a secure proprietary format; can significantly reduce the size of the backup data, but may still be very large in size.

More Information

CMF files can be used for backing up email data from Microsoft Outlook and IBM Lotus Notes. The software's EmailOptimizer (EMO) component stores the backup CMF files in the following locations in Windows:
