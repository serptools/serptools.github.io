What are ICD files and how to open them

Are you having problems opening an ICD file or are you simply curious about its contents? We're here to explain the properties of these files and provide you with software that can open or handle your ICD files.

What is an ICD file?

An .ICD file is an IronCAD 2D Drawing file.

These ICD files contain drawings created in the IronCAD Computer Aided Design software.

The files contain data like objects, lines, and other style information that makes up the drawing.
