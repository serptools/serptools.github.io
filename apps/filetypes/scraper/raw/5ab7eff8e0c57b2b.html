Pages Template

Developer Apple
Popularity
3.4  |  18 Votes
 

What is a TEMPLATE file?

A TEMPLATE file is an Apple Pages document template. It contains preset page layout information, document elements, and styles used to create a standardly-formatted document, such as a report or resume. Using TEMPLATE files, users can create multiple similar .PAGES documents without manually formatting each of them.

More Information

Apple Pages is the default word processor included with macOS. It allows users to create reports, posters, resumes, brochures, and other documents.

Optionally, Pages users can save a document they've created as a Pages template, to make it easier to create similar documents in the future. For example, if a user has created a heavily-formatted brochure, they may want to save that brochure as a Pages template. Then, instead of creating new brochures from scratch, the user can open their template to get started more quickly.

Pages saves document templates as TEMPLATE files. Primarily, users encounter these templates on Pages' Choose a Template screen, when creating a new document. There, Pages shows its default TEMPLATE files, as well as any templates a user has created.

How to create a Pages template

To save a Pages document that you are working on as a Pages template:

  1. Select File → Save as Template....
  2. Select Add to Template Chooser to add the template to your My Templates list.

Alternatively, you can select Save... to save the template to your computer as a TEMPLATE file. You may want to do this if, for example, you want to transfer and use the template in Pages on a device that is not connected to your iCloud account.

How to open a TEMPLATE file

To open a TEMPLATE file on a device with Apple Pages installed, either double-click (macOS) or tap (iOS) the file. Apple Pages will then ask whether you want to add the template to your Template Chooser, so you can select it from My Templates in the future, or just create a single new document using the template.

How to convert a TEMPLATE file

After opening a TEMPLATE file in the macOS version of Apple Pages, you can convert the document it contains to any of the following formats (by selecting File → Export):

  • .DOCX, .DOC - Microsoft Word Document
  • .PDF - Portable Document Format
  • .EPUB - Open eBook
  • .RTF, .RTFD - Rich Text Format
  • .TXT - Plain Text
  • Pages '09 Document

In the iCloud version of Pages, you can convert the document a TEMPLATE file contains to PDF, DOCX, or EPUB, by selecting the ... button and then Download a Copy....

Jigdo File Template

Developer Richard Atterer and Steve McIntyre
Popularity
1.0  |  1 Vote
 

A TEMPLATE file may also be a Jigdo file template. It is used to target and download pieces of a large file, so Jigdo can combine those pieces into a single file. TEMPLATE files are paired with accompanying .JIGDO files, which are used to finalize the file creation process.

More Information

Most often, technically-minded PC users use Jigdo (short for Jigsaw Downloader) to download and reconstitute disk images that have been split into several constituent files. To begin a download, users run the jigdo-lite script and specify a JIGDO file. Doing so allows Jigdo to retrieve an accompanying TEMPLATE file, which specifies all the files Jigdo must retrieve before creating the disk image.

After loading a TEMPLATE file, Jigdo will ask a user to specify which, if any, pieces of a disk image they already possess. When a user finishes specifying files, Jigdo uses the TEMPLATE and JIGDO file to download any missing constituent files and produce a finalized, complete disk image.

NOTE: Typically, disk images created using Jigdo contain and are used to install Linux-based operating systems.

How to open a TEMPLATE file

TEMPLATE files are not meant to be opened. For complete instructions on how to use TEMPLATE and JIGDO files with Jigdo, refer to Jigdo's documentation.
