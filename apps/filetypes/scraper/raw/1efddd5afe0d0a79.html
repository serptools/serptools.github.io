FrameMaker Maker Markup Language File

Developer Adobe Systems
Popularity
3.4  |  20 Votes
 

What is an MML file?

File created using the Adobe Maker Markup Language (MML); used by Adobe FrameMaker, a program used to produce and publish large structured documents; contains formatting instructions for page layout, graphics, text, and other page properties.

More Information

MML files may be imported into an existing FrameMaker document, or they may be used to generate a new page layout. They are stored in an ASCII text format.

Mathematical Markup Language File

Developer W3C
Popularity
4.3  |  6 Votes
 

An MML file may also be an XML document that includes one or more mathematical formulas written using the Mathematical Markup Language (MathML). MML files are used to integrate mathematical formulas into .HTML webpages. You can recognize MML files by the <math xmlns="http://www.w3.org/1998/Math/MathML"> element.

More Information

In 1998, the Worldwide Web Consortium (W3C) released the first version of MathML. This markup language helps developers describe mathematical formulas and integrate those formulas into webpages.

Most commonly, developers use MathML to write formulas directly within HTML and .XHTML documents. However, developers can also save MathML formulas in MML files and reference those files in HTML and XHTML documents. For example, if a developer plans to include the same MathML formula in multiple webpages, they can save the formula as a single MML referenced by multiple webpages.

Just like typical .XML documents, MML documents contain data structured using a series of tags. However, the tags MML documents contain primarily describe the components of a mathematical formula. For example, the <mo> tag is used to denote a mathematical operator, such as + or -, and the <mn> tag is used to denote a number.

How to open an MML file

You can open an MML file in Microsoft Visual Studio Code, GitHub Atom, or any other text or source code editor. Source code editors include helpful tools, such as syntax highlighting, that make viewing and editing MML files easier.

You can also open an MML file and edit the formula it contains in OpenOffice Math, LibreOffice Math, and Wolfram Research Mathematica.

Map Markup Language File

Developer Mapbox
Popularity
1.7  |  3 Votes
 

An MML file may also be a metadata file used by CartoCSS, a pre-processor used by Mapbox, OpenStreetMap, Mapnik, and other mapping applications. It defines the layers a map contains, the stylesheet used to set the map's appearance, and other settings. The data that MML files contain can be formatted using either JSON or YAML.

More Information

CartoCSS, also known as Carto, consists of both a CSS-based styling language and a related library that can be used to create and load geographic maps. These maps are typically accompanied by stylesheets (.MSS files) and metadata files that determine how the features they contain appear in mapping applications.

MML, or Map Markup Language, files specify the layers a map contains, the stylesheet it uses, its output image format, its initial center coordinates and zoom level, its maximum and minimum zoom level, and several other settings. Typically, an MML file references an MSS file to load stylesheet settings. However, users can also manually specify stylesheet settings within an MML file itself.

How to open an MML file

You can open an MML with Microsoft Visual Studio Code or any other text editor to view the data it contains. For instructions on how to use MML files with CartoCSS, refer to the program's official documentation.
