GenePix Results File

Developer Molecular Devices
Popularity
3.5  |  8 Votes
 

What is a GPR file?

Data file created by GenePix Pro Software, an analysis program for biological experiments; saves results for an experiment conducted using a GenePix Array List (.GAL file); used for storing quantitative analyses from microarray data.

More Information

GPR files use the Axon Text Format (.ATF files), which is a plain text, tab delimited format.

GoPro RAW Image

Developer GoPro
Popularity
3.8  |  5 Votes
 

A GPR file may also be a camera RAW image saved in the GoPro RAW (GPR) format. It stores image data captured by various GoPro camera models, such as HERO9 Black, Fusion, and GoPro MAX. Action-oriented photographers, such as extreme sports enthusiasts, travelers, and video producers, commonly edit GPR images they have captured with their GoPro cameras.

More Information

The GPR format is based on Adobe Systems' DNG format but features enhanced VC5 compression, resulting in smaller files without reducing quality. Also, the GPR format does not include an embedded .JPG preview like the DNG format. Instead, GoPro cameras save a separate JPG file with the GPR file.

GoPro cameras that support the GPR format allow you to capture images in the GPR format or the JPEG format. JPEG features heavier compression, which results in lower resolution images and smaller file sizes. You must turn on the RAW photo setting on the GoPro camera to capture photos as GPR files. Otherwise, the camera will capture images as JPG files.

When to capture images as GPR files

If you want to capture images at the highest quality with your GoPro camera, you should save them as GPR files. The higher resolution results in better-looking images and allows you to edit them more accurately.

However, if image quality is not crucial and storage space is more of a concern, you should choose to save images as only JPG files. You can also share JPG files more readily than GPR files since many image viewers and editors support the JPEG format.

How to open a GPR file

You can open a GPR image with several image editors and viewers, such as Adobe Photoshop with the Adobe Camera Raw (version 9.7 or later) plugin and Adobe Lightroom. To open a GPR file with Lightroom, follow these steps:

  1. Select File → Add Photos...
  2. Click Review for Import
  3. Select Add 1 Photo

How to convert a GPR file

Several image converters and editors, such as Adobe Adobe Lightroom and Adobe Photoshop (with the Adobe Camera Raw plugin), can convert GPR files to other formats.

For example, Adobe Lightroom can convert GPR files to the following formats:

  • .JPG - JPEG Image
  • .TIF - Tagged Image File Format
  • .DNG - Bitmap Image

Additionally, Adobe DNG Converter is a free utility that can convert GPR files to .DNG files.

Babylon Glossary Builder Project File

Developer Babylon
Popularity
2.5  |  2 Votes
 

Project file created by Babylon Glossary Builder, a program used to create custom glossaries for the Babylon application; contains glossary properties as well as terms and definitions; used to save and load glossary projects, as well as update glossaries.

More Information

Babylon Glossary Builder projects may be used to generate .BGL glossary files.

NOTE: Babylon Glossary Builder has been discontinued.
