Brick Hill Brick File

Developer Brick Hill
Popularity
4.1  |  20 Votes
 

What is a BRK file?

A BRK file is a game data file created by Brick Hill Workshop, a brick editor for the Brick Hill building block game. It contains all of the information about an element in Brick Hill, which may be one small brick structure or a whole world consisting of many bricks. BRK files store all of the brick information, scripts, and details about the environment in plain text.

More Information

Brick Hill is a free game that features gameplay similar to ROBLOX. Users can create their own custom avatars, build worlds with digital bricks, and interact with other gamers in those worlds.

Workshop is a free utility available for download on the Brick Hill website. The application enables Brick Hill users to develop their own worlds, or games, that can be uploaded to and hosted on the "node-hill" servers for other gamers to play. When a user creates and saves Brick Hill content in Workshop, the application creates a BRK file to store the information.

To create a BRK file with Workshop, select the "New file" icon, create your content, then click the "Save" floppy disk icon, and choose the save location for the BRK file.

How to open a BRK file

You can open a BRK file with Brick Hill Workshop in Windows. To open the BRK file, select the "Open file" icon, which looks like an open folder. You can also import a BRK file in another BRK file by selecting the "Import bricks" icon, which looks like an open folder with a plus (+) sign.

Since BRK files are saved in plain text you can open and edit them with a text editor, such as Microsoft Notepad (Windows) or Apple TextEdit (Mac), which come bundled with their respective operating systems.

Warning

BRK files may contain malicious scripts inserted by the creator of the BRK file. The scripts will only affect the person hosting the BRK file.

Draw Bricks Structure

Developer Bruno Sousa
Popularity
3.2  |  6 Votes
 

A BRK file may also be a structure created in Draw Bricks, a 3D brick-based building app available on Android. It contains JSON-formatted text that Draw Bricks uses to show a structure. Draw Bricks saves BRK files in the Play files/​DrawBricks/​gallery directory.

More Information

In Draw Bricks, users can create LEGO-like structures, such as houses, vehicles, and characters. If a user opens Draw Bricks' sandwich menu and selects Save, Draw Bricks saves the user's current structure as a BRK file. Each BRK file's default filename is the date and time at which it was created.

By default, Draw Bricks saves users' BRK files in the following directory:

Play files/​DrawBricks/​gallery

Users can create sub-directories within the Gallery directory, to organize their Draw Bricks structures.

How to open a BRK file

You can open BRK files in Draw Bricks (Android). To do so, place the file in your device's Play files/​DrawBricks/​gallery directory. The file will then appear in and be selectable from Draw Bricks' Open menu.

If you do not have access to Draw Bricks, or you want to see the JSON text your BRK file contains, you can also open your BRK file in any text editor, such as QuickEdit Text Editor (Android) or Google Chrome Text (Chrome).

Brooktrout Fax Document

Developer Dialogic
Popularity
1.8  |  5 Votes
 

A BRK file may also be a fax document saved in the Brooktrout Fax (BRK) format. It stores the digital copy of a fax sent via Dialogic Brooktrout fax software or hardware. BRK files often contain copies of scanned business-related documents.

More Information

Dialogic Brooktrout technology consists of fax boards and software small-to-medium businesses often use. The technology allows businesses to scan printed material, such as a receipt or financial paper document, as a BRK digital document and then send it to a telephone number or IP network.

How to open a BRK file

You can open a BRK file with Corel PaintShop Pro in Windows. To open a BRK file with PaintShop Pro, select File → Open.

How to convert a BRK file

Corel PaintShop Pro can convert a BRK file to various image file types, including the following formats:

  • .JPG - JPEG Image
  • .PNG - Portable Network Graphic
  • .BMP - Bitmap Image
  • .EMF - Enhanced Windows Metafile
  • .EPS - Encapsulated PostScript File
  • .TIF - Tagged Image File
  • .TGA - Targa Graphic
  • .WMF - Windows Metafile
