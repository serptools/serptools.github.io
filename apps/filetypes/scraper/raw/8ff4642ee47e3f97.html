What are CHM files and how to open them

Are you having problems opening a CHM file or are you simply curious about its contents? We're here to explain the properties of these files and provide you with software that can open or handle your CHM files.

What is a CHM file?

CHM files have multiple uses, and Windows Compressed HTML Help is one of them. Read more about the other uses further down the page.

Windows Compressed HTML Help File

Files with the .chm extension are most commonly associated with Microsoft help documentation files.

The CHM files are compressed to reduce the size and may consist of HTML pages and images. Often you will find an associated CHM file with software you install.

    All known file formats using extension .CHM

    While Windows Compressed HTML Help File is a popular type of CHM-file, we know of 4 different uses of the .CHM file extension. Different software may use files with the same extension for different types of data.

    4 known uses of the CHM file extension

    • Chemdraw Chemical Structure
    • DESIGN 3D Path
    • Windows Compressed HTML Help File

    Chemdraw Chemical Structure

    These files contain data describing a chemical structure. ChemDraw versions earlier than 4.0 used files with extension CHM as the default ChemDraw format. However, Microsoft started to use the same extension for compiled help files, and this caused problems as ChemDraw version 3 and earlier assigned itself as the default opener of CHM files.
