What is a TTR file?

A TTR file is a game replay created by TETR.IO, a Tetris variant available online and for desktop PCs. It contains JSON data used to show a game's statistics and play a video replay of that game. TTR files can be opened by dragging and dropping them into TETR.IO.

More Information

When a player finishes a game in TETR.IO, they can click the download button that appears beneath their final time to save their game stats and a replay of that game. TETR.IO saves the stats and replay locally as a TTR file, named replay.ttr.

Each TTR file contains JSON-formatted plain text data. This data specifies the user who played the game the replay was created from, the game's seed, the game's settings, and the inputs made during the game.

How to open a TTR file

To view the game stats and replay a TTR file contains, drag and drop the file into the PC or online version of TETR.IO. The stats the file contains will appear in TETR.IO, along with a REPLAY button that allows you to watch the game the file describes.

If you want to view the JSON data a TTR file contains, you can open it in Microsoft Visual Studio Code (cross-platform), GitHub Atom (cross-platform), or any other text editor.
