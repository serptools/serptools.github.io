    Panda3D 3D Model

    Files that contain the .egg file extension are most commonly associated with the Panda3D software application.

    Panda3D is a 3D development engine that is commonly used in game development projects. The .egg files define the transforms of a 3D model that is used within a game.

    Panda3D was originally created by Disney and the program is still used to create Disney games to this day.
