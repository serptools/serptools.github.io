Windows Installer Merge Module

Developer Microsoft
Popularity
3.8  |  12 Votes
 

What is an MSM file?

Database of a "merge module" that contains installation settings and properties for a specific software installer; includes files, resources, registry entries, and setup information; often contained within an .MSI file.

More Information

MSM files are also called "merge package files." Since they are usually part of an installer package, MSM files should not be opened manually.

Multisim Circuit Design File

Developer National Instruments
Popularity
4.2  |  5 Votes
 

Circuit design created by Multisim, a Simulation Program with Integrated Circuit Emphasis (SPICE) environment used to teach electronics theory and design and prototype circuitry; contains a design of an electrical circuit board, which includes place diodes, transistors, indicators, and power components.

More Information

To create the MSM file, select File → Save As....

The MSM file is used by the 2001 version of Multisim. As new versions of Multisim are released, new file extensions are used to save the circuit designs. For example, version Multisim 7 uses the .MS7 file extension.

My Singing Monsters Composer Song

Developer Big Blue Bubble
Popularity
3.6  |  5 Votes
 

An MSM file may also be a song file created by My Singing Monsters Composer, a music creation app for Android and iOS. It contains song data, including the notes determining how instruments will play in the composition. MSM files also store metadata, such as the song's name and creator.

More Information

Big Blue Bubble develops My Singing Monsters Composer, a mobile app that features monsters playing music. In the app, you are the maestro conducting an orchestra of monsters as you make music.

When you save your composition as a song, the app creates an MSM file to store the song data. You can also share the song with other My Singing Monsters Composer users by tapping the music notes button in the upper-left corner of the app, tapping the mail icon to the right of the song, then choosing how you want to share it (e.g., via email, cloud storage, etc.).

Common MSM Filenames

ComposerSong.msm - Default filename given to MSM files.

How to open an MSM file

You can open MSM files with My Singing Monsters Composer in Android and iOS. For example, to open an MSM file with the Android version of My Singing Monsters Composer, tap the music notes button in the upper-left corner of the app, then choose the song (provided the MSM file is located in the correct directory on the device).

UbiArt Framework Motion Data File

Developer Ubisoft
Popularity
2.0  |  1 Vote
 

An MSM file may also be a motion data file used by video games developed with the UbiArt Framework, including Rayman Origins and Child of Light. It stores animation or motion information the game references to animate game characters, objects, or visual effects appearing in gameplay. MSM files may include keyframe animation data, transformation instructions, or movement sequences tied to specific game events.

More Information

UbiArt Framework is a game engine created by Ubisoft that powers various 2D video games. The UbiArt engine reads MSM files during runtime to display smooth animations and to coordinate the motion of visual elements based on player interaction or scripted events.

NOTE: While UbiArt Framework was prominent in the 2010s, Ubisoft no longer develops it.

How to open an MSM file

MSM files are not intended to be opened or edited directly by players. Instead, the game with which the files are packaged accesses them when loading animation and motion information.

Just Dance Motion Synchronization Map

Developer Ubisoft
Popularity
2.0  |  1 Vote
 

An MSM file may also be a motion data file used by several older entries in the Just Dance series developed using Ubisoft's UbiArt Framework. It contains motion, synchronization, and mapping data that define specific dance moves, including their timing and structure. During gameplay, the engine references MSM files to determine when and how a player's movements should be evaluated, ensuring synchronization between the on-screen choreography and the player's actions.

More Information

The MSM format used in Just Dance is similar to the format used in MSM files by other UbiArt Framework titles, such as Rayman Legends and Child of Light. However, Just Dance uses a customized version of the UbiArt Framework that includes support for rhythm- and motion-based gameplay.

MSM files are integral to the game's scoring and choreography systems, with each MSM file corresponding to a particular dance move or sequence. When the game loads MSM files alongside a .WAV or .OGG song and SCD choreography script, it parses the MSM file in real-time to determine the move the player is supposed to make at the moment for instant feedback and scoring. For example, in the "Tribal Dance (Katana Version)" routine, MSM files like tribaldancealt_nunchku_pose.msm are used to represent specific poses or movements.

NOTE: MSM stands for "Motion Synchronization Map" or "Motion Sync Metadata."

How to open an MSM file

MSM files are not meant to be opened or edited directly by players. Instead, the Just Dance game with which the files are packaged accesses them when motion information.

Modders and developers working with Just Dance may attempt to modify MSM files with fan-made tools to change scoring difficulty, create custom routines for existing songs, and sync custom choreography. However, no known programs fully support the opening of MSM files.
