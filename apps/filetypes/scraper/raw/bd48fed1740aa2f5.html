What is a C2M file?

A C2M file is a game level used by Lexy's Labyrinth, an open-source, updated version of Chip's Challenge. It contains text and binary data Lexy's Labyrinth uses to load the level. The text data comprises the level's clues; the binary data comprises the rest of the level's structure. Players can create C2M files using Lexy's Labyrinth's Create tool.

More Information

Lexy's Labyrinth is a browser-based puzzle game based on the Chip's Challenge series of games. It is available to play at https://c.eev.ee/lexys-labyrinth/.

Players who want to create custom Lexy's Labyrinth levels can do so using the game's Create tool. They can then download the levels they've created as C2M files. Players can also download copies of Lexy's Labyrinth's default levels as C2M files. To do so, they select edit while playing a level and then export that level as a C2M file.

NOTE: Players can export groups of Lexy's Labyrinth levels as .ZIP archives that include a .C2G file. This allows players to save and load multiple levels at once.

How to open a C2M file

You can open a C2M file in Lexy's Labyrinth (Web). To do so:

  1. Open your web browser and navigate to https://c.eev.ee/lexys-labyrinth/.
  2. Scroll down and select the load files button.
  3. Navigate to and open your C2M file.

The file will then appear in your web browser as a playable level. To edit the level, select the edit button.
