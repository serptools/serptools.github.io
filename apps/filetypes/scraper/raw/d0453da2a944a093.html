Steam Game Backup Information File

Developer Valve
Popularity
3.9  |  35 Votes
 

What is a SIM file?

Backup information file used by Steam's backup service, which allows users to backup and restore games using DVDs or other external media; contains settings and other information for the backup, along with a corresponding .SIS file.

More Information

SIM files are used to restore a backup from .SID game data files.

SAP Tutor File

Developer SAP
Popularity
3.1  |  21 Votes
 

Tutorial created by SAP Tutor, a program used to record, edit, and play interactive self-learning units; contains a lesson, which may include screenshot images, text instructions, and assessment questions; used to train employees how to operate software programs.

More Information

SIM files may be played back using the SAP Tutor Player, which is bundled with the SAP Tutor package or available as a standalone program.

NOTE: The size of SIM files may vary depending on how many screenshot images are included in the tutorial.

SimLab Composer Scene

Developer Simulation Lab Software
Popularity
4.3  |  3 Votes
 

A SIM file contains a 3D scene created by SimLab Composer, a 3D modeling, animation, and rendering application. It stores one or more objects and the surrounding 3D environment. SIM files may also include animations, materials, lighting effects, and annotations.

More Information

SimLab Composer comes with various capabilities, including support for different types of models, 3D rendering, building VR scenes, and creating interactive VR training sessions. When saving a scene with SimLab Composer, the application creates a SIM file, which is the native SimLab Composer scene format.

To create a SIM file with SimLab Composer, select File → New, modify the scene, then select File → Save or Save As. After saving the file, you can close it, re-open it, then continue editing the scene in the file.

You can also export SimLab SIM scenes to other formats, such as .STL, .3DS, .DWF, .DWG, .OBJ, .DXF, .DAE, .3MF, .U3D, .USDZ, and .PDF. To export a SIM file with SimLab Composer, select File → Export.

How to open a SIM file

You can only open SIM 3D scene files with SimLab Composer (Windows and macOS). To open the file, select File → Open.

Aurora Image

Developer N/A
Popularity
2.6  |  8 Votes
 

A SIM file may also be a raster image saved in the Aurora image format. The Aurora format is obsolete and no longer used. Therefore, SIM files are quite uncommon, and most image viewers cannot open them.

How to open a SIM file

You can open a SIM file with XnViewMP (multiplatform). To do so, select File → Open.. from the program's menu bar. Then, navigate to and open your SIM file.

How to convert a SIM file

You can use XnViewMP to convert a SIM file to other image formats, including:

  • .JPG - JPEG Image
  • .GIF - Graphical Interchange Format
  • .PNG - Portable Network Graphic
  • .TIFF - Tagged Image File Format
  • .BMP - Bitmap Image
  • .PCX - Paintbrush Bitmap Image
  • .TGA - Targa Graphic
  • .JP2 - JPEG 2000 Core Image
  • .JXR - JPEG XR Image
  • .WEBP - WebP Image

The Sims 3 Sim File

Developer Electronic Arts
Popularity
2.8  |  4 Votes
 

A SIM file contains a "Sim", which is a virtual person created for The Sims 3, an open-ended life simulation game. It stores information about a pre-made or custom Sim person, which includes his or her personality and appearance. SIM files are typically used for sharing with other users and for backing up Sims persons.

More Information

The .SIM file is one of three main file types, along with .PACKAGE and .SIMS3PACK, used in The Sims 3 to contain custom content. SIM files are typically compressed in a .ZIP or .RAR file and need to be decompressed before being installed. There are many programs available to extract archives, such as Apple Archive Utility, Corel WinZip, and PeaZip. After you extract the SIM file from the ZIP or RAR file using a decompression program, place the file in the following directory:

  • Windows XP: C:\​Documents and Settings\​[username]\​My Documents\​Electronic Arts\​The Sims 3\​SavedSims
  • Windows Vista/7/8/10: C:\​Users\​[username]\​Documents\​Electronic Arts\​The Sims 3\​SavedSims
  • macOS: /​Users/​[username]/​Documents/​Electronic Arts/​The Sims 3/​SavedSims

NOTE: If there is no "SavedSims" folder, you can create one.

LogikSim Simulation File

Developer N/A
Popularity
2.0  |  1 Vote
 

A SIM file contains a circuit simulation created by LogikSim, an open-source program used to test and design logic circuits. It stores a circuit design, which includes wire connection points and various components.

SIM files are created and opened using LogikSim. The files are used to store all types of circuits, such as simple AND gates or complex computing systems. There is no other program that supports LogikSim SIM file type. But LogikSim is available to download for free.

Stagecast Creator Simulation

Developer Stagecast Software
Popularity
2.0  |  1 Vote
 

A SIM file may also be a simulation or video game created by Stagecast Creator, a discontinued visual programming app. It contains all the scripts and assets, such as images or sounds, needed to run the simulation or game. Those who want to edit the simulation or game a SIM file contains must open it with Stagecast Creator. Those who just want to run the simulation or game a SIM file contains can open it with Stagecast Player.

More Information

Stagecast Creator was a multiplatform app used primarily to teach children how to program. In Creator, novice programmers could create animated stories, games, and simulations. These programs were saved as SIM files.

SIM files are re-named .ZIP files, which store all the constituent files necessary to run a program. Those constituent files may be compiled scripts, image assets, audio assets, or other program resources. Notably, the files a SIM file contains do not use file extensions, and most of them are named either iMP### or MP###. Thus, determining exactly which file contains what asset can be difficult and time-consuming.

How to open a SIM file

You can open a SIM file with Stagecast Creator or Stagecast Player. Opening a SIM file with Stagecast Creator allows you to both play and edit the simulation or game the file contains. Opening a SIM file with Stagecast Players allows you only to play the simulation or game the file contains.

To open a SIM file with Stagecast Creator, start the program and select open a sim. Then, navigate to and open your SIM file.

If you want to extract the assets a SIM file contains, you can do so by renaming it to use the .zip extension (instead of .sim) and then opening it as you would any ZIP file.
