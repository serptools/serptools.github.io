Gerber Top Solder Paste Data File

Developer Ucamco
Popularity
4.2  |  9 Votes
 

What is a GTP file?

Data output file created by Printed Circuit Board (PCB) CAD tools such as Autodesk EAGLE; contains layout data for the top paste layer of the PCB for creating stencils to apply paste to the board; written in RS-274X, which is the most common and contemporary Gerber format.

More Information

The GTP file acts similar to a .PDF file. It is an exported version of the PCB layout and used to create stencils to apply paste to boards for adding surface-mount devices (SMD), like pads. It is meant to be viewed and referenced but not edited.

NOTE: The Gerber format was originally developed by Gerber Systems but is now maintained by Ucamco.

Common GTP Filenames

[projectname].gtp - The GTP file is named after the name of your PCB project.

GNOME Theme Package File

Developer The GNOME Project
Popularity
4.0  |  3 Votes
 

Theme package file used by GNOME, a program that allows users to customize their desktop settings in Linux; contains a theme_name directory that includes an index.theme file and gtk and metacity directories; saved in a gzip-compressed-tar format.

More Information

GTP files may be installed by double-clicking them in a GNOME environment.

Guitar Pro Tablature File

Developer Arobas Music
Popularity
3.0  |  3 Votes
 

A GTP file may also be a tablature (tab) data file created by Guitar Pro versions 1 and 2. It contains music score data for various stringed instruments, including guitar and bass. GTP files store MIDI data simulated by the Guitar Pro engine to play music, not actual audio data.

More Information

Arobas Music Guitar Pro is a multitrack composition program tailored to string instruments, mainly the guitar. It includes tab editing tools and converts compositions to various audio and document formats.

Guitar Pro 3 replaced the GTP file type with the .GP3 file. Beginning with version 3, Guitar Pro saves tabs as a file with an extension corresponding with the software version by default. For example, Guitar Pro 4 saves tabs as .GP4 files, Guitar Pro 5 saves tabs as .GP5 files. However, version 6 saves tabs as .GPX files.

Although Guitar Pro replaces file types with each new version, the software still supports previous Guitar Pro file types. For example, Guitar Pro 7 can open GPX, GP5, GP4, GP3, and GTP files.

NOTE: Arobas Music originally developed Guitar Pro as a tablature editor for Windows. However, it expanded the software to Mac OS X in July 2006 with version 5.

How to open a GTP file

You can open a GTP file with Arobas Music Guitar Pro (multiplatform). While the file type is old and obscure now, newer versions of Guitar Pro still support the file. To open a GTP file with newer versions of Guitar Pro, select File → Open....

You can also open a GTP file with the open-source Tuxguitar (multiplatform) and DGuitar (multiplatform) applications.
