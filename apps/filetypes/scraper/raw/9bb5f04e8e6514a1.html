Kingsoft Design Science Equation File

Developer Kingsoft
Popularity
3.4  |  67 Votes
 

What is a DSC file?

A DSC file is a settings file used by WPS Office (previously known as Kingsoft Office), an office suite which includes the Writer, Spreadsheets, and Presentation applications. It stores the settings for the Design Science Equation Editor, which includes the window size, interface style, and zoom magnification.

More Information

The DSC file is referenced by the Equation Editor which is available in the Writer, Spreadsheets, and Presentation application's ribbons. It allows the editor to be reopened by users and resume the same settings, such as the zoom magnification of the window and the size of the window that it was set to when it was previously opened.

Nikon Disk Identification File

Developer Nikon
Popularity
3.0  |  56 Votes
 

Contains information about photos taken with a Nikon digital camera (such as a CoolPix camera); saved on the memory card along with the image files; typically only a few kilobytes in size since it does not contain any image data.

More Information

DSC files may be hidden on the card since they are referenced by Nikon's digital camera software and not meant to be opened manually.

Many digital cameras save photos with filenames beginning with "DSC," but these image files typically have a .JPG extension.

NOTE: Nikon ViewNX is no longer maintained or available for download.

Debian Source Control File

Developer N/A
Popularity
3.1  |  12 Votes
 

Source file created for Debian, a free operating system; contains data describing the other files in the source package; formatted as a single paragraph consisting of a series of fields.

More Information

The DSC file is produced by the dpkg-source tool when the source archive is built and is based on the other files in the source package. When unpacked, the DSC file is referenced against the other files and directories in the source package.

The DSC file is part of the Debian source package along with an upstream tar ball file with the .TAR.GZ extension and any other files created for the Debian package. A source package is created with the dpkg-build package tool.
