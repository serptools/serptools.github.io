Fusion Flow File

Developer Blackmagic Design
Popularity
4.3  |  6 Votes
 

What is an FLW file?

Flow file created by Fusion, an advanced visual effects compositing application; saves a project, which includes frame, camera angle, image, and color channel information.

More Information

NOTE: The FLW file was replaced by the .COMP file in Fusion 5.

Kivio Flow Chart

Developer theKompany
Popularity
2.0  |  1 Vote
 

An FLW file contains a flow chart created by Kivio, an open source Linux program used to create flow charts and other types of diagrams. It stores data organized into interconnected objects that outline a process, such as instructions for creating a document or fixing a machine. FLW files are saved in XML format.

More Information

FLW files can only be opened by Kivio, which is included with KOffice. When KOffice development ceased in 2010, Kivio was merged into Calligra Flow, which is part of the Calligra Suite. Unfortunately, support for FLW files was dropped when Kivio was integrated into Calligra Flow.
