What are KML files and how to open them

Are you having problems opening a KML file or are you simply curious about its contents? We're here to explain the properties of these files and provide you with software that can open or handle your KML files.

What is a KML file?

KML files have multiple uses, and Google Earth Placemark is one of them. Read more about the other uses further down the page.

Google Earth Placemark File

KML stands for Keyhole Markup Language. Keyhole Inc. initially developed the KML file format for the Keyhole mapping application. Google acquired the Keyhole company and its software in 2004. The KML file format is now used primarily by the Google Earth application, although other mapping programs have been known to use this file format as well.

KML files use an XML format to store geographic modeling information. These files contain lines, points, images, and polygons. They are used to label locations and create overlay textures and camera angles for the maps created by the mapping applications.
