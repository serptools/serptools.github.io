Grave Composer Music Module

Developer N/A
Popularity
4.0  |  25 Votes
 

What is a WOW file?

A WOW file is a music module created by the Grave Composer (also known as "Mod's Grave") audio tracker, a tracking program for DOS-based PCs. It contains note patterns and sound samples and supports up to 8 channels. WOW files are similar to the Amiga Module (.MOD) format.

More Information

Grave Composer is an old music-tracking program that allowed MS-DOS users to create music. It is discontinued and no longer available for download.

Since the software is no longer in use, WOW files are rare. However, music-tracking enthusiasts may have old WOW files on their computers or share WOW music modules they created previously with the Grave Composer.

How to open a WOW file

You can open a WOW file with OpenMPT (Windows) by selecting File → Open. Once you open the WOW file with OpenMPT, the program converts it to a supported modern module format, such as OpenMPT (.MPTM) or MIDI (.MID).

How to convert a WOW file

OpenMPT can convert WOW files to the following formats:

  • .MPTM - OpenMPT Module
  • .MID - MIDI File
  • .IT - Impulse Tracker Module
  • .WAV - WAVE Audio File
  • .FLAC - Free Lossless Audio Codec File
  • .AU - Audacity Audio File
  • .OPUS - Opus Audio File
  • .OGG - Ogg Vorbis Audio File
  • .RAW - Raw Audio Data
  • .MP3 - MP3 Audio File

SparkMaker 3D Print File

Developer WOW Innovation Technology
Popularity
3.3  |  4 Votes
 

A WOW file contains instructions for printing a 3D model with a SparkMaker 3D printer. WOW files are created by Spark Studio and saved on an SD card.

More Information

The Spark Studio software accepts .STL files, and allows the user to make adjustments and set printing parameters. The software then slices the model and exports it to a WOW file for use with the printer. The file to be printed must be named print.wow.

NOTE: SparkMaker printer firmware update files also use the WOW extension.

Common WOW Filenames

print.wow - A file that contains a printable 3D model for a SparkMaker 3D printer.
