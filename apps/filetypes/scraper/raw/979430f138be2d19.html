What is an MPTM file?

An MPTM file is a music module file created by OpenMPT, an open-source audio module tracker. It is saved in the OpenMPT Module (OMTM) format, which is based on the Impulse Tracker (.IT) format but includes extended capabilities. MPTM files store note patterns, sound samples, sequences, and other audio data.

More Information

OpenMPT is short for Open ModPlug Tracker and is based on the original ModPlug Tracker. Olivier Lapicque initially developed the software and released it in 1997, then released its code as open source. A group of programmers and musicians now develop the software.

The application supports various sample and instrument audio formats such as .MDL, .STM, WAV, and PAT, but saves modules in its native MPTM format by default. You can create an MPTM by selecting File → New → OpenMPT Module, modifying the module, then selecting File → Save or Save As.... After saving the MPTM file, you can close the module, and re-open it for further editing.

How to open an MPTM file

You can open an MPTM file with OpenMPT in Windows. To open an MPTM file, select File → Open....

How to convert an MPTM file

OpenMPT can convert MPTM files to the following formats:

  • .MID - MIDI File
  • .IT - Impulse Tracker Module
  • .WAV - WAVE Audio File
  • .FLAC - Free Lossless Audio Codec File
  • .AU - Audacity Audio File
  • .OPUS - Opus Audio File
  • .OGG - Ogg Vorbis Audio File
  • .RAW - Raw Audio Data
  • .MP3 - MP3 Audio File
