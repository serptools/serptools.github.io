3hs Theme Extensible

What is an HSTX file?

An HSTX file is a theme used by 3hs, an app for hacked Nintendo 3DSes that allows gamers to install hShop content. It contains images and color settings used to change 3hs's appearance. 3hs users create HSTX files with the online 3hs theme creator.

More Information

HShop is an online Nintendo 3DS content repository that allows players to download 3DS games, DLC, apps, extras, and themes. While gamers can download and manually install content from hShop, many instead download content using 3hs, an associated 3DS app. 3hs can be installed only on 3DSes that have been hacked to use custom firmware.

Optionally, players can change 3hs's UI theme using HSTX files. Each HSTX file contains images that serve as various background and icon images, and color settings that determine various UI elements' colors. Players can install any number of HSTX files in 3hs's 3hs/​themes directory. Then, they can select those files from the app's theme selection menu.

How to create an HSTX file

To create an HSTX file, 3hs users use the theme creator available at https://hshop.erista.me/3hs/themegen. After they've finished naming, uploading images to, and specifying colors for their theme, they click the Create theme button to download an HSTX file that contains their theme's settings.

Common HSTX Filenames

Theme.hstx - The default name 3hs theme creator assigns to HSTX files.

How to open an HSTX file

You can open an HSTX file with 3hs theme creator (Web). To do so, click the Choose File button that appears in the Load existing HSTX: section. Then, navigate to and open your HSTX file.

To use an HSTX file with 3hs, place it in the following directory on your Nintendo 3DS's SD card (with 3hs installed):

sd:/​3ds/​3hs/​themes

Then, select the theme from 3hs's theme selection menu.
