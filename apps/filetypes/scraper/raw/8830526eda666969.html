Cryo Interactive APC Audio

The APC file is an audio file format used in games created by Cryo Interactive Entertainment. The audio file often comes in a compressed form. It can be mono or stereo and supports different sample rates. A sample rate is the measurement of the number of audio samples per second and is measured using hertz or kilohertz.

APC files can be converted into other traditional audio file formats which can be played by a majority of audio players.
