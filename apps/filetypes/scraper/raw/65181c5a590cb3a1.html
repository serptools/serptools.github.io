.ACR File Extension

4 file types use the .acr file extension.

  • 2.Audio Comparer Results File
  • 3.Acer eRecovery Management Backup File

ACRobot Script

Developer Acr-Soft
Popularity
3.6  |  11 Votes
 

What is an ACR file?

Script created with ACRobot (AutoCursorRobot), a program that creates scripts for mouse and keyboard actions; used for automating common tasks and speeding up repetitive jobs.

More Information

ACR files are created using ACRobot's built-in scripting language. However, they are recorded based on the user's actions and do not require any manual programming.

Audio Comparer Results File

Developer Bolide Software
Popularity
4.0  |  6 Votes
 

File generated by Image Comparer, a program used for identifying duplicate audio files on the hard drive; saves the results of comparing audio files (i.e., songs and music) with each other; enables users to review the comparisons later without having to re-process them again.

Acer eRecovery Management Backup File

Developer Acer
Popularity
2.8  |  6 Votes
 

Full system backup created with Acer's eRecovery Management tool, a backup utility included with Acer computers; contains the entire contents of a hard disk, including both the system and user files.

More Information

ACR files can be burned to a CD or DVD and can be used to restore an entire system using the eRecovery Management program.

American College of Radiology File

Developer National Electrical Manufacturers Association
Popularity
2.0  |  4 Votes
 

Medical image file format used by the National Electrical Manufacturers Association (ACR-NEMA); stores an image taken by radiological equipment; used to store patient records in a digital format.

More Information

The .DCM file format is more commonly used to store radiological images.
