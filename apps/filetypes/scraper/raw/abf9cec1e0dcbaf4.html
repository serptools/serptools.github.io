What is a 1TM file?

A 1TM file is an audio module created by 1tracker, a music tracker for Windows and Linux. It contains plain text information that defines the song the module contains, the instrument(s) used to play the song, and various song metadata. When opened in 1tracker, 1TM files appear as a grid that shows a song's channels and notes.

More Information

1tracker is a chiptune music tracker developed by Shiru (aka shiru8bit) of the website Shiru's Stuff. It allows users to create and export musical tracks that sound like 1-bit video game music.

Songs created in 1tracker are saved as 1TM files. The song a 1TM file contains can consist of up to nine channels, used to play notes with different instruments. Labels and breaks may appear at the beginning and end of various parts of the song, to denote different sections (such as an intro, verse, or chorus).

How to open a 1TM file

You can open a 1TM file with 1tracker (Windows, Linux). To do so, place the file in the same directory as 1tracker. Then, open 1tracker and press F3 to load your 1TM file.

You can also open a 1TM file with Microsoft Visual Studio Code (multiplatform) or any other text editor, to view the plain text data it contains.

How to convert a 1TM file

You can use 1tracker to convert the song a 1TM file contains to several formats, including:

  • .AY - AY Chiptune
  • .TAP - Commodore 64 Cassette Tape Image
  • .ASM - Assembly Language Source Code
