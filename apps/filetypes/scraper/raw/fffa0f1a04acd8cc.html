What is a Z64 file?

A Z64 file is a video game ROM image created from a Nintendo 64 (N64) game. It contains an exact copy of the data stored on an N64 game cartridge, including game data and save files. Z64 files allow users to play N64 games on their PC, using N64 emulation software.

More Information

N64 ROMs can be saved in a variety of file formats, including Z64, .N64, .JAP, and .V64. ROMS with the Z64 extension were likely created using Mr. Backup Z64, a device that allows players to create backup copies of their N64 games.

How to open a Z64 file

You can open and play a Z64 file using a Nintendo 64 emulator, such as Project 64 (Windows), Mupen64Plus (multiplatform), or N64oid (Android).
