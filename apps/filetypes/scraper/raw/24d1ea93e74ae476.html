Shockwave Media File

Developer Adobe Systems
Popularity
4.0  |  517 Votes
 

What is a DCR file?

A DCR file is an interactive media file saved in the Shockwave format for distribution on the web. It is often used for creating web games and other highly interactive content.

More Information

DCR files are similar to Flash (.SWF) files, but created with Adobe Director rather than Adobe Flash. Adobe Shockwaver Player can also open the file but runs as a web browser plug-in.

NOTE: Director was originally developed by Macromedia, which was acquired by Adobe at the end of 2005.

Kodak Digital Camera RAW Image

Developer Kodak
Popularity
3.8  |  216 Votes
 
NEW: Open your .DCR file online with File Helper.View Online

A DCR file may also be an image captured in the Kodak Digital Camera RAW (DCR) format by various Kodak digital cameras, such as the DCS 760M and DCS Pro Back Plus. It stores uncompressed or lossless image data exactly how the camera's sensor captured it. DCR files also contain metadata, such as the camera model name, sensor image dimensions, and focal length.

More Information

DCR is one of several proprietary camera raw formats developed by Kodak. Others include Kodak K25 (.K25), Kodak Photo-Enhancer (.KDC), and Kodak DCS200 Camera Raw (.KC2).

Users that will encounter DCR files

Professional and hobbyist photographers who own an older Kodak digital camera that captures images in the DCR format are most likely to encounter DCR files. For example, after shooting digital photographs in RAW mode, they may transfer their images as DCR files to their computer to view and edit them with software that supports the format.

How to open a DCR file

You can open DCR files with various image-editing and -viewing applications, including Microsoft Photos (with the Raw Image Extension installed), which is bundled with Windows and Apple Preview, which is bundled with macOS.

Other applications that support the camera RAW format are Adobe Photoshop (Windows and macOS), Adobe Photoshop Elements (Windows and macOS), ACD Systems ACDSee (Windows), and Adobe Photoshop Express (Android and iOS).

How to convert a DCR file

Various applications, including Adobe Photoshop and ACD Systems ACDSee, can convert DCR files to other formats.

For example, Photoshop can convert DCR files to many file types, including the following formats:

  • .DNG - Digital Negative Image
  • .JPG - JPEG Image
  • .PNG - Portable Network Graphic
  • .TIF - Tagged Image File Format
  • .GIF - Graphical Interchange Format
  • .WEBP - WebP Image
  • .BMP - Bitmap Image
  • .PSD - Adobe Photoshop Document

Delphi Component Resource File

Developer Embarcadero Technologies
Popularity
3.9  |  77 Votes
 

A DCR file may also be a component resource file created and used by Delphi, a Windows, OS X, Android, and iOS IDE. It contains a compiled resource, which includes a bitmap image that is used as a component icon for the component palette. DCR files should not be deleted.

More Information

The DCR file is the same as the binary .RES file generated by Delph. However, it is used to store the component icon instead of the application's main icon.

NOTE: The Delphi IDE includes an image editor that can open and create DCR files

Liberty Video Recording File

Developer High Criteria
Popularity
3.9  |  67 Votes
 

Video file created by High Criteria's Liberty video recording software products, such as Liberty Court Recorder, Liberty Police Interview Recorder, and Liberty Meeting Recorder; used for storing audio and video recorded in courtrooms, police interviews, and city council meetings.

More Information

DCR files can be played back using the free Liberty Court Player, which is available for Mac, Windows, iOS, and Android platforms.

NOTE: The DCR format used by Liberty software differs from the DCR format used by BIS Digital Digital Court Recorder, even though the two products are often used for the same purpose.
