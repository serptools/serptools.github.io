What is an OFX file?

An OFX file is a financial data file created in the Open Financial Exchange (OFX) format, an open format for transferring data between vendors, consumers, and financial systems. It contains transactions, statements, and other financial information. OFX files are typically sent to individual users as credit card or banking statements from a financial institution.

More Information

OFX is an SGML-based file format standardized in 1997 by Intuit, Microsoft, and CheckFree. It replaced the Open Financial Connectivity (.OFC) format used by Microsoft Money and is now the standard container for financial information under the Open Financial Exchange Consortium.

OpenFX Plugin

Developer OpenFX
Popularity
3.0  |  1 Vote
 

An OFX file may also be a video editing plugin saved in the OpenFX (OFX) standard that extends the capabilities of video editing apps, such as DaVinci Resolve, NUKE, and VEGAS Pro. It may store visual effects, lens flares, transitions, editing tools, workflow enhancements, and other types of features.

More Information

The OFX format is widely used for enhancing video projects in many professional video editing environments. For example, a video editor may install OFX plugins included in BORIS FX's Sapphire visual effects suite to their MAGIX Vegas Pro application to add lighting and lens flare effects to their video. Another example is installing an OFX plugin to DaVinci Resolve to add 3D motion blur transitions to a video project that are not packaged with the original video editor installation.

How to open an OFX file

You can open OFX files with various video editors in Windows, macOS, and Linux. Some well-known video editors that support OFX plugins include Blackmagic Design DaVinci Resolve (cross-platform), MAGIX VEGAS Pro (Windows), and The Foundry NUKE (Windows).
