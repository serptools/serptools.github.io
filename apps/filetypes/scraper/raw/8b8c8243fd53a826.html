What is a BTSEARCH file?

A .BTSEARCH file is a BitTorrent Search Engine Specificiation file.

The BitTorrent peer-to-peer file-sharing applications use the .btsearch extension. The BTSEARCH files control how the BitTorrent client searches for a torrent on a particular P2P search engine. This allows the search engines, such as Google, to be added to a user's built-in torrent search bar. The BTSEARCH files contain the name, URL, and description of the search engine that is being added.
