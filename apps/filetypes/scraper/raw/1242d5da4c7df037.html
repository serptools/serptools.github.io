What are INT files and how to open them

Are you having problems opening an INT file or are you simply curious about its contents? We're here to explain the properties of these files and provide you with software that can open or handle your INT files.

What is an INT file?

INT files have multiple uses, and Silicon Graphics (SGI) Image is one of them. Read more about the other uses further down the page.

Silicon Graphics (SGI) Image File

The .int file extension is commonly associated with graphics files in the Silicon Graphics image format. The INT files are used to store images with integers that represent the contents of each image pixel. INT files can be compressed or uncompressed and may contain color or monochrome images.
