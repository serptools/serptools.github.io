Picture Publisher Image File

Developer Corel
Popularity
3.8  |  24 Votes
 

What is a PPF file?

Image file created by Corel Picture Publisher, a program previously owned by Micrografx; contains a bitmap image; can be created and edited with various text, drawing, and painting tools in Picture Publisher.

More Information

PPF files are created by Picture Publisher versions 6, 7, 8, 9, and 10. Previous Picture Publisher versions use .PP4 and .PP5 files. Picture Publisher is no longer supported, but PPF files can still be opened by CorelDRAW Graphics Suite.

PlayStation Patch Format File

Developer N/A
Popularity
3.5  |  6 Votes
 

Patch file format used mainly for updating video game ROMs; originally developed for updating PlayStation games, but can also be used to patch NES, SNES, N64, and PSP games; may be created with PPF Studio and opened with PPF-O-Matic.

More Information

PPF files are similar to .IPS patch files, but are not limited to patching the first 16MB of a file. Therefore, PPF files can also be used to patch ISO disc images, such as CDs and DVDs.

Programs that open PPF files

Windows
PPF Studio
PPF-O-Matic
Category: Game Files
Updated: August 8, 2007

Edgecam Pathtrace Part File

Developer Planit Software
Popularity
3.6  |  5 Votes
 

Part file created by Edgecam, a computer-aided manufacturing (CAM) program used to design machine components; contains a part or component model, including its geometry, toolpaths, numerical control (NC) data, machine code, display textures, material specifications, and other part properties.

More Information

PPF files are used to save and load part project information. They may be used to generate automated machine processes.

Paint Shop Pro Soft Plastic Preset File

Developer Corel
Popularity
3.3  |  3 Votes
 

A PPF file may also be a preset file used by older versions of Corel Paint Shop Pro, a graphics-editing application. It contains information that defines a soft plastic texture, which users apply to images or graphic elements to create a plastic-like visual effect. Paint Shop Pro users typically add PPF effects when designing elements that require a glossy, plastic appearance, such as buttons and logos.

More Information

While modern versions of PaintShop Pro no longer support PPF files, you can find PPF files in the "Presets" folder within the installation directory of earlier Paint Shop Pro versions. You could also access and apply these presets through the software's effects menu to save time while ensuring a uniform look across your work.

NOTE: Corel rebranded Paint Shop as "PaintShop" in January 2010 when it released version X3 (13.00).

How to open a PPF file

Corel PaintShop Pro no longer supports PPF files.

Older versions of Corel Paint Shop Pro

Archicad Teamwork Project Preferences File

Developer GRAPHISOFT
Popularity
2.5  |  2 Votes
 

A PPF file contains preferences for a teamwork project (.PLP file) created with GRAPHISOFT Archicad architectural designing software. It stores preferences, including standards and working methods applied to the project, to maintain consistency across all Archicad users working on the project.

More Information

GRAPHISOFT Archicad is a building information modeling (BIM) application for Windows and macOS users. The primary users of the software are architects designing the appearance of buildings, including their interiors, exteriors, and landscapes.

Examples of preferences stored in a PPF file include dimensions, calculation units and rules, zones, and reference levels. PPF files also contain preferences regarding teamwork access.

How to open a PPF file

You can only open PPF files with GRAPHISOFT Archicad in Windows and macOS. To modify the preferences in the PPF file, you must hold access rights and reserve the "Project Preference" dialog box.

PopCap Games Particle Effect

Developer PopCap Games
Popularity
1.0  |  1 Vote
 

A PPF file may also be a particle effect that is applied to a PopCap Game Animation (.PAM) file. It is used to more precisely render a cloud, fire, explosion, fluid, or other particle-based object's movement in a PopCap video game. PPF files are not meant to be opened by players.

More Information

Many PopCap video games, including Plants Vs. Zombies 2 and Peggle 2, use PPF files to apply additional effects to PAM animations. In many games, PPF files are stored within .RSB asset archives. You may also see PPF files referred to as PIEffect files or PopAnim Image Effect files.

How to open a PPF file

There is currently no way to open or edit PPF files outside the PopCap games that use them.

Programs that open or reference PPF files

All Platforms
Various PopCap games, including Plants Vs. Zombies 2, Peggle 2, and Zuma's Revenge!
