Korg Trinity Song File

Developer Korg
Popularity
3.9  |  15 Votes
 

What is an SNG file?

Song file used by Korg MIDI-compatible devices, such as some synthesizers; stores musical notes and other audio information in a proprietary format; can be used for playing back songs and music loops on Korg electronic devices; may also be converted to other file formats, such as .MID.

MIDI Song

Developer N/A
Popularity
3.4  |  30 Votes
 

An SNG file contains MIDI song data, saved by one of various MIDI-compatible music trackers. Typically, SNG files are saved in a format similar to a standard .MIDI or .MID file. However, the SNG format varies slightly from program to program, so programs may be able to open some SNG files and not others.

More Information

Many music trackers released in the 1980s save songs that users create as SNG files. These files contain MIDI data, rather than actual audio. The music tracker that created the file can load the SNG file to play the song it defines, using various instrument samples.

Some music trackers that create SNG files include:

  • Adlib Tracker 1
  • DeskMate Music
  • Faust Music Creator
  • Soundtrakker 1
  • The Music Studio
  • GoatTracker

Often, other MIDI-compatible devices can play SNG files. But in some cases, only the program that created an SNG file can play the song it defines.

How to open an SNG file

Because the SNG format varies between programs, it's best to open your SNG file in the program that created it. However, if you no longer have access to the program that created your SNG file, you can try to open the file with VideoLAN VLC media player or Audacity. These programs can play standard MIDI files, and so might be able to play your SNG file.

n-Track Studio Song Project

Developer n-Track
Popularity
3.0  |  2 Votes
 

An SNG file may also be a song project created by n-Track Studio, a DAW for recording, editing, and mixing audio. It contains information about a project, including recorded or imported audio tracks on the timeline, workspace layout, and audio settings. Users create and open SNG files with n-Track Studio to produce songs from recorded audio, samples, and beats.

More Information

n-Track Studio is a professional-level DAW with various song production features, including a piano roll editor, step sequencer, DEnoiser for reducing background noise, and vocal and instrument modifying tools. As you produce a song with these features, you can save your progress as an SNG song project file.

For example, you can create an SNG file with the n-Track Studio in Windows or macOS by selecting File → Save or Save As. After saving the Song Project, n-Track Studio creates a folder with the same name as the song project (e.g., n-Track Studio saves the AudioProject.sng file in the "AudioProject" folder). In the folder, you will find the SNG file accompanied by a .DB database file and an "Audio" folder containing the WAV files the SNG file points to that store the audio imported in the project.

SNG files store imported audio separately to prevent them from ballooning in size. For example, if you import an .MP3 file into your song, n-Track Studio creates a WAV file that stores the audio, and the SNG file contains a reference to that WAV file. If you want to save the audio files imported into a project with the song project file, save your project in the .SGW Packed Song Project format (this is helpful when transferring or backing up projects).

NOTE: n-Track Studio also saves templates as SNG files. The software comes with SNG templates, and you can create them by selecting File → Export → Save As Template.

How to open an SNG file

You can open SNG files with n-Track Studio on various platforms, including Windows, macOS, Android, and iOS. For example, to open an SNG file with the Windows version of n-Track Studio, select File → Open.

How to convert an SNG file

n-Track Studio can convert SNG files to various file types, including the following formats:

  • .WAV - WAVE Audio File
  • .AIF - Audio Interchange File Format
  • .MP3 - MP3 Audio File
  • .WMA - Windows Media Audio File
  • .OGG - Ogg Vorbis Audio File
  • .M4A - MPEG-4 Audio File
  • .SGW - n-Track Studio Packed Song Project
