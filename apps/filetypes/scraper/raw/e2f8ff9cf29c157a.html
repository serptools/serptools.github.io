What is a TEE file?

A TEE file is a template created by TeeChart, a charting library used by programmers to visualize large sets of data. It contains chart settings, which are used to reproduce a chart when imported. TEE files include plot axis settings, background gradient color, and mouse zooming configuration.

More Information

TeeChart charts may be exported to a variety of image and data formats, including JPEG, BMP, PNG, GIF, XML, HTML, Excel, and PDF. The TEE file is saved in a a proprietary TeeChart format and is primarily used for sharing settings of a chart that you want to apply to other charts. You can view the settings of a chart when the TEE file is open and selecting Plot → Chart Settings for Active Plot.

NOTE: TeeChart is available for VCL/FMX, .NET, Java, PHP AX, and HTML5 software.
