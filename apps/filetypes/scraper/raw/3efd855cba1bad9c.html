Musepack Compressed Audio File

Developer Musepack
Popularity

What is an MPC file?

Audio file compressed with Musepack audio compression; based on MPEG-1 Layer-2 (.MP2) algorithms, but with several improvements; not as popular as .MP3 compression, but recognized as having higher-quality sound.

Electronic Arts MPCh Video File

Developer Electronic Arts
Popularity
2.7  |  3 Votes
 

An MPC file may also be a video file saved in the Electronic Arts MPCh format, which is used by various PlayStation 2 games, such as FIFA 2007 and Need for Speed: Underground. It is typically used to store title and logo sequences, opening movies, and in-game cutscenes.

More Information

MPC files are uncommon since they are only packaged with PlayStation 2 games. You will most likely only encounter them if you dump the resource files from a game onto your computer or if someone shares an MPC video with you that was dumped from a game.

NOTE: .MAD video files are also utilized in various Electronic Arts video games for PlayStation 2.

How to open an MPC file

You can open MPC files with VideoLAN VLC media player in Windows, macOS, or Linux. To open an MPC file, select Media → Open File... or drag and drop the MPC file onto VLC media player.

WWE 2K Moveset Pack

Developer Yuke's
Popularity
1.0  |  1 Vote
 

An MPC file may also be a moveset pack used by a game in the WWE 2K series. It contains animations used to animate a wrestler's moves. MPC files contain animations for both the wrestler performing a move and the wrestler being acted upon.

More Information

WWE 2K is a series of pro wrestling video games. Many games in the series store the animations used to animate wrestlers' moves in MPC files. These files contain the individual animations used for each of the moves included in a wrestler's movesets. For this reason, they are sometimes referred to as moveset packs.

MPC files are often stored in WWE 2K Game Data Archives (.PAC files). They are often linked with associated PARAM files.

How to open an MPC file

MPC files are not meant to be opened. Various games in the WWE 2K series load and use MPC files to show wrestling move animations.
