Steam Application Cache File

Developer Valve
Popularity
4.0  |  41 Votes
 

What is an ACF file?

An ACF file is a cache file used by Valve Steam, a software package used for downloading games. It saves information related to the downloaded software and is used for the original download as well as game updates.

More Information

ACF files are used by newer versions of the Steam content delivery system. Older versions use .GCF and .NCF files.

X-Plane Aircraft File

Developer Laminar Research
Popularity
3.6  |  16 Votes
 

Aircraft file used by X-Plane, a realistic flight simulator for pilots and simulator hobbyists; contains data that describes the properties of the aircraft, such as engine power, flight physics, mass properties, and limits.

More Information

You can create an ACF file using Plane-Maker, which is a utility that comes with X-Plane. However, Plane-Maker does not enable you to create a 3D model, it is only used to determine the layout of basic aerodynamic shapes and properties of the aircraft. You can use the AC3D application with the X-Plane plugin to create a 3D model in the X-Plane OBJ format.

NOTE: Before version 10 of X-Plane, the ACF file contained aircraft data in binary format. It now contains data in text format.

Adobe Photoshop Custom Filter File

Developer Adobe Systems
Popularity
3.0  |  11 Votes
 

Convolution kernel file created by Adobe Photoshop, an image-editing program; contains a NxN kernel matrix of values used to manipulate the neighboring pixels surrounding a given pixel; can contain positive and negative values as well as mathematical expressions.

More Information

Convolution kernel matrixes can be used for blurring, sharpening, embossing, or performing other raster graphic operations to an image; once saved in an ACF file, they can be repeatedly applied to images.

Google Earth Pro Aircraft

Developer Google
Popularity
3.5  |  2 Votes
 

An ACF file may also be a plain text file used to simulate an aircraft in Google Earth Pro's flight simulator. It defines the aircraft's name, description, and properties. By default, Google Earth Pro contains two ACF files: one use to simulate an F-16 and another used to simulate an SR22.

More Information

Google Earth Pro is the desktop PC version of Google Earth. It is primarily used to view satellite imagery and terrain maps, but it also includes many other features. One of those additional features is a flight simulator that allows you to simulate flying an airplane over the Earth.

By default, Google Earth Pro allows you to simulate flying an F-16 and an SR22. The settings that define how these aircraft fly are stored in ACF files. Google Earth Pro's ACF files are stored in its ~/​flightsim/​aircraft directory.

You cannot add additional ACF files to Google Earth Pro. However, you can copy, modify, and replace the app's existing ACF files, to modify how each simulated aircraft flies.

Common ACF Filenames

F16.acf - The file Google Earth Pro uses to simulate flying an F-16.

SR22.acf - The file Google Earth Pro uses to simulate flying an SR22.

How to open an ACF file

If you want to view or edit the data an ACF file contains, you can open it with Microsoft Visual Studio Code (multiplatform) or any other text editor. If you want to use an ACF file with Google Earth Pro (multiplatform), you must replace one of the app's two existing ACF files.
