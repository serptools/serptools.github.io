AudioCuentos Audiobook

Developer Salvat
Popularity
4.4  |  5 Votes
 

What is an SMP file?

An SMP file is an encrypted .MP3 file that can be found on a memory card used by a Salvat AudioCuentos or Märchenheld Sammlung audio player. It contains a children's audiobook. SMP files can be played only by AudioCuentos and Märchenheld audio players.

More Information

Both Salvat AudioCuentos and Märchenheld offer a series of children's audiobooks that are played by placing a figurine atop a square, blue audio player (sometimes referred to as an audio cube). The audio player contains a memory card that stores a number of SMP files, each of which contains an audiobook. When a child places a figurine atop the audio player, the player loads and plays that figurine's associated SMP file.

In most circumstances, those who use AudioCuentos and Märchenheld audiobooks should never have to encounter or manually open an SMP file. The only way you would encounter an SMP file is if you removed the memory card from an AudioCuentos or Märchenheld audio player and connected it to your computer.

How to open an SMP file

You cannot open or play an SMP audiobook file on a computer or mobile device. You can, however, convert an SMP file to an MP3 file using audiocube.py (cross-platform) or SMP2MP3 (Windows). Then, you can play the MP3 file in most media players.

The only way to natively play an SMP file is to place it on a memory card, place the memory card within an AudioCuentos or Märchenheld audio player, and then place the SMP file's associated figurine atop the player.

SmilePlant Project Data File

Developer Stellar Solutions
Popularity
3.6  |  7 Votes
 

An SMP file may also be a data file created by SmilePlant, a Computerized Maintenance Management System (CMMS) that helps plant administrators with supervision, management, and maintenance. It stores information about plant equipment, employees, and production and storage areas. SMP files are saved as one of a set of files in the SmilePlant project folder.

More Information

SmilePlant has been discontinued and is no longer maintained.

ScrapeMap File

Developer skyLyn Applications
Popularity
3.7  |  6 Votes
 

An SMP file may also be a ScrapeMap created by skyLyn Applications ScrapeMate, a program used to scrape information from websites and organize it in a database or spreadsheet. It stores information about the targeted data and how it should be organized and stored after it is scraped.

More Information

ScrapeMate is an application that makes it easier to collect information from websites and save it in a format you can view and modify. Unfortunately, skyLyn Applications no longer develops the software.

How to create an SMP file with ScrapeMate

You can create a ScrapeMap with ScrapeMate using the ScrapeMap Explorer pane. First, you choose the website to scrape, filter and sort the data you want to target, click the "Create ScrapeMap" icon, and choose the save location of the file. ScrapeMate then saves the ScrapeMap as an SMP file.

You will also be prompted to choose the output format of the data scraped from the website. For example, you can choose to export the data to "Excel" (.XLSX file), "CSV" (.CSV file), or "SQL" (.SQL file).

How to open an SMP file

You can open a ScrapeMap SMP file with ScrapeMate via the ScrapeMap Explorer pane. In the pane, click the "Open ScrapeMap" folder icon and navigate to the location of the SMP file.

SampleVision Audio Sample Format

Developer Turtle Beach Systems
Popularity
2.5  |  4 Votes
 

An SMP file may also be an audio file created by Turtle Beach SampleVision, a now-discontinued application for managing digital audio samples. It stores individual audio samples in a mono, 16-bit format and supports loop points, allowing audio sections to repeat seamlessly (especially helpful for creating background music or sound effects in digital compositions).

More Information

Turtle Beach SampleVision was a popular application in the 1990s for managing and transferring digital audio samples between a computer and compatible MIDI samplers. Producers and musicians working with Turtle Beach SampleVision were the most likely to use SMP files. However, since Turtle Beach Systems no longer develops the software and format, SMP files are now obscure.

How to open an SMP file

You can open SMP files with FMJ-Software Awave Studio in Windows. To open an SMP file with Awave Studio, select File → Open.

How to convert an SMP file

FMJ-Software Awave Studio can convert SMP files to various file types, including the following formats:

  • .AIF - Audio Interchange File Format
  • .WAV - WAVE Audio
  • .IFF - Interchange File Format
  • .OGG - Ogg Vorbis Audio
  • .MP3 - MP3 Audio
  • .FLAC - Free Lossless Audio Codec

Xionics SMP Image

Developer Xionics Document Technologies
Popularity
2.3  |  3 Votes
 

An SMP file may also be a black-and-white image created by Xionics Document Technologies fax software. Typically, SMP files contain an image of a faxed document.

How to open an SMP file

You can open an SMP file with XnViewMP. To do so, select File → Open... from XnViewMP's menu bar, and then navigate to and open your SMP file.

How to convert an SMP file

You can use XnViewMP to convert SMP files to the following image formats (and several others):

  • .JPG - JPEG Image
  • .PNG - Portable Network Graphic
  • .GIF - Graphical Interchange Format
  • .TIFF - Tagged Image File Format
  • .BMP - Bitmap Image
  • .WEBP - WebP Image

PhotoImpact Gallery

Developer Ulead
Popularity
2.0  |  2 Votes
 

An SMP file may also be a gallery file created by Ulead PhotoImpact (now Corel PhotoImpact), a program used to edit photos and digital art. It contains a gallery, which may include images, objects, image enhancements, or text/animation effects. SMP files also include gallery properties such as the number of tab groups, number of thumbnails, and size of the file.

More Information

Galleries allow you to store different effects, enhancements, or images then apply them to your artwork. The SMP file is created when you select Create from the Gallery Manager in "EasyPalette", name the file, choose the save location, and click OK. You can also create an SMP file by selecting Export. You can load an SMP Gallery file by selecting Load... in the Gallery Manager.
