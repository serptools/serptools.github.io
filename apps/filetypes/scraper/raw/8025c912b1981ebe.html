What is an RGSS2A file?

Encrypted file created by RPG Maker VX (RPGVX), a program used to create original role-playing games (RPGs); contains all data for the game, including graphics, audio, items, characters, and maps; used for protecting original game material with encryption.

More Information

RGSS2A files are created by selecting File → Compress Game Data... and checking the "Create Encrypted Archive" box.

NOTE: RGSS2A and .RGSSAD files are used by RPG Maker VX, whereas RPG Maker VX Ace uses the .RGSS3A file extension.

Common RGSS2A Filenames

game.rgss2a - Default name of the file.
