.ADX File Extension

2 file types use the .adx file extension.
  • 1.Approach Index File
  • 2.ADX Audio

Approach Index File

Developer IBM
Popularity
4.1  |  14 Votes
 

What is an ADX file?

An ADX file contains indexing information for a Lotus Approach relational database. It stores structured data to keep track of record locations. ADX files make the retrieval of specific data faster and more efficient, which decreases the time required for searches and query responses within a database.

More Information

Lotus Approach is a relational database management system that is included with the Lotus SmartSuite and was widely used in the 1990s. It has since become obscure, and you will likely only encounter ADX files in archived Approach databases still used by some organizations today.

How to open an ADX file

You should not attempt to open ADX files. Instead, Approach databases reference the files to decrease data searching and retrieval times.

ADX Audio

Developer CRI Middleware
Popularity
3.3  |  7 Votes
 

An ADX file may also be an audio file used by a video game developed using CRIWARE-ADX. Typically, ADX files contain background music played in a game. For example, Sonic Mega Collection contains music saved as ADX files.

More Information

CRI Middleware developed the ADX format for use in its own and other developers' video games. Notably, several games in the Sonic the Hedgehog series use ADX audio files.

Primarily, ADX files contain background music that is meant to be looped during gameplay (e.g., while a player is playing a specific level). ADX files are compressed using a lossy Adaptive Differential Pulse-Code Modulation (ADPCM) variant.

NOTE: Some Sofdec videos (.SFD files) contain interleaved ADX audio.

How to open an ADX file

You can open an ADX file with vgmstream (multiplatform) to play the audio it contains. You can also play ADX files with Foobar2000 or Nullsoft Winamp, after installing those programs' vgmstream plug-ins.

How to convert an ADX file

With the vgmstream plug-in installed, you can use Foobar2000 to convert ADX files to .WAV (WAVE Audio) and other formats.
