Adobe Photoshop Inks File

Developer Adobe Systems
Popularity
4.2  |  101 Votes
 

What is an API file?

Settings file used by Photoshop, a raster graphics editing program; stores color settings for ink colors that are used for printing images; enables users to save and load calibrated color settings, which is useful when establishing printing properties for a specific inkjet or laser printer.

Acrobat Plug-in

Developer Adobe Systems
Popularity
3.8  |  34 Votes
 

An API file may also be a plugin that extends the functionality of Adobe Acrobat and Reader applications. It contains information that enables features or support for file formats not natively supported, such as enhanced PDF editing or improved compatibility with different file types. Adobe or third-party developers may create and distribute API plugins to Acrobat and Reader users.

More Information

Plugins help make Acrobat and Reader more versatile and customizable for professional and personal users and their specific workflows. However, you need to ensure that the plugins are compatible with your version of Acrobat or Reader since compatibility issues have arisen due to software updates and advancing system architecture. For example, older versions of Acrobat and Reader use API files for plugins, whereas more recent versions of the programs typically use .ACROPLUGIN files.

How to open an API file

You can install API plugin files in older versions of Adobe Acrobat and Reader by placing them in the following directories and then restarting the applications:

  • Windows - C:\​Program Files (x86)\​Adobe\​Acrobat [version]\​Acrobat\​plug_ins
  • Mac - Applications/​Adobe Acrobat [version]/​Acrobat/​plug_ins

Newer versions of Acrobat and Reader may also support API files, but they typically use ACROPLUGIN files.
