Computable Document Format File (Legacy)

Developer Wolfram Research
Popularity
3.9  |  55 Votes
 

What is a CDF file?

A CDF file is an interactive document saved in the Computable Document Format (CDF), a document standard developed by Wolfram Research. It may contain text, visual elements, such as interactive graphs or images, Wolfram Language input or output, and page formatting information. Wolfram discontinued the CDF format in 2021 in favor of the interactive Mathematica Notebook (.NB) format.

More Information

Various types of Wolfram software users, such as teachers, scientists, and publishers, may save documents as CDF files. Typically, they open the files with Wolfram Research Player or Mathematica when displaying business presentations, reading journal articles, or interacting with e-learning materials such as textbooks and tutorials. The format also allows users to load visualization data on-demand from within the CDF file.

How to open a CDF file

You can open CDF files with Wolfram Research Player and Wolfram Research Mathematica in Windows, macOS, and Linux. To open a CDF file with either application, select File → Open.

How to convert a CDF file

Wolfram Research Mathematica can convert CDF files to various file types, including the following formats:

  • .NB - Mathematica Notebook
  • .TXT - Plain Text File
  • .TEX - LaTeX Source Document
  • .PS - PostScript File
  • .RTF - Rich Text Format File
  • .M - Wolfram Language Package
  • .PDF - Portable Document Format

CryENGINE Character Definition File

Developer Crytek
Popularity
3.3  |  15 Votes
 

A CDF file may also be a CryENGINE Character Definition file used by the CryENGINE game development platform. It stores references to character assets, such as .CHR (character) or .CGF (geometry) files, along with attachments like clothing, weapons, or other equipment. CDF files organize these elements so that the game engine can correctly load and display the character in the game environment.

More Information

Game designers, 3D artists, and developers working on PC and console games built with CryENGINE are most likely to encounter CDF files. By defining a character’s components and attachments in a CDF file, the engine can assemble and render the character dynamically during gameplay. This allows designers to manage multiple variations of a character, including different outfits, gear, or accessories, without modifying the core character model.

How to open a CDF file

You can open CDF files with CryENGINE SDK in Windows. The software provides tools for game asset management, character creation, and game testing.

Common Data Format

Developer GSFC
Popularity
3.2  |  15 Votes
 

A CDF file may also be a data file saved in the Common Data Format (CDF), a platform-independent format designed to store and share scientific data. It stores simple numbers and complex datasets in a binary format that enables researchers and government agencies to share and use the data on any computer or software.

More Information

Some examples of who uses the format:

  • NASA scientists might use the format to store data from satellites that measure solar wind and magnetic fields in space.
  • Climate researchers may use the format to share atmospheric data, such as temperature and pressure readings, across different labs and institutions.
  • Space weather centers may use CDF files to distribute real-time alerts about solar storms that could affect satellites and power grids.
  • Physicists might use CDF files to save results from particle simulations and make them available to other researchers for verification.

One of the most common ways users create CDF files is with NASA’s GSFC CDF software toolkit. The toolkit supports multiple operating systems and includes tools for writing, reading, and visualizing data, enabling users to reliably access the same information across platforms.

How to open a CDF file

You can open CDF files with several programs, including GSFC CDF software, Weisang FlexPro, and The MathWorks MATLAB (with CDF Patch). ISTP CDF Skeleton Editor can also open CDF files.

Nominal Small Business Accounting Company Data File

Developer bsmArt Consulting
Popularity
3.2  |  6 Votes
 

A CDF file may also be a company data file created by bsmArt Consulting Nominal Small Business Accounting, a discontinued software program designed to help small businesses manage their finances. It contains all of a company's accounting information, including account balances, sales and purchase items, payment terms, tax codes, and payroll data for employees.

More Information

Businesses used CDF files to keep an organized record of their financial activity. By loading a CDF file into the software, users could view and update their accounts, generate invoices, track expenses, and prepare basic financial reports. This made it easier for business owners to monitor cash flow and meet tax obligations.

How to open a CDF file

NetCDF Format File

Developer NetCDF
Popularity
3.2  |  6 Votes
 

A CDF file may also be a scientific data file saved in the Network Common Data Form (NetCDF) format. It stores data in multidimensional arrays, where each array (variable) can represent measurements collected over time, space, or other dimensions, such as latitude, longitude, and time. CDF files may also contain metadata describing the data's meaning, units, and structure, helping users more easily share and analyze their data.

More Information

Researchers and scientists use CDF (or more commonly, .NC) files to store large datasets that can be accessed and analyzed across different operating systems and programming languages, and are widely used in the fields of meteorology, oceanography, and environmental science.

NOTE: NetCDF is a set of data formats, programming interfaces, and tools developed by Unidata, part of the University Corporation for Atmospheric Research (UCAR).

How to open a CDF file

You can open CDF files with Panoply, a viewer developed by NASA’s Goddard Institute for Space Studies. NcView is another common choice for opening CDF files.

Affymetrix Chip Definition File

Developer Affymetrix
Popularity
2.7  |  7 Votes
 

A CDF file may also be a library used by Affymetrix DNA microarray analysis software. It contains the probe set definitions for a GeneChip microarray, including the exact coordinates of each probe cell on the chip. CDF files also specify how individual probes are grouped into probe sets, which represent genes or target sequences used during data analysis.

More Information

Biologists, geneticists, and bioinformaticians working with Affymetrix microarray experiments are the most likely users to encounter CDF files. CeneChips are physical DNA microarrays produced by Affymetrix that contain thousands of probes designed to detect specific DNA sequences or gene expression levels.

The CDF file maps each probe to its intended target, so researchers can accurately match the raw measurements of intensity data in the corresponding .CEL files to the correct gene or sequence. Without the CDF file, it would be impossible to properly analyze the raw microarray data.

How to open a CDF file

You can open CDF files with Affymetrix Tiling Analysis Software in Windows. VSN GenStat and RMAExpress also support the files.

Content Definition File

Developer ECMG
Popularity
2.5  |  6 Votes
 

A CDF file may also be an XML-based data file saved in the Content Definition File (CDF) format, a format for exchanging content between different Enterprise Content Management (ECM) systems. It contains a structured list of content objects, such as documents, records, or folders, along with their associated metadata. Examples of metadata stored in CDF files include titles, authors, creation dates, classifications, and security permissions, enabling systems to recreate the content in another repository accurately.

More Information

ECM vendors, system integrators, and organizations that need to migrate or synchronize large amounts of enterprise content are the users most likely to encounter CDF files. The Enterprise Content Management Group (ECMG) developed the CDF format as part of its Content Transformation Services (CTS) framework to provide a common framework for moving content between different ECM solutions, even when those systems used different internal formats.

How to open a CDF file

You can open CDF files with ECMG FileMatica in Windows. However, the program is discontinued.

Since the format is based on XML, advanced users can also inspect or edit CDF files in a text or XML editor if they need to troubleshoot or adjust their content migration process.

Channel Definition Format

Developer N/A
Popularity
2.6  |  5 Votes
 

A CDF file may also be a data file used to distribute frequently updated information, such as news updates, weather reports, or stock information, over the Internet. It stores the channel's title, description, URL, update frequency, and other metadata that allows browsers or news readers to access and display the content. CDF files are largely obsolete today.

More Information

The files were primarily associated with Microsoft's Active Channel technology in Internet Explorer, which allowed automatic delivery of web content in the late 1990s and early 2000s. When a user subscribed to a channel defined by a CDF file, the software automatically checked the source for updates and downloaded new content to the user's computer, helping them to stay informed without manually visiting each website.

NOTE: Modern web technologies, such as RSS, Atom feeds, and push notifications, have replaced CDF for distributing frequently updated content.

C.Scope Data Log

Developer C.Scope International
Popularity
2.0  |  1 Vote
 

A CDF file may also be a data log created by a C.Scope cable avoidance device. It contains a record of how and when the device was used, as well as the device's model type, serial number, and other data. CDF files can be used to analyze a device's use and view cable survey data, using the C.Scope PC Toolkit or C.Scope Relay mobile apps.

More Information

C.Scope International produces pipe and cable location equipment that allows utility and construction professionals to locate underground pipes and cables. C.Scope's cable avoidance devices produce CDF files, which contain a record of when and how the devices were used, as well as the data the devices collected.

Devices generate a new CDF file each time they are used. Each CDF file contains the following data:

  • Device model type
  • Device serial number
  • Automatic self test result
  • User identity
  • Date and time of use
  • Device detection mode
  • Signal strength
  • Signal generator/transmitter frequency
  • Cable/pipe depth
  • Sonde depth

Each device can store roughly 12 months' worth of CDF files at one time. Device operators and auditors can transfer a device's CDF files to a PC or mobile device, to view the files' data in C.Scope PC Toolkit or C.Scope Relay.

How to open a CDF file

You can open a CDF file in C.Scope PC Toolkit (Windows) or C.Scope Relay (Android, iOS). For example, to open a CDF file in C.Scope PC Toolkit, open the program and select the Analyse tab. Then, select Open, navigate to your CDF file, and open the file.
