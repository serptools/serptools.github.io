Bitdefender Virus Definition File

Developer Bitdefender
Popularity
3.2  |  13 Votes
 

What is a CVD file?

File used by Bitdefender antivirus and Internet security software; saves the main collection of virus signatures for the software which help recognize viruses when the program scans the computer; is periodically refreshed automatically by the software.

More Information

CVD files can be supplemented by .IVD files, which contain incremental virus definitions for the latest updates. However, when CVD definitions are updated, the IVD definitions are reset until new updates are available or until the next CVD refresh.

ClamAV Virus Definitions File

Developer Sourcefire
Popularity
3.2  |  9 Votes
 

File used by ClamAV, an open source antivirus program; contains virus definitions for updating the list of recognized viruses in ClamAV; saved as a digitally signed tarball that contains one or more virus databases.

More Information

Common CVD files are daily.cvd and main.cvd. Main.cvd contains the latest stable base virus database definitions and daily.cvd contains daily virus definition updates that can be scheduled for download and installation.

Canvas X Drawing

Developer Canvas GFX
Popularity
3.5  |  2 Votes
 

A CVD file may also be a drawing created by the Mac version of Canvas GFX Canvas X Draw, a technical illustration application. It may store raster and vector graphics, annotations, and custom shapes. CVD files also contain drawing settings, image effects, and layers.

More Information

Canvas X Draw is available for Windows and macOS, but only the Mac version supports CVD files (the Windows version saves drawings as .CVX files, by default). ACD Systems originally developed Canvas X software, but it shifted development responsibilities to Canvas GFX In 2017.

With the Canvas X Draw application, you can design various types of drawings, from flowcharts to presentations to the blueprint for a structure. When you save your drawing with the Mac version of Canvas X Draw, the application creates a CVD file to store its contents. After saving the drawing, you can close it, then re-open it for further editing.

How to create a CVD file with Canvas X Draw

You can create a CVD file with Canvas X Draw in macOS by selecting File → Save... or Save As..., then choosing CVD - Canvas X Draw from the "Format" dropdown menu (should be set as this type, by default).

How to open a CVD file

You can open a CVD file with Canvas GFX Canvas X Draw in macOS. To open a CVD file with Canvas X Draw, select File → Open....

How to convert a CVD file

The Mac version of Canvas GFX Canvas X Draw can convert CVD files to many file types including the following formats:

  • .CVDTPL - Canvas X Draw Template
  • .DWG - AutoCAD Drawing
  • .DXF - Drawing Exchange Format
  • .PSD - Photoshop Document Format
  • .JPG - JPEG Image
  • .PNG - Portable Network Graphic
  • .GIF - Graphical Interchange Format
  • .SVG - Scalable Vector Graphics File
  • .CGM - Computer Graphics Metafile
  • .PDF - Portable Document Format
