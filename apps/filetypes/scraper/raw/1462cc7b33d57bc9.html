What is a TMX file?

TMX files have multiple uses, and Translation Memory Exchange is one of them. Read more about the other uses further down the page.

Translation Memory Exchange File

Translation Memory Exchange (TMX) files are a widely-used file format for storing translation memory data.

These files allow for the exchange of translation memory data between different CAT (Computer-Assisted Translation) tools. This is important for translation agencies and companies that work with multiple CAT tools.

TMX files contain multiple translation units, each consisting of a source segment and its corresponding translation. This format allows for easy management and sharing of translation memories.

TMX files also support multiple languages and multiple levels of context, making them a versatile format for storing translation memory data.
