Scrambled Wii Disc Image

Developer N/A
Popularity
2.9  |  28 Votes
 

What is a WII file?

Disc image format used by the homebrew community for Nintendo Wii discs; saves a Wii game in a scrambled format; can be unscrambled and converted to the .ISO format with Wii Unscrambler and then burned to a standard disc; sometimes used for backing up Wii games.

More Information

The WII format is not supported by Nintendo, the developers of the Wii game console system.

Minecraft: Wii U Edition World

Developer 4J Studios
Popularity
4.3  |  4 Votes
 

A savegame.wii file likely contains a Minecraft: Wii U Edition world. Minecraft modders rename Wii U Edition worlds to savegame.wii for use with MCCToolChest, a modding application. Prior to being renamed to savegame.wii, Wii U Edition worlds have extensionless filenames like 210722230748.

More Information

MCCToolChest allows Minecraft modders to load, edit, and convert Minecraft: Legacy Console Edition worlds to other formats (such as the .DAT format used by Minecraft: Java Edition). To load a Wii U Edition world in MCCToolChest, a modder must first rename that world to savegame.wii.

Modders can also use MCCToolChest to convert a Java Edition Minecraft world to a Wii U Edition world. To do so, they must:

  1. Load a savegame.wii file.
  2. Select Tools → Convert → From PC.
  3. Use the Convert From PC window to select and convert their Java Edition world.

The converted Java Edition world is saved over the existing savegame.wii file. To use this file with Minecraft: Wii U Edition, a modder must rename the file to use its original, extensionless filename and then replace it within their game's files.

Common WII Filenames

savegame.wii - The name modders must assign a Minecraft: Wii U Edition world before loading it in MCCToolChest.

How to open a WII file

You can open a savegame.wii file with MCCToolChest (Windows). To do so:

  1. Select File → Open from MCCToolChest's menu bar.
  2. In the Open window, select savegame.wii Files from the file type selection menu.
  3. Navigate to and open your savegame.wii file.

After you do, the world data your savegame.wii file contains will appear in MCCToolChest.

How to convert a WII file

You can use MCCToolChest to convert the Minecraft: Wii U Edition world a savegame.wii file contains to the world formats used by the Classic and Legacy PlayStation and Xbox versions of Minecraft. For complete instructions on how to do so, refer to MCCToolChest's documentation.

Asura Engine Wii Game Data Archive

Developer Rebellion Developments
Popularity
3.0  |  4 Votes
 

A WII file may also be a game data archive used by a Nintendo Wii video game created using the Asura Engine. It contains game assets, such as in-game music and sound effects. The Wii version of The Simpsons Game contains WII files.

More Information

The Asura Engine is a video game development engine created by Rebellion Developments. Many games published by Rebellion and other publishers, such as Electronic Arts, have been developed using the Asura Engine.

Most games developed using the Asura Engine store their audio assets in .ASR files. However, the Wii versions of some games, such as The Simpsons Game, store these assets in WII files instead.

How to open a WII file

WII files are not meant to be opened by players. Modders have yet to create tools for opening and extracting data from a game's WII files.
