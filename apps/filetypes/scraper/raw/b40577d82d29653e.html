What is a PRFPSET file?

A PRFPSET file is a video effect preset file used by Adobe Premiere Pro, a professional video editing and production application. It contains effect settings, such as motion, opacity, and time remapping values, in the XML standard. PRFPSET files can be saved and applied to other video clips.

More Information

Premiere Pro presets save you time by saving the settings of an effect. After saving the effect preset you can apply it to future video clips instead of having to recreate the effect from scratch. PRFPSET files also allow you to easily share effects with other Premiere Pro users.

NOTE: To export a PRFPSET file, open the "Effects" pane in Premiere Pro, choose the preset you would like to export, click the three-line hamburger icon (or right-click the preset in the pane), and select Export Presets....

How to open a PRFPSET file

You can open PRFPSET files with Adobe Premiere Pro. To import a PRFPSET file, open the "Effects" pane in Premiere Pro, click the three-line hamburger icon (or right-click in the pane), and select Import Presets....

You can also open PRFPSET files with an XML or source code editor since the file stores its information in the XML standard. However, the file is not meant to be manually edited with an XML editor since modifications may break the preset.
