What is an SCS11 file?

XML file created by Show Cue System (SCS) 11, a program used for playing music and sound effect cues during live theatre performances; saves an ordered list of audio files, each of which can be played when triggered by the user.

More Information

SCS11 files store descriptive information about each audio file that helps the live show audio engineer know what the sounds contain. SCS11 files can also store hotkey shortcuts that play certain sounds when triggered. This is useful for impromptu or commonly used sounds.

NOTE: SCS11 files do not store actual audio data, but instead store references to the audio files. Therefore, you need to make sure to transfer all referenced audio files with the SCS11 file to successfully move the project to another computer.
