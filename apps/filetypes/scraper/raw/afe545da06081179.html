Samsung Theme File

Developer Samsung
Popularity
3.3  |  14 Votes
 

What is an SMT file?

Mobile device theme file used by various Samsung mobile phones models; includes graphics (.QMG files) for the background image, buttons, and other user interface items; also includes an XML file that defines the different screen layouts for the theme; saved in the compressed .ZIP format.

More Information

If you download a new SMT file to your PC, you can install it using the following procedures:

  1. Connect your phone to the PC.
  2. Copy the SMT file to the phone's storage card within the Themes directory.
  3. Disconnect your phone.
  4. Navigate to your phone applications and choose Settings → Display and Light → Theme.
  5. Choose the new theme.

NOTE: SMT files are created using the Samsung Theme Designer, a program that allows Samsung developers to design custom themes. Once a theme has been created, you can select the Export Theme button to create the SMT file. The button is located in the Tools Quick Access panel.

Autodesk Shape Manager Text File

Developer Autodesk
Popularity
2.0  |  2 Votes
 

An SMT file contains a CAD 3D model saved in the Autodesk Shape Manager (ASM) format. It stores 3D geometry and structure information of a part in ASCII text format. SMT files can be transferred between various Autodesk applications, such as AutoCAD, Inventor, and Moldflow Adviser, without losing any model fidelity.

More Information

The ASM format was developed from Autodesk ShapeManager, which is a 3D geometric modeling component of Autodesk Inventor and various other Autodesk applications that was developed and released in the early-2000s. The format allows you to maintain the integrity of a CAD model when transferring it between different CAD applications in complex workflows. For example, you can design a prototype in Autodesk Inventor, export the model as an SMT file, then import it into Moldflow Adviser in order to simulate the model as a plastic mold.

ASM consists of two different variations and extensions:

  • Autodesk Shape Manager Binary (.SMB file extension) - This variation is saved in binary format.
  • Autodesk Shape Manager Text (.smt file extension) - This variation is saved in ASCII text format.

How to open an SMT file

You can open an SMT file with various Autodesk applications, such as AutoCAD, Fusion 360, FeatureCAM, and Moldflow Adviser.

To open an SMT file in Autodesk Moldflow Adviser, click the Home tab → Import panel → Import, select the "Files of type" dropdown list, choose "All Models," navigate to the SMT file, then click Open.

To open an SMT file in Autodesk FeatureCAM, select File → Import, select the SMT file, and click Open.

Since SMT files are saved in text format you can also open them with a text editor, such as Microsoft Notepad (Windows) or Apple TextEdit (macOS). Keep in mind that opening SMT files with a text editor will not display the model as if it was opened in a CAD application.
