What is a file extension anyway

File extensions are used to help Windows find and launch the correct program when you click on a file. For example, if you click on a .PDF document, Windows will automatically launch a PDF viewer if you have one installed.

What is a file extension?

A file extension is the characters after the last dot in a file name. For example, in the file name 'myfile.dat', the file extension is dat. It helps Windows select the right program to open the file.
