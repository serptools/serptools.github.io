What is a ZIM file?

Archive created in the OpenZIM format, a standard file format for storing Wiki content offline; organizes articles, references, and other Wiki information in a compact format; supported by readers such as Kiwix and ZIMReader.

More Information

Developers can use the free ZIMLib C++ library to incorporate OpenZIM content in their applications.

SimLab Composer Package

Developer Simulation Lab Software
Popularity
2.5  |  2 Votes
 

A ZIM file may also be a Zip-compressed package that contains a SimLab Composer 3D Scene .SIM file and its resource files. It is utilized to share 3D scenes with other SimLab users as one package. ZIM files are typically created with SimLab Composer, but may also be created by other 3D applications, such as SketchUp and 3ds Max, with the SimLab exporter plugin.

More Information

SimLab Composer is a 3D modeling, animation, and rendering application. You can use the application for various purposes, including VR training tutorials, architectural models, and 3D product visualizations.

To create a ZIM file with SimLab Composer, create a scene, select File → Export, then choose the "SimLab Archive" format from the dropdown menu.

How to open a ZIM file

You can open ZIM files with SimLab Composer (Windows and macOS). To open the file, select File → Open. You can also open ZIM files with SimLab CADVRter (packaged with SimLab Composer) and SimLab CAD Viewer (Android and iOS).

Since ZIM files are compressed with Zip compression you can open them with a Zip utility and directly access the SIM file and the resource files. To do this, rename the .zim file extension to .zip, then open it with a Zip utility.
