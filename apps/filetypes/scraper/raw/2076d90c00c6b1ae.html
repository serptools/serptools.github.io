What are RC files and how to open them

Are you having problems opening a RC file or are you simply curious about its contents? We're here to explain the properties of these files and provide you with software that can open or handle your RC files.

What is a RC file?

RC files have multiple uses, and Resource File Script is one of them. Read more about the other uses further down the page.

Resource File Script

These files are resource script files used by different software compilers. A compiler is a tool used by a programmer to generate executable programs from source code.

Resource scripts contain instructions on how a compiler should generate the program's associated resource file. A resource file contains external data like icons, text, images, etc.

For a resource that exists in a separate file, such as an icon or cursor, the script specifies the resource and the file that contains it. For some resources, such as a menu, the full definition of the resource exists within the script.
