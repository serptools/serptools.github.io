Fortnite Replay File

Developer Epic Games
Popularity
3.9  |  144 Votes
 

What is a REPLAY file?

A REPLAY file contains a replay of gameplay in Fortnite, a sandbox survival where players cooperate to collect resources, build fortifications, and fight enemy creatures. It stores game information used by Fortnite to replay gameplay. REPLAY files allow Fortnite users to save their replays and rewatch them for entertainment or strategic purposes.

More Information

REPLAY files can only be played by Fortnite. Also, the files do not store actual video and cannot be converted to a video file.

If you want to share a REPLAY file and play it in a different program, you must use a screen recording program to record the playback of the REPLAY file. To do this, start a recording of your screen, play the REPLAY file in Fortnite, then end and save the recording. You can then share the newly-saved video file with another user.

Geometry Dash ReplayBot Replay

Developer matcool
Popularity
4.3  |  4 Votes
 

A REPLAY file may also be a replay created by ReplayBot, a player-created Geometry Dash gameplay recording tool. It contains a recording of a player completing a Geometry Dash level. With ReplayBot installed, you can load and play REPLAY files in the Windows version of Geometry Dash.

More Information

Geometry Dash is a multiplatform game in which players click in time with music to jump over obstacles and complete levels. The game's passionate player base has created many mods for its Windows version.

ReplayBot is a mod that allows players to create, save, and play replays of their Geometry Dash gameplay. These replays are saved in REPLAY files. ReplayBot includes some sample REPLAY files, which players can use to test out the mod.

How to open a REPLAY file

You can open a REPLAY file in Geometry Dash with the ReplayBot mod installed (Windows). To do so, open Geometry Dash, and then open ReplayBot. Select the LOAD button to load your REPLAY file, and press the PLAY button to play it.

Rocket League Replay

Developer Psyonix
Popularity
3.3  |  6 Votes
 

A REPLAY file may also be a Rocket League match replay file. It contains game data that Rocket League uses to save and play match replays. By default, Rocket League saves REPLAY files in the ~/​Documents/​My Games/​Rocket League/​TAGame/​Demos directory.

More Information

Rocket League is a free-to-play video game in which players play soccer with rocket-powered cars. After a Rocket League match concludes, players can save a replay of that match by selecting the Save Replay button that appears in the game's post-match menu. These replays are saved in REPLAY files.

Contrary to what you might expect, REPLAY files are not video files (and thus cannot be opened in media players). Rather, they contain all the game data that Rocket League records during a match. The game uses this data to reconstruct a match in its entirety, so players can examine every play from every angle when they watch a Rocket League replay.

How to open a REPLAY file

Rocket League players can open REPLAY files in Rocket League. To play a Rocket League replay:

  1. Go to Rocket League's main menu.
  2. Select Extras.
  3. Select View Replays.
  4. Select the REPLAY you'd like to play.

Descenders Replay

Developer RageSquid
Popularity
2.0  |  2 Votes
 

A REPLAY file may also be a race replay created by the mountain bike racing game Descenders. It contains a recording of a player's run down a Descender's course. Players can choose to create a REPLAY file when they finish a run.

More Information

Descenders is a multiplatform video game that allows players to race mountain bikes on many different downhill courses. The Windows version of Descenders saves players' game replays as REPLAY files, typically to one of the following directories:

  • ~/​steamapps/​common/​Descenders/​Replays
  • ~/​AppData/​LocalLow/​RageSquid/​Descenders/​Replays

These files can be played only in Descender, using the game's Replay feature. If you want to share a Descender replay with someone else, either they must have Descender, or you must make a screen recording of the replay.

How to open a REPLAY file

You can open a REPLAY file in Descenders (Windows). To do so, place the file in your Descenders/​Replays directory. Then, open Descenders and use the Replay feature to play your REPLAY file.
