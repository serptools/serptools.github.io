DaVinci Resolve Project

Developer Blackmagic Design
Popularity
3.8  |  21 Votes
 

What is a DRP file?

A DRP file is a project file created by Blackmagic Design DaVinci Resolve, a video editing and color correction program. It contains data used to load a video editing project in Resolve, as well as the project's properties. DRP files do not directly contain any video, audio, or other media files. Instead, they contain references to the media files a project includes.

More Information

DaVinci Resolve is a popular free video editing program. It allows users to combine, layer, trim, and add effects to video, audio, and image files, before combining the files into a final, rendered video.

By default, DaVinci Resolve does not save users' video editing projects in a single file. Instead, it saves the configuration files that comprise a project in Resolve's installation directory. However, if you want to manually back up a project, you can export the project as a DRP file. You can do this by selecting File → Export Project... from Resolve's menu bar.

NOTE: If you save your project as a DRP file, keep in mind that you will also have to retain copies of the media files your project contains, as DRP files contain only a record of a project's timeline, edits, and settings.

How to open a DRP file

You can open a DRP file in Blackmagic Design DaVinci Resolve (multiplatform) by selecting File → Import Project... from the program's menu bar. If you edit your project and want to back up your changes, you'll have to re-export the project as a new DRP file.

How to convert a DRP file

DaVinci Resolve can convert DRP files to various formats, including:

  • .MP4 - MPEG-4 Video
  • .MOV - Apple QuickTime Movie
  • .AVI - Audio Video Interleave
  • .MXF - Material Exchange Format
  • .DPX - Digital Picture Exchange
  • .AAC - Advanced Audio Coding
  • .WAV - WAVE Audio
  • .EXR - OpenEXR Image

DrawPad Image Project

Developer NCH Software
Popularity
2.8  |  4 Votes
 

A DRP file may also be an image project file created using NCH Software DrawPad, a graphic design application available for Windows and macOS. It contains data used to show the shapes, text, images, and other layers that comprise a DrawPad image. DRP files can be used to export images as .PNG, .JPG, .BMP, .ICO, .SVG, .PDF, and .EPS files.

More Information

DrawPad is a basic graphic design program that allows designers to create simple layer-based images. Images created in DrawPad can include any number of layers, each of which may contain stylized text, drawings, or imported images.

When a designer saves an image project in DrawPad, the project is saved as a DRP file. DRP files can be opened only in DrawPad, but designers can use DrawPad to export the image their DRP file contains as a PNG, JPG, or another type of image file.

How to open a DRP file

You can open DRP files (DrawPad Image Projects) in NCH Software DrawPad (Windows, macOS). To open a DRP file in DrawPad, select File → Open... from the program's menu bar. Then, navigate to and open your DRP file.
