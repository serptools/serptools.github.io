Interchange File Format Image File

IFF is an acronym for Interchange File Format. Files that contain the .iff file extension are used to exchange documents across multiple computer platforms, allowing a user running on one platform to share files with a user running on an entirely different computer platform.

The universal IFF file format was created by Electronic Arts and Commodore. Files that contain the .iff file suffix may contain text, graphics, and even audio data.
