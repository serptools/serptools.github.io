Windows Media Player Playlist

Developer Microsoft
Popularity
3.9  |  84 Votes
 

What is a WPL file?

Playlist of songs or videos that can be played by Windows Media Player; may be used to create dynamic playlists; stores a list of references to the actual files, but not the files themselves; written in the WPL 1.0 format, which is extended from the XML format; can be used in Windows Media Player 9 or later.

GTA Binary Item Placement File

Developer Rockstar Games
Popularity
3.8  |  25 Votes
 

Game file used by Grand Theft Auto (GTA) games such as GTA 4 and GTA: San Andreas; saved in a binary file format and contains item location data such as spawn points, weapon locations, and custom zone information; sometimes edited by the modding community to create game customizations.

WPS Plus Text Document

Developer DEC
Popularity
2.7  |  15 Votes
 

A WPL file may also be a text document created by DEC WPS Plus (version 4.1 or later), a word processing program developed for DOS operating systems. It contains document information, including paragraph styles, font types, sizes, bold, italics, alignment, and other text formatting features in the proprietary WPL formatting. While WPL files were common during the era of DOS-based computing, they are now considered obsolete.

More Information

Digital Equipment Corporation (DEC) developed WPS Plus, which became a popular tool for creating and editing text documents in the 1980s and early 1990s. It offered formatting and editing features that were popular among business and personal users.

How to open a WPL file

You can open WPL files with DEC WPS Plus in Windows. However, the program is no longer developed.
