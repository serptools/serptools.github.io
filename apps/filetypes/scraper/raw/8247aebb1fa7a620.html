.FOX File Extension

2 file types use the .fox file extension.
  • 1.InfoZoom Data File
  • 2.Furcadia Art File

InfoZoom Data File

Developer humanIT
Popularity
3.5  |  11 Votes
 

What is a FOX file?

Native data storage format used by InfoZoom, a data visualization program; contains table data, as well table structure, data formats, summaries, queries, analysis groups, color coding, and permissions information.

More Information

FOX files can be opened by InfoZoom Professional and Business but are not recognized by InfoZoom Analyzer.

Furcadia Art File

Developer Furcadia
Popularity
3.1  |  8 Votes
 

Graphics file used by Furcadia, a multiplayer online game set in a magical world with speaking animals; saved in the Furcadia Object Extended (FOX) format, which supports multiple bitmap graphics in the same file; used for saving avatar sprites, button images, and other game graphics.

More Information

FOX files can be edited with the Furcadia FOX Editor (fsheditor.exe), which is included with the Furcadia installation.

NOTE: FOX files replace the older ".fsh" (Furcadia shape), ".fs2" (Furcadia extended shape), and ".fbj" (Furcadia shape property) formats.
