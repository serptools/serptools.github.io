Encoded File

Developer N/A
Popularity
3.6  |  717 Votes
 

What is an ENC file?

An ENC file is a generic data file that stores encoded information, saved in one of many proprietary or standard formats. Typically, ENC files are encoded to prevent users from modifying the data they contain or using them in an unauthorized manner. However, some ENC files are encoded to optimize them for a particular use, such as transfer over the internet.

More Information

In some cases, ENC files may be encoded using a standard encoding format. For example, some ENC files are encoded using standard uuencode (.UUE) encoding. You can decode standardly-encoded ENC files with any program that supports the format in which they are encoded.

However, most ENC files are encoded using proprietary encoding formats. Only the program that created and/or uses these ENC files can decode them and read the data they contain.

How to open an ENC file

If your ENC file is encoded using a standard encoding format, you can decode it with a program that supports that encoding format.

If your ENC file is encoded using a proprietary encoding format, you may be able to open it with the program that created it. However, many ENC files are not meant to be opened by users. Instead, they contain encoded data that a program loads as needed.

Electronic Navigation Chart

Developer NOAA
Popularity
3.4  |  126 Votes
 

An ENC file may also be a navigation chart saved in the National Oceanic and Atmospheric Administration (NOAA) Electronic Navigational Charts (ENC) format. It contains a vector data set used for marine navigation. NOAA and other agencies, including international hydrographic and charting agencies, maintain databases of ENC files, for use by government, commercial, and recreational vessels.

More Information

Each NOAA ENC file contains data (referred to as chart features) that navigational systems use to show a detailed representation of a section of the U.S. coastline and/or marine environment. This data can include coastal topography, bathymetry, landmarks, place names, and marine boundaries.

NOAA ENC files are saved in the S-57 format, which is a data standard defined by the International Hydrographic Organization (IHO). This allows those using any navigational device that supports the S-57 format to use ENC files.

NOTE: You can download ENC files from NOAA's website.

How to open an ENC file

The exact steps required to load and use ENC files with marine navigation devices and software vary. To use ENC files with a marine navigation device or application, refer to your device or application's documentation.

If you want to view the navigation chart that a single ENC file contains, you can do so by opening the file with CARIS Easy View (Windows). You can also view and download up-to-date ENC files from NOAA ENC Viewer, a comprehensive map and database of all of NOAA's ENC files.

Cabal Online Data File

Developer ESTsoft
Popularity
3.3  |  135 Votes
 

An ENC file may also be a data file used by ESTsoft Cabal Online, a massively multiplayer online role-playing game (MMORPG). It contains compressed and encoded game data. While most players will never need to open or modify Cabal Online's ENC files, modders sometimes decompress ENC files to examine and modify their contents.

More Information

In some cases, Cabal Online ENC files can become corrupted, preventing players from opening and playing the game. Most often, this issue occurs after a player has applied a game patch. To attempt to fix this issue, players can fully uninstall and reinstall Cabal Online.

How to open an ENC file

You can use Cabal ENC Tool (Windows) to decompress an ECN file, allowing you to examine its contents. You can also use Cabal ENC Tool to package files into an ENC file.

Encore Musical Notation

Developer Passport Music Software
Popularity
3.4  |  70 Votes
 

An ENC file may also be a musical score created by Passport Music Software Encore, a scorewriter for Windows and macOS. It contains musical notation, such as notes, measures, time signatures, and key signatures. It also contains the score's title and composer information.

More Information

Developer Don Williams created Encore in 1984. The first version of the program was published by Passport Designs; subsequent versions of the program were published by other companies, including GVOX and Passport Music Software.

Encore allows composers to create, play back, and print musical scores. These scores are saved as ENC files. To create a score, composers can either import .MIDI files, record themselves playing music via a MIDI device, or manually place and edit notes (as well as other notation).

How to open an ENC file

You can open an ENC file with Passport Music Software Encore (Windows, macOS) or Myraid Harmony Assistant (Windows, macOS).

How to convert an ENC file

You can use Passport Music Software Encore or Myraid Harmony Assistant to convert ENC files to the .MXL (Compressed MusicXML) format, which is supported by many music composition and notation programs.

CopySafe Protected PDF File

Developer ArtistScope
Popularity
3.0  |  56 Votes
 

An ENC file may also be a copy-protected .PDF file created with ArtistScope CopySafe PDF software. It stores a PDF document in a secure format that prevents images and text from being copied from the document and protects against Printscreen and screen capture commands. ENC files may also include printing protection, password protection, and date expiration.

More Information

ArtistScope offers several apps for creating and reading secured PDF documents. Options include ArtistScope CopySafe PDF Protector (commercial), CopySafe PDF Converter (commercial), and CopySafe PDF Reader (free).

How to create an ENC file with CopySafe PDF Protector

CopySafe PDF Protector app secures your PDF documents as ENC files. In the app, you can choose from various security settings to protect the document when creating an ENC file. After selecting your security settings, choose the save name and destination and click Finish.

How to open an ENC file

You can open ENC files with the free ArtistScope CopySafe PDF Reader app in Windows. For example, to open an ENC file with CopySafe PDF Reader, select File → Open.

How to convert an ENC file

ArtistScope does not provide software for converting ENC files.

Smart Switch Encrypted App Backup

Developer Samsung
Popularity
3.2  |  6 Votes
 

An ENC file may also be an encrypted copy of an Android app's .APK file. Samsung Smart Switch, a backup app for Galaxy smartphones, creates ENC files as part of its backup process. Because ENC files are encrypted, you cannot open or install them as you would unencrypted APK files.

More Information

Samsung Smart Switch is the official backup app used to transfer a user's contacts, photos, call history, apps, and other data from one Samsung Galaxy phone to another. When creating a backup, Smart Switch creates many encrypted files, including ENC files.

Each ENC file is an encrypted copy of an APK file installed on a phone. Smart Switch uses ENC files to reinstall backed-up apps during the backup restore process. You cannot use ENC files to manually reinstall backed-up apps, because Smart Switch must decrypt the files before they can be used.

How to open an ENC file

ENC files are not meant to be opened. You can use Samsung Smart Switch to restore a Galaxy backup that contains ENC files to a Galaxy phone. Doing so will install the apps your ENC files contain on your phone.
