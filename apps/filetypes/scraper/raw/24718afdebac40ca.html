Better Portable Graphics Image

Developer Fabrice Bellard
Popularity
4.1  |  87 Votes
 

What is a BPG file?

A BPG file is a raster image saved in the Better Portable Graphics (BPG) format. It contains an image that is similar in quality to a .JPEG. However, because the BPG format uses more efficient file compression, BPG images typically have smaller file sizes than similar-quality JPEGs.

More Information

Developer Fabrice Bellard created the BPG file format in 2014, as a potential replacement for the JPEG format. Bellard based the BPG format's compression method on the High Efficiency Video Coding (HEVC) compression standard, which is more efficient than JPEG compression.

The BPG format supports:

  • The same chroma formats as JPEG
  • 8 to 14 bits per channel, providing a higher dynamic range than JPEG
  • Lossless compression
  • Animated images

Despite its advantages over the JPEG format, the BPG format is not widely used. However, Bellard's work on the BPG format inspired the creation of the .AVIF format, which also uses a video compression standard to create high-quality compressed images.

How to open a BPG file

You can open a BPG file with XnViewMP (multiplatform), BPGviewer (Windows), or Simple BPG Image viewer (multiplatform).

How to convert a BPG file

You can use XnViewMP to convert BPG files to many other image formats, including:

  • .JPG - JPEG Image
  • .GIF - Graphical Interchange Format
  • .PNG - Portable Network Graphic
  • .TIFF - Tagged Image File Format
  • .BMP - Bitmap Image
  • .PCX - Paintbrush Bitmap Image
  • .TGA - Targa Graphic
  • .JP2 - JPEG 2000 Core Image
  • .JXR - JPEG XR Image
  • .WEBP - WebP Image
