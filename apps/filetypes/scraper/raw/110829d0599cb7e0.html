What are PFD files and how to open them

Are you having problems opening a PFD file or are you simply curious about its contents? We're here to explain the properties of these files and provide you with software that can open or handle your PFD files.

What is a PFD file?

PFD files have multiple uses, and PlayStation 3 Savegame Data is one of them. Read more about the other uses further down the page.

PlayStation 3 Savegame Data

These Sony PlayStation 3 PFD files contain saved game control data files that allow users to stop gameplay at any point in time and return to the point at which they were playing when they are ready to resume their gaming.

How to open PFD files

Important: Different programs may use files with the PFD file extension for different purposes, so unless you are sure which format your PFD file is, you may need to try a few different programs.
