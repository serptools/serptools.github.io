All known file formats using extension .IGS

While Ignition Spreadsheet is a popular type of IGS-file, we know of 4 different uses of the .IGS file extension. Different software may use files with the same extension for different types of data.

4 known uses of the IGS file extension

  • Ignition Spreadsheet
  • Indigo Renderer Scene
  • Initial Graphics Exchange Specification
