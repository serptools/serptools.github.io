What are ADF files and how to open them

Are you having problems opening an ADF file or are you simply curious about its contents? We're here to explain the properties of these files and provide you with software that can open or handle your ADF files.

What is an ADF file?

ADF files have multiple uses, and Act! Database is one of them. Read more about the other uses further down the page.

Act! Database

Act!, previously Sage Act, is a marketing and CRM software solution. They name their database files using the .adf file extension.

      SNAP Archive Data File

      These ADF files are archive files created in SNAP Surveys, an online survey management service.

      ADF files consist of the MDF (Metadata File) and the RDF (Raw Data File). Snap Surveys use this file type to help you run surveys that you can either store or share the data you collect. You can use Snap WebHost to run and manage online surveys.

      Advanced Data Format Database

      These ADF files contain numerical data that is stored in an Advanced Data Format Database. ADF files consist of nodes, and each node follows the same rules: There is one "central root", every child node has a parent, and the root node has no parent. The nodes form a tree structure to allow for multiple file branching. When opening an ADF file, the root receives information from the calling program to access the tree. A search gets run on the tree to find the needed information or to store new data.

      Amiga Disk image File

      These ADF files contain the disk images used by the Amiga home computers and emulators. ADF files are typical images of the Amiga-formatted track, held on the 880 KiB disk. EasyADF is an app that allows you to download ADF files and copy them onto your Amiga disk. ADF Opus, a Windows-based program, will enable you to create your own ADF files. Acorn also used these ADF files in their computers and emulators.

      PS/2 MicroChannel Adapter Description File

      These ADF files contain the configuration information of the MicroChannel Adapter card. ADF files are text files that store information about the card's memory addressing and interrupts.

      MicroChannel cards were the basis of early plug-and-play systems. You insert your MCA card with a custom floppy disk (this came with your pc) to add new cards (video, memory, printer, etc.) to your pc.

      This allowed you to add new cards to your original hardware without the help of an expensive technician. The downside was that you could not add or change your PC's cards if you do not have the custom floppy disk.

      You usually do not need to edit or view these files, as they only contain information usable by the software program that uses them.

      SuperDOS Adapter File

      These ADF files contain the information and configuration of the BlueBird SuperDOS, a multi-tasking multi-user operating system used to run business applications over a "dumb" terminal. This replaced expensive minicomputers with a single PC.

      The SuperDos ADF file runs when you boot your PC. BlueBird SuperDOS assisted car rental agencies and movie/storage companies to reduce the number of expensive computers on every desk.
