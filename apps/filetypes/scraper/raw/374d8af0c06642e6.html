Mass Effect Package

Developer Electronic Arts
Popularity
4.4  |  11 Votes
 

What is a PCC file?

File used by Mass Effect 2 and 3, a first person shooter game based on the Unreal Engine 3 gaming engine; may contain various game data, including models, textures, and rooms; commonly used to modify (or "mod") game content.

More Information

PCC files are typically found in the BioGame\​CookedPC directory within a Mass Effect 2 installation. An example PCC file is BIOG_HMF_ARM_SHP_R.pcc, where "HMF" stands for a Human Female character.

NOTE: PCC files are similar to .UPK files.

Photocomposition Code File

Developer N/A
Popularity
3.5  |  2 Votes
 

A PCC file may also be a text file used by the Government Publishing Office's (GPO) text editing system to publish official information products. It contains photocomposition codes and typesetting instructions. PCC files are helpful for updating United States Code, which consists of the general and permanent laws of the United States under 54 titles, such as General Provisions, The Congress, The President, Copyrights, Labor, and Transportation.

More Information

The GPO is the federal government's official resource for producing, preserving, indexing, authenticating, disseminating, procuring, and cataloging official information products of the U.S. Government. These information products include official publications of Congress and the White House and U.S. passports.

NOTE: PCC files can be opened with any text editor.
