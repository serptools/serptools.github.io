All known file formats using extension .MEF

While CA Visual Objects Module Export File is a popular type of MEF-file, we know of 4 different uses of the .MEF file extension. Different software may use files with the same extension for different types of data.

4 known uses of the MEF file extension

  • Camera RAW Image File (Mamiya)
  • CA Visual Objects Module Export File
  • MAIET Encrypted File System
  • Mirror II Emulation File
