Cube 2 Demo File

Developer Wouter van Oortmerssen
Popularity
4.5  |  6 Votes
 

What is a DMO file?

A DMO file is a game demo file created by various games using the Cube 2 game engine, such as Red Eclipse, Blood Frontier, and Cube 2: Sauerbraten. It contains a recording of a single or multiplayer session, including player movements, game events, and a chat log, in the Cube 2 Demo (DMO) format. DMO files enable games running the Cube 2 engine to replicate an exact replay of the match.

More Information

Cube 2: Sauerbraten is a free first-person shooter game running the Cube 2 game engine that is most commonly associated with the DMO file type. When you load the game, you can enable "/recorddemo 1" before playing a match or deathmatch so the game records it to a DMO file.

After the match, game clients can download the DMO file from the host server that recorded it. Once downloaded, the game can play the files in local mode, allowing you to view the demo from any angle in the game.

NOTE: DMO files do not contain video data. If you would like to record a video session, choose an alternate recording function (e.g., Cube 2: Sauerbraten's movie recording functionality allows you to record a match as an .AVI file).

How to open a DMO file

You can open DMO files with the Cube 2 game engine games that created them. For example, Cube 2: Sauerbraten can play DMO files the game created to store a game demo.

Derive Demo

Developer Texas Instruments
Popularity
1.5  |  2 Votes
 

A DMO file may also be a demo created by Texas Instruments Derive, a now-discontinued algebra software used to solve, visualize, and analyze mathematical problems. It contains a demonstration of a mathematical function or operation that the Derive software can execute. Demo users often save tutorials as DMO files to showcase specific problem-solving techniques and present pre-made mathematical workflows to help users learn or explore mathematical concepts.

More Information

Soft Warehouse initially developed Derive in the 1980s until Texas Instruments acquired it in 1989. Students, educators, and researchers often used Derive to perform symbolic and numeric computations, including algebra, calculus, trigonometry, and linear algebra. They would save demonstrations as DMO files, which they could replay as demos.

NOTE: Texas Instruments ceased development of Derive in 2007, replacing it with TI-Nspire CAS, and the DMO format has become obsolete.

How to open a DMO file

The only known program that can open DMO files is Texas Instruments Derive. However, Derive is no longer developed.
