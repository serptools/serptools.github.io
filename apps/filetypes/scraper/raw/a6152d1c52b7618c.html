PlayStation 3 Update File

Developer Sony
Popularity
4.2  |  290 Votes
 

What is a PUP file?

A PUP file is a file or patch used to update the PlayStation 3 (PS3) system software. It may include security patches, bug fixes, and extra operating system features. PUP files are typically named PS3UPDAT.PUP.

More Information

You can choose to update the PS3 console by downloading the PUP file from the Internet directly to the console. You can also download the PUP file from the Internet to your computer, then transfer the update file using storage media, such as a hard disk or USB storage device.

When transferring the PUP file from a storage device to the PS3, save it in the /​PS3/​UPDATE folder. To update the PS3, select Settings → System Update in the console, then press the "X" button on the controller to begin the update process.

NOTE: To extract the contents of a PUP file in Windows, use the PS3 PUP Extractor program.

Puppy Linux DotPup Installer Package

Developer Puppy Linux

Installation package used by Puppy Linux, a variant of the Linux operating system designed for ease of use; referred to as a "DotPup" file; runs an installer when opened; similar to an .EXE or .MSI file used by Windows software installers.

More Information

The PUP format has largely been replaced by the .PET format with newer versions of Puppy Linux.

PlayStation 4 or 5 Update File

Developer Sony
Popularity
3.7  |  13 Votes
 

A PUP file may also be a system update file used by the Sony PlayStation 4 (PS4) or Playstation 5 (PS5). It contains firmware that is meant to be installed on the corresponding game console. PS4 PUP files are named PS4UPDATE.PUP. PS5 PUP files are named PS5UPDATE.PUP.

More Information

Typically, the PS4 and PS5 automatically update their system software as needed, via a connection to the Internet. However, a user can also manually update their console's software. To do so, the user must first download the appropriate PUP file from PlayStation.com.

You might want to manually update your console's software if:

  • The automatic update process fails
  • Your console is not connected to the Internet

You can also use a PUP file to perform a factory reset on your device. Doing so will overwrite your device's current firmware and delete all the data on your console, including saved game data.

How to open a PUP file

To install a PUP file on a PS4 or PS5:

  1. Download the appropriate PUP file (PS4UPDATE.PUP or PS5UPDATE.PUP) from PlayStation.com.
  2. On a FAT32- or exFAT-formatted USB drive, create a folder named PS4 or PS5. In that folder, create another folder named UPDATE.
  3. Place your PUP file in the UPDATE folder.
  4. Connect your USB drive to your console.
  5. Start your console in Safe Mode. To update your console's software, select Update System Software. To factory reset your console, select Reinstall System Software.
  6. Select Update from USB Storage Device → OK.
