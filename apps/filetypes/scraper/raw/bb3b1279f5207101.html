Ability Photopaint Studio Image

Developer Ability Plus Software
Popularity
2.8  |  11 Votes
 

What is an APX file?

Image created by Ability Photopaint Studio, a drawing program included with Ability Office; support multiple editing layers as well as brushes, fills, shapes, and other standard raster graphic editing features; can be used for generating graphics used in Ability Write documents (.AWW files).

More Information

APX files can be saved to standard bitmap formats such as .GIF, .BMP, .JPG, and .PNG.

Archicad Add-on

Developer GRAPHISOFT
Popularity
3.5  |  2 Votes
 

An APX file may also be an add-on that extends the capabilities of the Windows version of GRAPHISOFT Archicad architectural designing application. It contains executable binary code and resources required to add functionality to the software. Examples of functionality available via APX files include support for exporting and importing additional formats, library objects, and design tools not included with the software.

More Information

GRAPHISOFT Archicad is a building information modeling (BIM) application with tools to design, visualize, and document the appearance of building interiors and exteriors. However, there are capabilities Archicad does not provide out of the box that some users require. In these cases, users can install additional features via APX add-ons.

Archicad users can build custom add-ons for Archicad. However, developing add-ons requires additional tools, including GRAPHISOFT's API Development Kit and Microsoft Visual Studio (Windows) and Apple Xcode (macOS).

NOTE: While the Windows version of Archicad uses APX files for add-ons, the macOS version supports add-ons saved as .BUNDLE files.

How to open an APX file

You can open APX files with GRAPHISOFT Archicad in Windows. To open an APX file, select Options → Add-On Manager, choose the add-on, then select OK.
