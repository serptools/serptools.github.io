Manga Studio Story File

Developer CELSYS
Popularity
4.1  |  12 Votes
 

What is a CST file?

Page layout file created by Manga Studio, a program used for creating comic books; saves the storyline for the comic book and can include multiple chapters, multiple pages (.CPG files) per chapter, and layouts for each page; used for specifying the structure of the comic book.

More Information

Manga Studio became Clip Studio Paint in 2015. It is also known as ComicStudio in Japan.

Director External Cast File

Developer Adobe Systems
Popularity
3.5  |  4 Votes
 

Cast file created by Adobe Director, a movie creation program originally developed by Macromedia; stores properties for generating the video output; allows a cast to be reused for multiple videos.

More Information

CST files contain external casts, which are fully delinked from the target movie. Internal casts, on the other hand, cannot be delinked from the movie and may only be used for the movie they are linked to.

Since Adobe Director has been replaced by Adobe Flash, CST files are not commonly used anymore.

Canvas Custom Set File

Developer ACD Systems
Popularity
2.8  |  5 Votes
 

Settings file created by Canvas, a technical illustration program; contains application keyboard shortcuts used to access program functions; allows users to save custom workspace preferences.

More Information

CST files can include key combinations for selections of menus, tools, inks, pens, and fonts. The default custom set is saved in the DefaultKeyboard.cst file, which is located in the /​Extras/​ directory of the Canvas installation.

Gary Gadget Plug-in

Developer Viva Media
Popularity
2.5  |  4 Votes
 

Plugin file used by Gary Gadget: Building Cars, a car designing, building, and driving game for children; adds features to the game such as car parts or bonus games.

More Information

CST plugin files are available to download from the Gary Gadget: Building Cars website. When the download is complete, place the CST file(s) in the Gary Gadget\​Building Cars\​Plugins directory of the game installation location. If the Plugins folder doesn't exist, you will need to create it. After placing the file(s) in the folder, restart the game and the plugin(s) will be available.

NOTE: The Viva Media Gary Gadget: Building Cars program is no longer available for download.

CatSystem2 Scene Script

Developer Windmill
Popularity
3.5  |  2 Votes
 

A CST file may also be a script used by a visual novel developed using the CatSystem2 game engine. It contains data used to load and play a visual novel scene. This includes metadata and commands used to display text, play audio, and show sprites and animations.

More Information

CatSystem2 is a visual novel game engine developed by Windmill. It is the successor to the original CatSystem, which was developed by Cat Works Team. Various developers use CatSystem2 to create visual novels. For example, Frontwing's Grisaia series and Azarashi Software's Amakano series both use CatSystem2.

Games developed using CatSystem2 include scripts saved as CST files. Each script contains commands used to load and play a particular scene. For this reason, you may also see CST files referred to as CatSystem2 scenes or CST scenes.

How to open a CST file

You can open a CST file with CatSceneEditor (multiplatform) or TriggersTools.CatSystem2 (multiplatform). Doing so will allow you to edit the data the CST file contains.

Citra Save State

Developer Citra Emulator Project
Popularity
3.5  |  2 Votes
 

A CST file may also be a save state created by Ctira, a discontinued Nintendo 3DS emulator. It contains a record of a player's progress in a 3DS game. Players can load a CST file in Citra to resume that saved game (after loading the associated game ROM).

More Information

Citra is a multiplatform app that allows gamers to play Nintendo 3DS games on PCs and Android smartphones. To do so, gamers load a Nintendo 3DS game ROM (usually saved as a .CXI or .3DS file).

When playing a game in Citra, players can save their progress at any time (not just at the game's normal save points). Doing so produces a Citra save state file, which uses the .cst extension.

To create a CST file, players either select Menu→ Emulation → Save State or use Citra's save state hotkeys. They can then save their CST file to any of Citra's 10 save state slots.

NOTE: Citra save states are often not compatible between different Citra versions. So, a CST file created by one version of Citra may not open in another version.

How to open a CST file

You can open a CST file with the discontinued Citra emulator. However, save states are not transferable between Citra installations. Thus, you can open a CST file only with the instance of Citra that created it. (And you cannot transfer CST files between installations or devices.)

To load a save state from within Citra, load the associated game ROM, and then select Menu→ Emulation → Load State. Citra will then show you available CST files for that game.

ContentServ Template

Developer ContentServ
Popularity
2.7  |  3 Votes
 

Template for creating a new ContentServ document; may be used by an Enterprise Marketing Management Solutions (EMMS) project for managing marketing, sales, and communication processes.

More Information

ContentServ templates include CSTags, which define content that is replaced by output from the template engine when the document is accessed.
