What are ALB files and how to open them

Are you having problems opening an ALB file or are you simply curious about its contents? We're here to explain the properties of these files and provide you with software that can open or handle your ALB files.

What is an ALB file?

ALB files have multiple uses, and Cubase Arrangement File Backup is one of them. Read more about the other uses further down the page.

Cubase Arrangement File Backup

Files that contain the .alb file extension are commonly used by the Cubase music software. Cubase is used to produce and record music tracks and is a type of Digital Audio Workstation.

Cubase uses the .alb extension to create backups of musical arrangement files created within the software application.

In version 5.0r6 the backup naming scheme of Cubase changed. Previously the file extension was changed so that the last letter was changed to a 'b' (for "backup"). For instance, a backup of an arrangement file (*.ALL) would be a (*.ALB) file. Now it will be a '*.ALLB' file.
