ArcGIS Layer Package

Developer ESRI
Popularity
3.0  |  3 Votes
 

What is an LPK file?

GIS data file used by ArcGIS Desktop, a mapping and visualization software package; contains a map layer and its data that can be imported into another map or globe; used for sharing self-contained map layers with others; can be uploaded to the ArcGIS Online service for sharing over the Internet.

More Information

LPK files can be used by both ArcMap and ArcGlobe, which are included as part of the ArcGIS Desktop software.

Tecplot Layout Package

Developer Tecplot
Popularity
2.5  |  2 Votes
 

Visualization package created by Tecplot, a scientific visualization application; saves a plot layout (.LAY file) as well as the data required to generate the visualizations; useful for cataloging images or making large documents.

More Information

You can save Tecplot layouts to LAY files, but these files only specify how to display data and do not include the data itself. Therefore, you can package both the layout and the data into a single viewable file using the LPK format.
