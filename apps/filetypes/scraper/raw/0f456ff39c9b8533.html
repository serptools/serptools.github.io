.THM File Extension

5 file types use the .thm file extension.

Sony Ericsson Theme File

Developer Sony Ericsson
Popularity
3.5  |  124 Votes
 

What is a THM file?

A THM file is a theme file used by Sony Ericsson mobile phones. It stores settings used to customize the user interface of the cell phone and includes custom window and text colors, icons, wallpaper, and other images. THM files can be created and edited using the free Themes Creator from Sony Ericsson, which is no longer developed.

Video Thumbnail File

Developer N/A
Popularity
3.5  |  72 Votes
 

A THM file may also be a thumbnail image generated by digital camcorders and software programs when saving a video clip. It contains a small image from the video clip's first frame and is displayed to help a user identify a clip when visually browsing through a collection of video clips. THM files often are saved in the JPEG format.

More Information

When you are capturing video with a digital camcorder or camera, the device creates a thumbnail of the video and saves it to its memory card or internal storage. Then, when you import video from a camcorder or camera, the software you are using to transfer the video also transfers the associated THM files.

Each THM file transferred from a device to a computer shares the same filename as its associated video clip. For example, a THM file for clip001.avi is created as clip001.thm.

How to open a THM file

You will likely never need to open a THM file since the software program you use to browse video clips displays the thumbnail image next to the video clip.

However, if you encounter a THM file and need to open it individually, you can attempt to open it directly with an image viewer, such as Microsoft Photos, Microsoft Paint, or Apple Preview. If that does not work, you can change the .thm file extension to .jpg and attempt again to open it with an image viewer.

NOTE: If the THM file is not saved in JPEG format, you will likely be unable to open it with an image editor.

How to convert a THM file

If the THM file is saved in JPEG format, you can convert it to other formats with many image editors, including Microsoft Paint, Adobe Photoshop, and Apple Preview.

For example, Paint can convert THM files saved in JPEG format to the following formats:

  • .JPG - JPEG Image
  • .JFIF - JPEG File Interchange Format
  • .PNG - Portable Network Graphic
  • .GIF - Graphical Interchange Format
  • .TIF - Tagged Image File Format
  • .BMP - Bitmap Image
  • .HEIC - High Efficiency Image Format
  • .DIB - Device Independent Bitmap

Generic Thumbnail Image

Developer N/A
Popularity
3.0  |  23 Votes
 

A THM file may also be a thumbnail image, which is a small, icon-sized image that represents a larger image. It is saved in a bitmap format similar to a .JPG file but at a lower resolution. Digital camera and photo or video-editing apps often create THM files to store small previews of larger images and videos.

More Information

The purpose of THM thumbnails is to provide a quick preview that gives you an idea of the image, video, clip art, etc., you are browsing. Since the thumbnail is saved at such a low resolution, it is not meant to be opened and viewed the same as the original file for which it stores the preview.

How to open a THM file

You can open THM files with various image viewers and editors, such as Microsoft Photos (Windows) and Apple Preview (Mac). For example, to open a THM file with Preview, select File → Open.

How to convert a THM file

Apple Preview and Microsoft Photos can convert THM files to other formats.

For example, Preview can convert THM files to the following formats:

  • .JPG - JPEG Image
  • .PNG - Portable Network Graphic
  • .TIF - Tagged Image File Format
  • .EXR - OpenEXR Image
  • .HEIC - High Efficiency Image Format
  • .PDF - Portable Document Format

GoPro Thumbnail Image

Developer GoPro
Popularity
2.0  |  1 Vote
 

A THM file may also be a raster image saved in the GoPro Thumbnail format. It stores image data similar to the JPEG format but also contains GoPro-specific metadata, including the date and time it was captured and the settings of the camera. GoPro cameras, such as the HERO2 and HERO3, automatically generate THM thumbnail images from captured footage to help users preview images and video.

More Information

Select GoPro cameras create THM and .LRV files to accompany captured footage to aid you as you browse through your footage on the camera screen or your mobile phone (using the Quik app). For example, a GoPro HERO2 user will find a THM and LRV file next to their .MP4 video.

The THM file is a small preview (typically 160x120) created from a single frame in your footage that is also small in size to allow for quicker loading times as you scroll. Additionally, the LRV file is a low-resolution video utilized during the editing process to reduce the amount of computing power required to edit a video.

Deleting GoPro THM files

You can delete THM files to remove clutter, but it may affect the speed at which you can preview footage on the camera or in the Quik app. However, the actual footage will not be affected.

Common THM Filenames

GOPR0[numbered video].THM - GoPro cameras name the THM file after the MP4 file it accompanies. For example, if you just recorded the 120th video, GoPro names the MP4 video GOPR0120.MP4 and the thumbnail GOPR0120.THM.

How to open a THM file

THM files are not meant to be manually opened. Instead, GoPro cameras and the Quik app (iOS and Android) reference them for quicker loading times as you browse through footage.

However, since the files are saved in a JPEG format, you may be able to view them with an app that supports JPEG images.
