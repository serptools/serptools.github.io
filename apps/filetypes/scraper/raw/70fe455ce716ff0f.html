.PMF File Extension

3 file types use the .pmf file extension.
  • 1.ESRI Published Map File
  • 2.PlayStation Portable Movie
  • 3.Partition Image File

ESRI Published Map File

Developer ESRI
Popularity
3.6  |  17 Votes
 

What is a PMF file?

Read-only map file created with ArcMap, an application included with ESRI Desktop GIS; stores a constrained version of an .MXD map file; can be viewed with the free ArcReader application, which allows maps to be shared with users who have not purchased the commercial ArcGIS software; often emailed to clients or co-workers in collaborative office environments.

More Information

PMF files can limit the data views, layouts, navigation, printing, querying, panning, and zooming capabilities of the map. They may also be password protected. This helps to protect the confidentiality and privacy of data.

Published maps do not contain actual geographical data. Instead, they link to data sets at a central remote location, which may be hosted by an ArcMap Server or available over the Internet. Therefore, users need appropriate network access to the referenced data repository to view the maps.

NOTE: PMF files are actually created by ArcGIS Publisher, an extension included with ArcMap.

PlayStation Portable Movie

Developer Sony
Popularity
3.7  |  15 Votes
 

A PMF file may also be a video saved in the PlayStation Movie Format (PSMF). It contains a movie, music video, video game cutscene, or other video meant to be played on the Sony PlayStation Portable (PSP).

More Information

The PlayStation Portable is a handheld gaming and entertainment console that Sony released in 2005. It allows users to download and play movies and other videos. These videos (and videos stored on PSP discs) are saved in the PSMF file format and most often use the .pmf extension. (Some PSMF videos instead use the .PMP extension).

Users who want to extract PMF files from PSP disk images can do so using utilities such as UMDGen. They can then play the videos their PMF files contain on their PCs.

How to open a PMF file

You can open PMF files and play the videos they contain with VideoLAN VLC media player (multiplatform) or MPlayer (multiplatform).

How to convert a PMF file

You can use VideoLAN VLC media player to convert PMF files to other video and audio formats, including:

  • .MP4 - MPEG-4 Video
  • .WEBM - WebM Video
  • .TS - Video Transport Stream
  • .ASF - Advanced Systems Format
  • .OGG - Ogg Vorbis Audio
  • .MP3 - MP3 Audio
  • .FLAC - Free Lossless Audio Codec
  • .WAV - WAVE Audio

Partition Image File

Developer Eassos
Popularity
3.3  |  3 Votes
 

A PMF file may also be a partition image file created by Eassos DiskGenius, a program used for partition management and data recovery on Windows. It contains an exact copy of a selected partition, including its file system, structure, and stored data. PMF files are used to back up partitions so they can be restored if the original partition becomes damaged or data is lost.

More Information

IT professionals and system administrators often use DiskGenius and PMF files for backup, recovery, and system migration purposes. Users can create a PMF file by:

  1. Selecting Tools from the menu.
  2. Choosing Clone Partition To Image File
  3. Specifying the source partition, selecting a destination for the image file, and clicking Start.

NOTE: DiskGenius was formerly known as PartitionGuru.

How to open a PMF file

You can open PMF files with Eassos DiskGenius in Windows. To restore a partition from a PMF file:

  1. Select Tools and choose Restore Partition From Image File.
  2. Locate the PMF file and select the target partition.
  3. Click Start.
