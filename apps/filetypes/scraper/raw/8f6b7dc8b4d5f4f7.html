What are GP4 files and how to open them

Are you having problems opening a GP4 file or are you simply curious about its contents? We're here to explain the properties of these files and provide you with software that can open or handle your GP4 files.

What is a GP4 file?

GP4 files have multiple uses, and CCITT Fax Format is one of them. Read more about the other uses further down the page.

CCITT Fax Format

Files that contain the .gp4 file extension are used by a variety of fax software applications. The .gp4 file extension is used to save generic fax image files.

Prior to digital fax software products, users had to have a fax device to send and receive fax messages. Once the Internet evolved, software developers began creating digital fax products, allowing users to send and receive faxes using their computers and an Internet connection.

GP4 files are saved in the CCITT file format. This standardized Group 4 format stores 1-bit or bi-level images using lossless file compression. The GP4 file format is able to compress the fax files into smaller file sizes than older file format versions without losing image quality.

  • GreenPAK Designer Project
  • Guitar Pro v4 Tablature

Guitar Pro v4 Tablature

Guitar Pro 4 allows for the editing of guitar and bass tablature files and the creation of scores for stringed instruments, such as the bass and the guitar. The music files created and saved by the Guitar Pro 4 software are saved with the .gp4 file extension. Guitar Pro 3 used file extension gp3.

While the files contain scores used to create music, the files do not have any actual audio data in and of themselves.
