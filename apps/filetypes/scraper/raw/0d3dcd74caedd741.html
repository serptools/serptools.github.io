What is an MPO file?

An MPO file is a stereoscopic image saved in the Multi Picture Object (MPO) format, which consists of two 2D .JPG images combined into one 3D image. Various digital cameras, such as the Fujifilm FinePix Real 3D series, capture images as MPO files, and Nintendo 3DS may also create them. Users with 3D TVs typically view MPO files as 3D pictures, but various computer applications, such as StereoPhoto Maker and XstereO Player, also support them.

More Information

The Camera & Imaging Products Association (CIPA) developed the MPO format. The specification allows for saving any number of JPG files in an MPO file, but the most common use is for stereoscopic images.

The Fujifilm FinePix Real 3D series cameras are one of the most common cameras supporting Multi Picture Object images. When digital cameras capture images as MPO files, they capture them with the two lenses on the digital camera, then combine and view them side-by-side as a single 3D image.

The image appears in 3D when displayed on a supporting 3D device like a 3D TV. Various companies, including Canon, Sony, Panasonic, LG, Kodak, and Samsung, support the MPO format. Nintendo 3DS may also create MPO files. The files are 640x480, which is the screen size of the device.

NOTE: Another common stereoscopic image format is the Stereo JPEG Image format, which uses the .jps file extension (.JPS files).

How to open an MPO file

You can view MPO files with various image viewers, including StereoPhoto Maker (Windows), XstereO Player (macOS), and StereoPhotoView (Linux). Standard JPEG image viewers, such as Microsoft Photos (Windows) and Apple Preview (macOS) can also open MPO files. However, you may only see one of the images, generally the left image if there is a right and left image.

How to convert an MPO file

Various applications, such as Apple Preview and mpo2jps, can convert MPO files to other formats.

For example, mpo2jps can convert MPO files to .JPS files and Apple Preview can convert MPO files to the following formats:

  • .JPG - JPEG Image
  • .PNG - Portable Network Graphic
  • .TIF - Tagged Image File Format
  • .JP2 - JPEG 2000 Core Image
  • .HEIC - High Efficiency Image Format
  • .EXR - OpenEXR Image
  • .PDF - Portable Document Format

Also, you can convert an MPO file to an anaglyph image (stereoscopic photograph with two images superimposed) with a stereoscopic image processing program, such as StereoPhoto Maker or XstereO Player. After converting the MPO file, you can view the image with red/green 3D glasses.
