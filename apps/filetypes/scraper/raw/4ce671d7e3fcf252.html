Studio Store Visualizer 3D Environment File

Developer Esko
Popularity
2.4  |  5 Votes
 

What is a 3DW file?

A 3DW file contains a 3D environment created by Esko Studio Store Visualizer, a module that can be added to Esko Studio to visualize product designs in retail store environments. It stores information about a 3D retail environment, which includes lighting, floor, aisles, and shelving information.

More Information

3DW files are linked to .VTPR project files. When a VTPR file is opened in Studio Store Visualizer, the program loads the information in the 3DW file. The VTPR file and the linked 3DW file should be located in the same folder. VTPR files are saved by default in the following location:

~/​Documents/​Store Visualizer <version>/​Projects

VTPR templates are located in the following location:

~/​Documents/​Store Visualizer <version>/​Templates

If you share a VTPR file with another user, you must include the linked 3DW file and any linked .L3DW external library files, or else you will not be able to visualize the product in the 3D environment.

3D World Studio Environment

Developer Leadwerks
Popularity
2.5  |  4 Votes
 

A 3DW file may also be a 3D environment created in Leadwerks 3D World Studio, a 3D modeling program used alongside the Leadwerks game engine. It contains 3D models, such as buildings and terrain, lighting details, and other information that comprises an in-game environment. 3DW files are now most notably used in the indie game SCP - Containment Breach.

More Information

Leadwerks is a cross-platform game engine that allows developers to create games for Windows and Linux. In the past, Leadwerks developers could use 3D World Studio to create in-game environments and save those environments as 3DW files. However, 3D World Studio has since been discontinued and replaced by Leadwerks 3. 3D World Studio does not work in Windows 8 or later versions of Windows.

However, some contemporary games still use 3DW files. For example, Joonas Rikkonen's SCP - Containment Breach, an indie horror game, uses 3DW files to load several of its in-game environments. This has led some SCP - Containment Breach enthusiasts to create programs that can open, edit, and convert 3DW files, so they can mod SCP environments.

NOTE: In 3D World Studio, 3DW files could be exported as Leadwerks-, Torque-, 3D Game Studio-, Dark Basic Pro-, and Blitz3D-compatible file types.

How to open a 3DW file

While 3D World Studio does not work on newer versions of Windows, those who want to open and edit a 3DW file can do so in Containment Breach Room Editor (Windows), an unofficial 3DW editor created by juanjp600.
