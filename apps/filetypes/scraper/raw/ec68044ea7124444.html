What are GED files and how to open them

Are you having problems opening a GED file or are you simply curious about its contents? We're here to explain the properties of these files and provide you with software that can open or handle your GED files.

What is a GED file?

GED files have multiple uses, and GEDCOM Family History is one of them. Read more about the other uses further down the page.

GEDCOM Family History File

Files that contain the .ged file extension are GEDCOM genealogy data files. GEDCOM is the abbreviation for Genealogical Data Communications.

GED files usually contain family history and genealogy data saved in a standardized ASCII text format. This allows the files to be opened by various genealogy applications on multiple computer platforms, making it easy to share the family history information online.

The GED files that a user creates can contain various family history facts, such as names, genders, dates of birth, relationships between family members, and other family tree details. In addition, GED files may also contain links to media files relevant to a user's family history.
