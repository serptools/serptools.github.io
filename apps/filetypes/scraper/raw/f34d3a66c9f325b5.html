Pentax Electronic File

Developer Pentax
Popularity
2.8  |  25 Votes
 
NEW: Open your .PEF file online with File Helper.View Online

What is a PEF file?

A PEF file is a raw camera image created by Pentax digital cameras. It stores image information exactly how it was captured by the camera's CCD sensor, with no compression or color adjustment. Professional photographers typically use PEF files because they store higher-quality images.

More Information

PEF files are saved in the Pentax Electronic format utilized by various Pentax camera models. Pentax cameras may store images in slightly different variations of the format. For example, images saved in the Pentax Electronic format may also be saved with a .PTX file extension.

The PEF format is one of many raw image formats utilized by various camera manufacturers. Other raw file types include:

  • Canon - .CR2 and .CR3 files
  • Hasselblad - .3FR and .FFF files
  • Minolta - .MRW and .MDC files
  • Nikon - .NEF and .NRW files
  • Olympus - .ORF file
  • Ricoh - .J6I file
  • Samsung - .MRW and .MDC files
  • Sony - .ARW, .SRF, and .SR2 files

How to open a PEF file

You can open and edit PEF files with various image editing programs, including the software bundled with Pentax digital camera purchases. You can also open PEF files with Microsoft Photos (bundled with Windows) and Apple Preview (bundled with macOS).

Some third-party image editors that support PEF files include Microsoft Photos (bundled with Windows), Adobe Photoshop (Windows and macOS), ACD Systems ACDSee (Windows), and Corel PaintShop Pro (Windows).

PEF Braille Book File

Developer DAISY Consortium
Popularity
2.0  |  1 Vote
 

File created in the Portable Embosser Format (PEF), an XML format used to store braille books in digital form; includes unambiguous embossing specifications; enables braille books to be stored and shared in an unambiguous exchange format.

More Information

PEF was developed to handle some cases of ambiguity from other braille formats, such as ASCII braille.

PEF files can be opened and created using Braille Utils, a Java library for PEF manipulation. PEF files can also be generated from .ODT documents using odt2braille, a plugin for OpenOffice.org 3.0 or later.
