Are you having problems opening a PLY file or are you simply curious about its contents? We're here to explain the properties of these files and provide you with software that can open or handle your PLY files.

What is a PLY file?

PLY files have multiple uses, and Polygon File Format is one of them. Read more about the other uses further down the page.

Polygon File Format

The Standford Graphics Laboratory created the Polygon File Format as an easy way to store 3D polygon data. It is a versatile format applicable to point clouds, polygon meshes, and CAD designs among other things.

PLY supports both ASCII and binary formats and supports multiple element types, properties, and comments, making it possible to store a wide range of data in a single file.

PLY is widely used in the fields of Computer Graphics and Computer Vision, such as 3D scanning, virtual reality, modeling, and digital fabrication. It is also supported by multiple software tools including 3D scanners, 3D modeling programs, and programming libraries.
