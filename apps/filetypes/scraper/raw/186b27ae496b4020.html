What are STP files and how to open them

Are you having problems opening a STP file or are you simply curious about its contents? We're here to explain the properties of these files and provide you with software that can open or handle your STP files.

What is a STP file?

STP files have multiple uses, and STEP Model Data (ISO-10303) is one of them. Read more about the other uses further down the page.

STEP Model Data (ISO-10303)

STEP Model Data is a 3D product model data exchange format. These files contain three-dimensional object data and are supported by a variety of different mechanical design programs.

    All known file formats using extension .STP

    While STEP Model Data (ISO-10303) is a popular type of STP-file, we know of 8 different uses of the .STP file extension. Different software may use files with the same extension for different types of data.

    8 known uses of the STP file extension

    • Dynamic Publisher Stamp
    • Molecular Imaging STP SPM Data
    • SignalTap II Captured Data
    • Spectrum Sound Tracker Pro 2 Chiptune
    • Steem Patch
    • STEP Model Data (ISO-10303)
    • SuperFlow Template
    • Verity Collection Stop List

    Steem Patch

    Steem is an open-source Atari ST emulator for Windows and Linux. It can run almost any Atari ST program. However, programs that do not run out-of-the-box can be updated with a patch. This is what these STP files contain - updates to Atari ST programs so they can be run in the Steem emulator.
