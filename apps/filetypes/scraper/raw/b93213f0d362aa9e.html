What are MCD files and how to open them

Are you having problems opening a MCD file or are you simply curious about its contents? We're here to explain the properties of these files and provide you with software that can open or handle your MCD files.

What is a MCD file?

MCD files have multiple uses, and MiniCAD/VectorWorks 3D Drawing is one of them. Read more about the other uses further down the page.

MiniCAD/VectorWorks 3D Drawing

The .mcd file extension is a 3D graphics and CAD file type directly associated with MiniCAD. MiniCAD is a CAD and building information modeling (BIM) software application developed by Diehl Graphsoft in the mid-1980s. Diehl Graphsoft was acquired by Nemetschek and renamed VectorWorks.

VectorWorks is a global design software primarily used in architectural designs and landscape projections. VectorWorks is used in creating 2D, 3D, production management, and presentation designs. MCD files represent saved drawings and 3D renderings created and saved by using VectorWorks.

MCD files were used VectorWorks version 1 through 12. But now, it uses the VWX file extension in versions 2008 and later. Version 2008 was the successor of version 12.

MCD files can also be converted to other vector-based images and other formats like .mcl, .mcs, .pdf and .vwx. MCD files can be converted to VWX files using newer Vectorworks versions after the 2008 version. VWX and some MCD files can also be viewed in free Vectorworks Viewer programs.

    While MiniCAD/VectorWorks 3D Drawing is a popular type of MCD-file, we know of 6 different uses of the .MCD file extension. Different software may use files with the same extension for different types of data.
