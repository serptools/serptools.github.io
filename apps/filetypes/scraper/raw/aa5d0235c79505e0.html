Affymetrix Probe Results File

Developer Affymetrix
Popularity
3.9  |  31 Votes
 

What is a CEL file?

A CEL file is a data file created by Affymetrix DNA microarray image analysis software. It contains the data extracted from "probes" on an Affymetrix GeneChip and can store thousands of data points, which may make it large in file size. CEL files can be processed by software algorithms and visualized on a 2D grid as part of an overall genome experiment.

More Information

An Affymetrix GeneChip is a biological chip that contains a microarray which is valid for one experiment. To create these chips, a glass or silicon slide is arrayed with probes, which report expression levels (intensities) as perfect match (PM) and mismatch (MM) values, based on whether they are complementary to the original DNA sample. These values may be used for studying DNA alterations, called single-nucleotide polymorphisms (SNPs), which are used in human and animal disease research.

The CEL file format has multiple versions, which use different formatting. For example, version 3 uses ASCII text formatting while version 4 uses a binary format.

NOTE: CEL files require a corresponding .CDF file, which is the dictionary for the raw probe-level data stored in the CEL file. MATLAB has a built-in function called affyread that can be used to read CEL files in the Windows version of the software.

Celestia Script File

Developer Celestia Development Team
Popularity
3.6  |  5 Votes
 

Script used by Celestia, a free space browsing application; written using a specific Celestia syntax and includes commands for moving the viewing perspective throughout the universe; used for creating 3D universe tours.

More Information

CEL files may also include more advanced features, such as instructions to load custom 3D models into the space tour. They can also include captions to provide interesting facts to the observer.

If you download or receive a CEL file, you can open it by double-clicking the file, or by selecting File → Run Script... from the application menu and then choosing the CEL file.

The CEL format has been replaced by the .CELX format in recent versions of Celestia. However, Celestia still supports CEL files. The newer CELX format uses the Lua scripting language, which is more expressive and powerful than the CEL language.

NOTE: CEL files are saved as plain text files and are edited using a text editor.

FLIC Animation

Developer Autodesk
Popularity
1.0  |  5 Votes
 

A CEL file may also be an animation saved in the FLIC animation format, most popularly used by Autodesk Animator and Animator Pro. It contains a series of computer-generated frames. CEL files are compressed using lossless RLE compression.

More Information

Autodesk Animator is a 2D animation program that Autodesk released in 1989 for MS-DOS. The program was succeeded by Animator Pro, which Autodesk released in 1991.

In most cases, Animator and Animator Pro saved animations users created as .FLI and .FLC files, respectively. However, some Animator and Animator Pro animations are instead saved as CEL files. CEL files are identical to FLI and FLC files, except they contain an additional chunk of data in the format's optional file header.

How to open a CEL file

You can open a CEL file with Corel VideoStudio (Windows) and Apple QuickTime Player (Mac). You may have to change the file's extension to .fli or .flc before you can open it.
