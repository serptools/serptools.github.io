What is an SR2 file?

An SR2 file is a raw camera image created by Sony digital cameras such as the R-1. It stores uncompressed image data in the Sony RAW 2 (SR2) format exactly how the camera's sensor CCD captured it. Sony photographers commonly work with SR2 files because they can more accurately edit them than compressed images stored at a lower quality.

More Information

The SR2 format is an extension of the Sony RAW (.SRF) format. In addition, Sony also develops the Alpha Raw (.ARW) image format utilized by select Sony digital cameras, such as the a7R III and a6300 models.

SR2 is one of many RAW image formats utilized by various camera manufacturers. Other formats include:

  • Nikon - .NEF and .NRW files
  • Olympus - .ORF file
  • Ricoh - .J6I file
  • Samsung - .MRW and .MDC files
  • Canon - .CR2 files
  • Minolta - .MRW and .MDC files
  • Hasselblad - .3FR and .FFF files

How to open an SR2 file

You can open SR2 files with the software bundled with the Sony digital cameras that capture images in the SR2 format.

You can also open SR2 files with Microsoft Photos in Windows (with the Raw Image Extension), CyberLink PhotoDirector (Windows and macOS), XnViewMP (multiplatform), and Corel AfterShot Pro (multiplatform).

How to convert an SR2 file

Various image editors and viewers, including Adobe Lightroom and Corel AfterShot Pro, can convert SR2 files to other formats.

For example, Adobe Lightroom can convert SR2 files to the following formats:

  • .JPG - JPEG image
  • .TIF - Tagged Image File Format
  • .DNG - Digital Negative Image
