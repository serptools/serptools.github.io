Mozilla Archive

Developer Mozilla
Popularity
3.6  |  29 Votes
 

What is a MAR file?

Application update file used by Mozilla programs such as Firefox, Thunderbird, and SeaMonkey; contains files that update the program to a new version; run by the Mozilla update program (update.exe in Windows), which is included with Mozilla software installations; stored in a custom file format, but Mozilla has made the specification publically available.

More Information

Mozilla programs automatically download and run MAR files, so the typical user will never manually open a MAR file. However, MAR files can be used to manually update a Mozilla program without opening it, which can be useful for system administrators.

The Mozilla development team generates nightly builds for their software products that include a MAR update package. Therefore, users can apply software updates with the latest release each day.

NOTE: MAR files created by the Mozilla team use the filename convention [program]-[version].complete.mar and [program]-[version].parial.mar for complete and partial software updates.

MSN Explorer Archive

Developer Microsoft
Popularity
2.8  |  4 Votes
 

A MAR file may also be an archive used by MSN Explorer, an integrated web browser, email, and instant messaging client developed by Microsoft. It contains .HTM, .JS, .XSL, .GIF, and other constituent files used to load a section of MSN Explorer. For example, the file mail.mar contains files used to load MSN Explorer's email functionality.

More Information

MSN Explorer is an integrated Internet access application available to those who have an MSN Internet subscription. For example, MSN Explorer is available to those with MSN Dial-Up Internet Access. Microsoft integrated MSN Explorer into Windows XP. Versions of the program are also available for Windows 7, 8, and 10.

The Windows XP version of MSN Explorer stores many of its core files in MAR archives. These archives, and the files they contain, are used to load various MSN Explorer features. MSN Explorer's MAR files can be found in the following directory:

~/​MSN Explorer/​MSNCoreFiles

Common MAR Filenames

mail.mar - Contains files used by MSN Explorer's Mail feature.

ui.mar - Contains MSN Explorer UI elements.

How to open a MAR file

MAR files are not meant to be opened. However, you can use brz's MarEx extraction tool (Windows) to extract the files a MAR file contains. You can also use MarEx to pack files into a MAR file.

Mariner Calc Spreadsheet

Developer Mariner Software
Popularity
2.7  |  3 Votes
 

Spreadsheet created with Mariner Calc, a lightweight spreadsheet application for Mac OS X; contains data organized into rows and columns; supports functions, charts, named ranges, sorting, and auto-summation.

More Information

MAR files are saved in a proprietary format that can only be opened with Mariner Calc.
