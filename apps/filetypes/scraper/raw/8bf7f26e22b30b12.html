.FNC File Extension

3 file types use the .fnc file extension.

  • 2.Vue Functions File

QuickCAM 3D CNC File

Developer Denford
Popularity
3.4  |  5 Votes
 

What is an FNC file?

Computer numerical control (CNC) file used by QuickCAM software; contains geometry information and instructions for a CNC milling machine; paths can be created from 3D models or 2D images.

Vue Functions File

Developer E-on Software
Popularity
3.5  |  4 Votes
 

File used by E-on Vue, a 3D environment editing application; contains instructions that generate visual properties of a 3D scene; may apply noise, turbulence, or other patterns; used for storing reusable functions that can be applied to multiple scenes.

More Information

Vue functions can be edited visually in the Function Editor. However, this component is only included with some versions of the Vue software.

Programs that open or reference FNC files

E-on Vue

Category: 3D Image Files
Updated: November 9, 2010

Frogans Player Network Certificate

Developer STG Interactive
Popularity
2.3  |  4 Votes
 

Certificate that permits the initialization of frogans networks; frogans are elements that appear on the Desktop and are written in the Frogans Slide Description Language (FSDL); the Frogans Network System Language (FNSL) is used to operate frogans networks.

More Information

FNC files authorize connections to the main frogans network where addresses start "frogans*" on clients' systems.

Programs that open or reference FNC files

Windows
STG Interactive Frogans Player
Mac
STG Interactive Frogans Player
Linux
STG Interactive Frogans Player
Category: Settings Files
Updated: July 7, 2008
