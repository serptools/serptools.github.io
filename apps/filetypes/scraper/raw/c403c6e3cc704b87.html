ADPCM Compressed Audio

Developer Actions Semiconductor
Popularity
3.8  |  31 Votes
 
NEW: Open your .ACT file online with File Helper.View Online

What is an ACT file?

An ACT file is an audio file recorded by a Chinese MP3 player, MP4 player, or voice recorder. It typically contains a voice memo or a song recorded off the radio. ACT files are compressed using Adaptive Differential Pulse Code Modulation (ADPCM) and have a bitrate of 8 kbit/s.

More Information

Various models of Chinese media players (often referred to as S1 MP3 players) and voice recorders can save audio recordings in the ACT format. This lossy format was developed by Actions Semiconductor and is usually a device's lowest-quality recording format. (Audio recorded at higher quality is typically saved as a .WAV file.)

To create an ACT file, users record audio using either a device's built-in microphone or FM radio. They can then play the ACT file on their device or transfer it to their computer, where they can convert it to the WAV format using conversion software included with their device.

How to open an ACT file

You can open most ACT files with File Viewer Plus (Windows), Logipole Konverter (Windows), or SoX (Linux). However, these programs may not be able to open ACT files created by later devices, which are saved in a different format than previous devices' ACT files.

How to convert an ACT file

You can use File Viewer Plus (Windows), Logipole Konverter (Windows), or SoX (Linux) to convert most ACT files to the .WAV format and other audio formats. However, you may not be able to open or convert ACT files saved in more recent versions of the ACT file format.

Adobe Color Table File

Developer Adobe Systems
Popularity
3.6  |  9 Votes
 

Color lookup table (CLUT) file used by Adobe Photoshop, an application used for editing raster graphics; stores a collection of pre-defined colors; used for applying a CLUT to an image in the "Save for Web" dialog in Photoshop.

More Information

ACT files are used for optimizing images for Web publishing. By adding or reducing the number of colors in the CLUT, users can examine the quality of the image versus the resulting file size before saving it.

Adobe color tables can be loaded in the Save for Web dialog box by selecting the Color Table menu and choosing Load Color Table.... Several predefined ACT files are included with the Photoshop installation in the /​Presets/​en_US/​Save for Web Settings/​Color Tables/​ directory.

NOTE: CLUTs are used by some image formats in the Save for Web dialog, such as .GIF. However, they are not used by .JPG and .PNG files.

DS Game Maker Action File

Developer Invisionsoft
Popularity
3.7  |  7 Votes
 

File used by DS Game Maker, an application used to create Nintendo DS games; contains a game action written in C; used for displaying graphics, playing sounds, creating transitions, and controlling game logic, as well as other actions.

More Information

Custom actions can be created from templates provided in the DS Game Maker Action Editor interface. ACT files are stored in the \​Actions\​ directory of the DS Game Maker installation.

ACT files are saved with a corresponding .ACTX file, which contains a description for the action.

NOTE: Invisionsoft DS Game Maker is no longer developed.

Genesis3D Actor File

Developer Eclipse Entertainment
Popularity
3.4  |  7 Votes
 

Three-dimensional character, or "actor," saved in the Genesis 3D format; may include object motions, skins, and materials; actor files can be created directly within Genesis3D or by converting other 3D formats to the ACT format using Genesis3D Actor Studio.

More Information

Genesis3D is an open-source real-time 3D rendering engine used primarily for creating 3D games. The Genesis 3D engine pack includes the Actor Studio, Actor Viewer, and other files. More information is available at the Genesis3D website.

FoxPro Documenting Wizard Action Diagram

Developer Microsoft
Popularity
2.8  |  4 Votes
 

An ACT file may also be a diagram created by the Documenting Wizard in Microsoft Visual FoxPro to represent the structure and relationships within the code of a FoxPro application. It contains information that shows how various application components are connected or related, such as functions, procedures, and data structures. Developers reference ACT files to better understand and document their application's internal workings, making it easier to maintain or modify the code in the future.

More Information

ACT diagrams are particularly helpful for documenting large or complex projects, especially for collaboration. For example, a lead developer may generate an ACT file using the Documenting Wizard (Tools → Wizards → Documenting) in Visual FoxPro to show a new developer joining the team how the application's different functions, procedures, and tables interact. The new developer can then use this diagram to quickly grasp the relationships between components, making it easier to navigate the codebase and implement modifications.

NOTE: Microsoft published the last version of Visual FoxPro (9.0) in 2007.

How to open an ACT file

You can open ACT files with Microsoft Visual FoxPro through the Documenting Wizard tool. However, the software is discontinued.
