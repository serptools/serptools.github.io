What is a W32 file?

Developer file used by Visual Studio, Make, and other programs used for building Win32 (Windows) application executables (.EXE files); saved in a plain text format and contains instructions for constructing the EXE file; often named Makefile.w32.

WinCAPs Subtitle File

Developer Sysmedia
Popularity
3.1  |  7 Votes
 

Native subtitle file format used by Sysmedia WinCAPs subtitle preparation software; stores subtitle information and timecodes, as well as other settings; used for adding subtitles and closed captions to both recorded video files and live video streams.
