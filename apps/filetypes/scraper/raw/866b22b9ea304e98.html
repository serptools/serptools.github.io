AbiWord Document

Developer AbiSource
Popularity
3.1  |  62 Votes
 

What is an ABW file?

An ABW file is a word-processing document created by AbiSource AbiWord, a free word processor developed by the AbiSource community. It contains formatted text and supports advanced document layouts, images, tables, lists, and footnotes. AbiWord users often save reports, letters, memos, and other documents as ABW files.

More Information

AbiWord is an open-source word processor with many standard features, including text-formatting tools and spell check. While older versions of the application are available for Windows, macOS, and Linux, AbiSource only actively develops the software for Linux since version 3.

How to create an ABW file

ABW is the default file format in which AbiWord saves documents. To create an ABW file with AbiWord:

  1. Select File → New or New using Template.
  2. Modify the document.
  3. Select File → Save or Save As.

How to open an ABW file

You can open ABW files with AbiSource AbiWord (multiplatform). To open an ABW file, select File → Open.

Since the ABW format is based on the XML standard, you can also open ABW files with XML, source code, and plain text editors. However, these programs will not display the document the same as AbiWord. Therefore, you may only want to open an ABW file with an XML, source code, or plain text editor if you need to extract the text from the document and do not have access to AbiWord.

How to convert an ABW file

AbiWord can convert ABW files to various file types, including the following formats:

  • .DOCX - Microsoft Word Document
  • .DOC - Microsoft Word Document (Legacy)
  • .ODT - OpenDocument Text Document
  • .RTF - Rich Text Format
  • .TXT - Plain Text
  • .HTML - HTML Webpage
  • .MHTML - MIME HTML Archive
  • .AWT - AbiWord Template
  • .ZABW - Compressed AbiWord Document
