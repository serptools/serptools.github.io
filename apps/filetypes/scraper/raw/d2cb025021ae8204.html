StarMath Formula File

Developer Oracle
Popularity
4.1  |  27 Votes
 

What is an SMF file?

Math formula created by OpenOffice Math and other variants of the office suite, such as LibreOffice Math; stores the syntax and formatting of a mathematical equation or expression; replaced with the newer .ODF format, but is still supported by Apache OpenOffice.

More Information

NOTE: The SMF format was originally used by StarMath through StarMath version 5. StarMath 6 and Apache OpenOffice version 1 adopted the .SXM format, which was then replaced by the OpenDocument ODF format. Apache OpenOffice still supports the SMF format even though it has been deprecated.

NOTE: Apache OpenOffice was formerly known as OpenOffice.org.

Standard MIDI File

Developer MMA
Popularity
3.7  |  14 Votes
 

An SMF file contains Musical Instrument Digital Interface (MIDI) data, which is a standard for digital music communication. It is similar to the common .MIDI file (and .MID files), storing musical performance data, such as note sequences, tempo, volume, and instrument information. However, SMF files typically contain more complex music compositions.

More Information

Musicians, producers, and composers may use the SMF format since it allows playback across various devices and platforms that support MIDI. Additionally, there are different types of SMF files, with variations for different sequencing and playback needs.

NOTE: The MIDI Manufacturers Association (MMA) develops the SMF format.

How to open an SMF file

You can open SMF files with various digital audio workstations (DAWs) and MIDI sequencers, such as Image-Line FL Studio, Steinberg Cubase, and Ableton Live. For example, you can open an SMF file with FL Studio by selecting File → Open.

Serif Metafile

Developer Serif
Popularity
2.9  |  7 Votes
 

Vector image created in the Serif Metafile (SMF) format, a proprietary format that extends the Windows Metafile (WMF) format to include better line, fill, and text definitions; used for sharing vector graphics between Serif applications, including DrawPlus and PagePlus.

More Information

Serif discontinued PagePlus and replaced it with Affinity Publisher in 2017. Serif also discontinued DrawPlus in 2015. The last version of the software was X8.

SkinMagic Toolkit File

Developer Appspeed
Popularity

Shortcut Maker Collection

Developer Rushikesh Kamewar
Popularity
3.5  |  2 Votes
 

An SMF file may also be a shortcut collection created by Shortcut Maker, an Android app that allows users to create custom home screen shortcuts for apps, settings, activities, and other Android features. It contains structured data that defines the behavior and appearance of each shortcut in a saved collection, such as the shortcut’s label, icon, target activity or intent, and any custom settings the user applied. By saving shortcuts in an SMF file, users can back up their configurations or transfer them between devices and users without having to recreate everything manually.

More Information

Android power users or those who enjoy customizing their device’s home screen layout are the primary users of Shortcut Maker. When a user saves a group of shortcuts they’ve configured, the app stores them in an SMF file. Shortcut Maker and SMF files make it easier to organize access to apps and actions, especially on devices with many installed apps or complex workflows.

How to open an SMF file

You can open SMF files with Shortcut Maker on an Android device. Shortcut Maker automatically loads SMF files, but if you need to restore shortcuts saved in an SMF file:

  1. Tap on the History tab located at the bottom of the screen.
  2. Tap the three-dot menu icon in the top-right corner.
  3. Select Restore from the dropdown menu.

Once you restore the shortcuts from an SMF file, they will appear in your Shortcut Maker history. From there, you can review, edit, or add them to your home screen as needed.
