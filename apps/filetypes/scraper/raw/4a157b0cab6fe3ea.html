Raptor Flowchart File

Developer Martin Carlisle
Popularity
3.3  |  33 Votes
 

What is a RAP file?

A RAP file is a flowchart created by RAPTOR, a flowchart-based programming environment. It contains a flowchart, which is a collection of connected symbols that represent instructions to be executed and when they are to be executed.

More Information

The RAP file is the main file type associated with Raptor. It is created in order to store a flowchart, which is used for designing a Raptor program. When you initially create a flowchart it only contains "Start" and "End" symbols. You can add symbols in between those symbols, such as "Assignment", "Call", "Input", "Output", "Selection", and "Loops".

You can create a RAP file by selecting File → New and save it by selecting File → Save or Save As.

You can open a RAP file by selecting File → Open.

The RAPTOR program is designed to help students visualize algorithms.

Scarab Ransomware Encrypted File

Developer N/A
Popularity
4.0  |  4 Votes
 

A file with a .rap extension is a file that has been encrypted by a variant of Scarab ransomware, a type of malware utilized by cybercriminals. It is encrypted with AES and RSA 2048-bit encryption so it is not possible to open the file by simply changing the file extension.

More Information

Scarab ransomware renames standard files that are important to the victim with a .rap extension. The files are typically documents, images, videos, and backup files, such as .AVI, .DOC, .XLSX, and .JPG files.

The purpose of ransomware is to take your files hostage and force you to pay the perpetrator to unlock your files. It may be introduced to your computer through an executable file disguised as another type of file. This may be through spam emails, counterfeit advertisements, or fraudulent system requests.

Once the executable file is run, it begins scrambling your files, renaming them with the .rap extension, and encrypting them. The virus then generates a .TXT ransom note (HOW TO RECOVER ENCRYPTED FILES.TXT) informing you of the takeover and what you are meant to do to recover your files.

NOTE: RAP files became prevalent in late 2018 and 2019 and are similar to BOMBER files.

How to open a RAP file

No known program can restore RAP files to their original state. The best way to recover your files is to retrieve them from a recent backup or perform a System Restore to a point before your computer was infected.

Warning

FileInfo.com recommends you never pay a ransom to decrypt files encrypted by ransomware. Paying a ransom encourages ransomware distributors to continue their efforts, and there is no guarantee that paying a ransom will give you access to your files. Instead, you should restore your data from a recent backup created before the ransomware infected your computer.

PlayStation 3 Game Rights File

Developer Sony
Popularity
1.5  |  10 Votes
 

A RAP file contains a decryption key that verifies ownership of a PlayStation 3 (PS3) game. RAP files are packaged with a game, and the PS3 console references the RAP file when loading the game to allow the user to play the game. While most PS3 gamers will never encounter RAP files, gamers attempting to play games with PS3 emulators and Homebrew-enabled PS3 systems need the appropriate RAP file to activate the associated game.

More Information

Sony uses RAP, Rights Information File (RIF), and Sony Encrypted Data (.EDAT files) to apply Digital Rights Management (DRM) protection to PS3 games and system data. When you load a game downloaded from the PlayStation Network (PSN), the PS3 references the decryption key to verify ownership of the game.

NOTE: Using RAP files you did not purchase is piracy and illegal.

How to open a RAP file

You should not attempt to open RAP files. Instead, you can activate a PS3 game stored in a PKG file on your Homebrew-enabled PS3 by installing the appropriate RAP file with the reActPSN or PSNPatch utilities on your PS3. Or, you can load a RAP file in RPCS3, a PS3 emulator for Windows and Linux.

NOTE: Do not rename the RAP file or else it will become invalid.
