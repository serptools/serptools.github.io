PDFill Project

Developer PlotSoft
Popularity
3.5  |  26 Votes
 

What is a PFL file?

A PFL file is a PDF editing project created by PlotSoft PDFill. It contains a .PDF file and a record of edits made to that file. Edits may include insertions, deletions, highlighting, and comments.

More Information

PDFill is a PDF editor available for Windows. It allows users to import and edit a PDF file and then export the edited version of their document.

While editing a PDF, PDFill users can save their current editing project as a PFL file. Doing so allows the user to retain a copy of their edits and make further edits in the future (by reopening the PFL file).

How to open a PFL file

You can open a PFL file with PlotSoft PDFill (Windows). To do so:

  1. Select File → Open Project to Resume Editing from PDFill's menu bar.
  2. Navigate to and open your PFL file.

How to convert a PFL file

PlotSoft PDFill can export the PDF that a PFL file contains in several formats, including:

  • .PDF - Portable Document Format
  • .PNG - Portable Network Graphic
  • .JPG - JPEG Image
  • .GIF - Graphical Interchange Format
  • .BMP - Bitmap Image

To export as a PDF, select File → Save the edited PDF As. To export in other formats, select File → Export.

PhotoFiltre Studio Plugin

Developer PhotoFiltre
Popularity
3.2  |  5 Votes
 

A PFL file may also be a plugin used by PhotoFiltre Studio, a simple image editing program. It contains information that enhances PhotoFiltre Studio's built-in features, extending the software's functionality when a user installs it. Examples of PFL plugins include tools, effects, or adjustments, such as color correction, sharpening, or artistic filters.

More Information

PhotoFiltre Studio is a free image editing tool designed for basic photo manipulation tasks. It provides various tools and features, such as filters, color adjustments, cropping, and special effects like blur, sharpening, and color correction.

Users, such as graphic designers, photographers, and hobbyists, looking for tools not included with PhotoFiltre Studio's core functionality, can install plugins via PFL files. Once they install the plugins, they can access the new effects or tools in the software's interface.

How to open a PFL file

You can open PFL files with PhotoFilter Studio in Windows. To use a PFL file, you must place it in the Plugins folder in the PhotoFiltre Studio installation location (likely C:\​Program Files (x86)\​PhotoFiltre).

Family Lawyer Document

Developer Nolo
Popularity
3.0  |  6 Votes
 

Legal document created by Quicken WillMaker or Family Lawyer do-it-yourself legal document preparation software; can be used to store a will or estate management document, or other legal documents.

More Information

PFL files are used by older versions of the Quicken WillMaker and Family Lawyer software. Quicken WillMaker 2006 was still capable of importing older PFL files.

NOTE: Family Lawyer software was originally developed by Intuit but was transferred to Broderbund and eventually discontinued.
