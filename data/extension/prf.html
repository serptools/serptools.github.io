Outlook Profile File

Developer Microsoft
Popularity
3.6  |  43 Votes
 

What is a PRF file?

A PRF file is a user profile referenced by Microsoft Outlook that contains user information and preferences. It includes mail account settings, signatures, references to mail folders, and other settings.

More Information

When Outlook imports a PRF file, the user profile settings are immediately updated. PRF files can also be created using Microsoft's Office Resource Kit. The files can then be deployed to multiple systems using the Custom Installation Wizard (CIW) or other maintenance wizard included with the Office Resource Kit.

Windows System File

Developer Microsoft
Popularity
3.5  |  11 Votes
 

A PRF file may also be a settings file used by Microsoft Windows or another Microsoft program. It stores a user's Windows or program preferences.

How to open a PRF file

PRF files are not meant to be opened. Attempting to open, edit, or move a PRF file may cause the program that uses that PRF file to function incorrectly.

PICS Rules File

Developer W3C
Popularity
3.0  |  10 Votes
 

Web profile created in the Platform for Internet Content Selection (PICS) format; contains filtering rules that allow or block access to certain URLs; used for implementing Internet safety policies, such as parental or teacher controls, which limit access to certain websites for students and children; also used for sharing PICS profiles with others.

More Information

The PICS format was proposed in the mid 1990s, and was adopted by some Web technologies. However, it is not widely used today and has been superseded by the Protocol for Web Description Resources (POWDER).

NOTE: PRF files were supported by some Web browsers and Web servers, such as Internet Explorer and Microsoft IIS.

Korg Keyboard Performance

Developer Korg
Popularity
3.5  |  2 Votes
 

A PRF file contains a performance for a Korg keyboard, such as the PA80 or 600 models. It stores the settings that comprise a performance playable on a keyboard. Korg users primarily play them on a keyboard but may view them with KORG PA Manager on their computer.

More Information

Korg produces various electronic musical instruments, including keyboards and synthesizers. The PA keyboard models support various file types that store different information, such as styles (.STY) and sounds (.PCG).

How to open a PRF file

You can open PRF files with KORG PA Manager in Windows. To open a PRF file with KORG PA Manager, click "File Viewer" and then select SET Browser.
