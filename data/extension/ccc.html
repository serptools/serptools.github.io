Dev-C++ Code Completion Cache

Developer N/A
Popularity
2.4  |  22 Votes
 

What is a CCC file?

Cache file used by Dev-C++, , an open-source Integrated Development Environment (IDE) for C/C++ software programming; contains data utilized by the program to complete C++ code, which enables quicker development.

More Information

The cache file is created when you accept the autocompletion function when Dev-C++ is first run. After accepting the function, the program will parse some files and create the CCC cache. The file is located in the \%APPDATA%\Dev-Cpp\ directory.

Comic Chat Conversation

Developer Microsoft
Popularity
3.0  |  2 Votes
 

A CCC file may also be a transcript of a chat conversation conducted using Microsoft Comic Chat, a discontinued IRC application. It can be opened and viewed as either a text transcript or as a comic strip. CCC files can be opened only in Comic Chat.

More Information

Microsoft Comic Chat allowed users to conduct online chats using avatars that looked like comic book characters. The program automatically adjusted each avatar's expression, placement, and gestures based on the content of participants' conversations. Users could also manually set their avatars' emotions using Comic Chat's emotion wheel.

Comic Chat users could save records of their chat conversations. These records were saved as CCC files. Unfortunately, because Comic Chat has been discontinued since 1999, there are no other known programs that can open CCC files.

How to open a CCC file

You can open a CCC file in Microsoft Comic Chat (Windows), which is still available online as an unofficial, third-party download. Depending on which version of Windows you use, you may need to run Comic Chat in Compatibility Mode.
