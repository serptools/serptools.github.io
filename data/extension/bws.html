What is a BWS file?

Supplementary disk image file created with BlindRead or BlindWrite software; contains sub codes, or sub channel data, related to a specific disc image; optionally saved along with a .BWI and .BWT file.

Brady Workstation Label

Developer Brady
Popularity
3.0  |  1 Vote
 

A BWS file may also be a printable label created by Brady Workstation or Brady Express Labels. It contains an editable product or material label, which a user can send to a Brady label printer for printing. For example, a Brady Express Labels user can send a BWS file to a Brady M211 printer, to print the label the file contains.

More Information

Brady Workstation and Brady Express Labels are label design programs that allow users to create and print labels for products, laboratory materials, hazardous materials, and other items. These labels may consist of icons, barcodes, text, and other graphic design elements.

Each label a user creates in Brady Workstation and Brady Express Labels is saved as a BWS file. After a user finishes creating a label, they can send that label's BWS file to a compatible Brady label printer, to print that label. Brady Workstation is compatible with all Brady label printers except the M211. Brady Label Express is compatible with the Brady M211, BMP51, M610, M611, M710 and BMP61 printers.

How to open a BWS file

You can open a BWS file with Brady Workstation (Windows) or Brady Express Labels (Android, iOS), to edit the label it contains or send it to a compatible Brady label printer. (See above for a list of printers with which each program is compatible.) Brady Workstation is available as a free trial; Brady Express Labels is entirely free.
