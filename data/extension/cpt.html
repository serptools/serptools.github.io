Corel PHOTO-PAINT Image

Developer Corel
Popularity
3.3  |  44 Votes
 

What is a CPT file?

A CPT file is a bitmap image created by Corel PHOTO-PAINT, a photo editing program included with CorelDRAW Graphics Suite. It contains an image that supports layers, alpha channels, transparency, and other effects. CPT files are the native files PHOTO-PAINT saves to store images users create or modify with the software.

More Information

Amateur and professional photographers and graphic designers may use Corel PHOTO-PAINT to create or edit images for various purposes. Examples include advertisements, photo collages, social media and web graphics, and artistic illustrations.

When you save an image with PHOTO-PAINT, the application creates the CPT file, by default, to store the image and all the edits you have applied. After saving the CPT file, you can close it and re-open it to apply future edits.

How to create a CPT file with PHOTO-PAINT

To create a CPT file with PHOTO-PAINT:

  1. Select File → New.
  2. Name the file, choose the dimensions and other settings, then click OK.
  3. Edit the image.
  4. Select File → Save or Save As, choose the save location, and click Save.

How to open a CPT file

You can open a CPT file with the Corel PHOTO-PAINT application included with CorelDRAW Graphics Suite (Windows and macOS). To open a CPT file with PHOTO-PAINT, select File → Open.

You can also open CPT files with Corel Presentations (Windows) and XnViewMP (multiplatform).

How to convert a CPT file

Several applications, including CorelDRAW Graphics Suite and XnViewMP, can convert CPT files to various formats.

For example, the PHOTO-PAINT application included with CorelDRAW Graphics Suite can convert CPT files to many file types, including the following formats:

  • .JPG - JPEG Image
  • .PNG - Portable Network Graphic
  • .BMP - Bitmap Image
  • .GIF - Graphical Interchange Format
  • .TGA - Targa Graphic
  • .TIF - Tagged Image File Format
  • .WEBP - WebP Image
  • .EPS - Encapsulated PostScript File
  • .PSD - Adobe Photoshop Document
  • .PDF - Portable Document Format

Ccrypt Encrypted Archive

Developer Peter Selinger
Popularity
3.2  |  14 Votes
 

File archive encrypted with ccrypt, a utility for encrypting and decrypting files and streams; based on the Rijndael cipher and uses an asymmetric encryption algorithm; requires a password to open.

More Information

Ccrypt is developed as an open-source program and is available for Mac, Windows, and Unix systems.

Compact Pro Archive

Developer Cyclos
Popularity
2.4  |  10 Votes
 

File archive compressed with Compact Pro, a file compression program for Mac OS Classic (Mac OS 9 and earlier); uses a fast and efficient compression algorithm.

More Information

Development of Compact Pro stopped in the late 1990s and the program is no longer supported.

DTS Compact Audio File

Developer DTS
Popularity
1.4  |  5 Votes
 

Audio file used by DVD authoring programs; contains audio, timecode, and header data; used in conjunction with video data to create DVDs.

More Information

NOTE: CPT files cannot be used to author Blu-ray discs. The .DTSHD format can be used instead.

Pocket Slides Presentation

Developer Conduits Technologies
Popularity
1.0  |  5 Votes
 

A CPT file may also be a presentation created by Conduits Pocket Slides, a mobile presentation software designed for Pocket PC devices, specifically PDAs. It contains information about one or more slides, including text, images, and formatting details that make up the presentation.

More Information

Pocket Slides software was popular in the early 2000s because it offered mobile users a way to create, view, and edit presentations on the go. The program supports various presentation features, including slide transitions, and text formatting, making it a helpful tool for professionals and students needing mobile presentation options.

Due to the rise of more advanced smartphones and the shift away from Windows Mobile as a major platform, Pocket Slides and its CPT format gradually became obsolete by the mid-to-late 2000s. Conduits Technologies actively developed the program until around 2007, with version 2.51 being its last major release.
