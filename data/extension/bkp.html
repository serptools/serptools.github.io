Backup File

Developer N/A
Popularity

What is a BKP file?

BKP files are typically backup files, which contain a redundant copy of a document, database, project, or other data. Some applications automatically create BKP files at regular intervals, to back up users' data. This allows users to restore their data if the original version of the backup file goes missing or becomes corrupted.

More Information

Programs usually store BKP files alongside the data they're backing up or in a backup-specific directory (e.g., Backups or Backup). In most cases, BKP files have filenames similar to the files for which they are a backup. For example, a BKP file created from database.db would likely be named database.db.bkp or database.bkp.

How to open a BKP file

In many cases, you can open a BKP file by:

  1. Removing .bkp from its filename.
  2. Opening it as you would a normal version of that file.

For example, you can open database.db.bkp by changing its name to database.db and opening it as you would a DB file. (If your BKP file uses only the .bkp extension, instead rename the file to use its normal extension.)

In other cases, however, you must follow program-specific instructions to open a BKP file. In these cases, refer to your program's backup documentation.

Zapback Backup File

Developer Zaplots
Popularity
3.3  |  24 Votes
 

Backup file created by Zapback, a simple Windows backup and restore utility; contains an uncompressed copy of the original file, but with the string " - Backup.bkp" appended to the original filename.

More Information

BKP files can only be used for creating backups of single files.

Backup Your Mobile Backup Data

Developer Artur Jaszcyzk
Popularity
3.0  |  3 Votes
 

A BKP file may also be a data file created by Backup Your Mobile, a backup app for Android devices. It contains a portion of a user's backup data. Backup Your Mobile uses BKP files when restoring users' backups, which do not themselves have a file extension.

More Information

For example, if a Backup Your Mobile user creates a backup that backs up the apps on their device, Backup Your Mobile will also create a BKP file that contains a list of those applications. Backup Your Mobile uses this list to restore the user's applications, as it restores the associated backup.

NOTE: Artur Jaszcyzk discontinued the Backup Your Mobile app.

How to open a BKP file

BKP files are not meant to be opened. However, because BKP files are plain text files, you can open and view them with any text editor. Keep in mind that editing a BKP file may prevent you from being able to restore Backup Your Mobile backups.
