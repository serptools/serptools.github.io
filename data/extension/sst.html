Serialized Certificate Store File

Developer Microsoft
Popularity
3.6  |  11 Votes
 

What is an SST file?

Digital certificate downloaded from a Certificate Authority (CA); installed as a root certificate on Windows computers and is used to validate the identities of websites and software programs; typically saved in a folder along with other .PFX certificate files.

More Information

The SST file format preserves all certificate store properties and is useful for transferring certificates between computers.

Children of the Nile Texture

Developer Tilted Mill Entertainment
Popularity
2.4  |  5 Votes
 

An SST file may also be a texture used by Children of the Nile (CotN), a video game in which players maintain an ancient Egyptian city. It is used to skin an associated .CEM file. SST files are not meant to be opened outside CotN.

More Information

Children of the Nile, also known as Immortal Cities: Children of the Nile, is a city simulation game in which players build buildings, issue edicts, and tend to citizens' needs. Tilted Mill Entertainment released the first edition of CotN in 2004 and an Enhanced Edition in 2008.

The game's SST files are used to add textures to 3D models that represent people, animals, and buildings. For example, an SST file might contain a brick texture used to skin an obelisk.

NOTE: The game Empire Earth also uses SST files. Empire Earth's SST files are formatted slightly differently than CotN's SST files.

How to open an SST file

Children of the Nile SST files cannot be opened outside the game. Empire Earth SST files can be opened and viewed using SST-tool, a fan-made SST library.

How to convert an SST file

You can use SST-tool to convert Empire Earth SST files to the .TGA format. You can also use SST-tool to convert TGA images to the Empire Earth SST format.

Artec Studio Project

Developer Artec 3D
Popularity
2.7  |  3 Votes
 

An SST file may also be a project created in Artec Studio, a 3D scanning, modeling, and data processing application. It contains all of the project's 3D data, including data obtained via scanning and post-processing actions. Versions 16 and above of Artec Studio use SST files; previous versions stored project data in a corresponding folder and A3D project file.

More Information

Artec Studio is a computer-assisted design (CAD) application that allows users to scan a mechanical component, import that scan as a 3D model, and then modify that model using numerous design tools. Artec Studio also allows users to import and export models in other file formats, such as .STEP, for use with other CAD and computer-assisted manufacturing (CAM) programs.

In versions 16 and above, Artec Studio projects are saved as SST files. Each SST file contains all that project's scans, meshes, and CAD objects.

How do I create a new SST file in Artec Studio?

To create a new SST file in Artec Studio versions 16 and above, select File → New project from the program's menu bar. Then, save your new project. Artec Studio will save the project as an SST file.

How to open an SST file

You can open an SST file in Artec Studio versions 16 and above (Windows). To do so, select File → Open project from the program's menu bar.

How do I export the scans, meshes, and CAD objects an SST file contains?

After you open an SST file in Artec Studio, you can export the scans, meshes, and CAD objects your SST file contains in various file formats, by selecting File → Export → Scans, File → Export → Meshes, or File → Export → CAD objects.
