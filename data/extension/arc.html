Norton Backup Archive

Developer Symantec
Popularity
4.0  |  142 Votes
 

What is an ARC file?

An ARC file is a backup file created by Norton software products, such as Norton 360 and Norton Online Backup. It contains a backup of computer files created automatically by the software, which may include personal user files as well as system files. Since ARC files can include all personal and system files, they may be very large in size.

More Information

ARC files are used for storing and recovering files in the event of accidental deletion or system failure. They may be backed up to an off-site location using Norton Online Backup.

NOTE: Norton 360 no longer supports ARC files.

Compressed File Archive

Developer System Enhancement Associates
Popularity
3.5  |  53 Votes
 

An ARC file may also be a compressed file archive created either as a Macintosh Hierarchical File System (HFS) archive or an MS-DOS archive using standard MS-DOS compression. It contains a collection of compressed files, each being compressed potentially with a different compression technique.

More Information

ARC files were commonly used in the late 1980s and early 1990s. However, they are rarely used now and have been replaced by newer, more efficient compression algorithms.

NOTE: While ARC files were used primarily for MS-DOS and older versions of Macintosh computers, several current programs can still open them, such as StuffIt and WinZip.

Minecraft: Legacy Console Edition Archive

Developer 4J Studios
Popularity
3.3  |  7 Votes
 

An ARC file may also be a compressed archive used by discontinued console versions of Minecraft, which are collectively referred to as Minecraft: Legacy Console Edition. It contains UI elements and textures, language files, and structure data.

More Information

In the past, developer 4J Studios published console-specific versions of Minecraft. These versions of Minecraft were named for the console on which they could be played. For example, the PlayStation Vita version of Minecraft was named Minecraft for Playstation Vita.

Players often refer to these versions of Minecraft, which have since been replaced by Minecraft Bedrock Edition, as Minecraft: Legacy Console Edition. All Minecraft: Legacy Console Edition games include ARC archives. While players are not meant to open or edit these files (or the game data they contain), Minecraft: Legacy Edition modders have created tools that allow them to extract assets from and add assets to ARC files.

How to open an ARC file

You can open and manage ARC files with ARC Studio (Windows) or MCCToolChest (Windows).

Internet Archive Web Archive

Developer Mike Burner and Brewster Kahle
Popularity
3.3  |  6 Votes
 

An ARC file may also be an Internet Archive web archive. It contains website content retrieved using crawlers, for archival purposes. This content may include HTML data, images, scripts, stylesheets, and other webpage elements.

More Information

The Internet Archive is an online repository for a variety of archived web content. This content, as well as metadata about how and when the content was retrieved, is stored in ARC files. Typically, ARC files contain several pieces of archived content, aggregated for easier storage and retrieval.

Internet Archive developers Mike Burner and Brewster Kahle created the ARC format. In 2008, the International Internet Preservation Consortium published an updated version of the ARC format, known as the Web ARChive (.WARC) format.

How to open an ARC file

You can open an ARC file with Webrecorder Player (multiplatform). Doing so will allow you to view the archived content the ARC file contains.

WWE 2K18 and 2K19 Cache File

Developer Yuke's
Popularity
2.7  |  6 Votes
 

An ARC file may also be a game data file used by WWE 2K18 and 2K19. It contains cached game data. The games' ARC file is named Chunk0.arc.

More Information

Both WWE 2K18 and 2K19 cache certain game data to speed up the games' startup time. The games store this cached data in and load it from Chunk0.arc.

Most WWE 2K18 and 2K19 players will never encounter or need to modify Chunk0.arc. However, modders who have installed 2K18 or 2K19 mods must delete or clear Chunk0.arc, to clear the games' cache so their installed mods load correctly. Otherwise, the games will crash at startup.

Common ARC Filenames

Chunk0.arc - The name assigned to WWE 2K18 and 2K19's ARC files.

How to open an ARC file

Chunk0.arc is not meant to be opened.
