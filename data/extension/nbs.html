Minecraft Note Block Studio File

Developer David Norgren
Popularity
4.2  |  34 Votes
 

What is an NBS file?

An NBS file contains an audio project created by Minecraft Note Block Studio (NBS) or Open NBS, which are third-party programs used to create custom Minecraft music. It contains note blocks organized to be played as an in-game song. NBS files also contain song metadata, including the song's name, author, and length.

More Information

In Minecraft, note blocks are blocks that play musical notes when attacked or powered by redstone. Minecraft modders can use either version of NBS to create custom Minecraft songs, which are meant to be played by:

  • Importing them as a group of note blocks
  • Using custom commands (with cheats enabled)

Each NBS project is saved as an NBS file. This file lists the note blocks a song uses as well as what notes those blocks play.

Users can play the song an NBS file contains in either version of NBS. Users can also export the song as a:

  • .MP3, .WAV, .AIFF, or .FLAC file
  • MCEdit .SCHEMATIC file
  • Minecraft data pack

Exporting a song as an MP3, WAV, AIFF, or FLAC file allows users to play it in any common media player. Exporting a song as a SCHEMATIC file or data pack allows users to import the song into a Minecraft world.

NOTE: Minecraft Note Block Studio can import .MID files. Minecraft users often use this feature to convert their favorite MIDI songs into Minecraft music.

How to open an NBS file

You can open an NBS file with Open Note Block Studio (Windows) or Minecraft Note Block Studio (Windows). Open Note Block Studio is actively maintained, while Minecraft Note Block Studio is not.

NTI Backup Set

Developer NTI
Popularity
2.7  |  9 Votes
 

Backup file created by NTI Backup Now, a backup and restore program for Windows; references a corresponding NTI Backup File (.NBF extension), which contains the backup data; used for saving and loading the backup set.

More Information

NBS files differ from .NPS files, which are hard drive partition backup sets. An NBS backup is called a "File-Based Backup" in NTI Backup Now, while an NPF backup is called a "Drive-Based Backup."
