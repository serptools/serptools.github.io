BlackMagic Custom Palette

Developer NeuralTek
Popularity
4.0  |  2 Votes
 

What is a BCP file?

Palette of custom colors used by BlackMagic, a Photoshop plug-in for colorizing black-and-white, sepia, and IR photos; stored by default in the palettes folder within the BlackMagic directory; can be loaded and saved using the Load and Record buttons in the BlackMagic Image window.

More Information

BlackMagic is developed by NeuralTek and is based on Timebrush RLC (Real Life Color) technology. The same technology is also used to colorize black and white movies.

Mario Kart Wii Binary Camera Parameter

Developer Nintendo
Popularity
1.0  |  1 Vote
 

A BCP file may also be a game data file used by Mario Kart Wii, a racing game for the Nintendo Wii. It controls how Mario Kart Wii's camera rotates around a character before a race or battle begins. Mario Kart Wii contains four BCP files, named battle_camera.bcp, mission_camera.bcp, race_camera.bcp, and start_camera.bcp.

More Information

Before a Mario Kart Wii race or battle begins, the game's camera rotates around and shows the player and their kart. BCP files are binary files that control these camera rotations. The files are stored in Mario Kart Wii's common.szs file.

Race_camera.bcp controls how the pre-race camera rotates for Grand Prix, VS Race, Time Trial, and most other races. Mission_camera.bcp controls how the camera rotates in widescreen Competitions. Battle_camera.bcp controls how the camera rotates in Battles. It is unknown what start_camera.bcp controls.

How to open a BCP file

Players who want to edit the hex values a BCP (Mario Kart Wii Binary Camera Parameter) file contains can do so using BCP Table (Windows), a player-created BCP file editor.
