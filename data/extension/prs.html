Harvard Graphics Presentation

Developer Harvard Graphics
Popularity
4.0  |  26 Votes
 

What is a PRS file?

Presentation or slide show created with Harvard Graphics presentation software; may include text, images, charts, graphs, and other media; typically created from a pre-built template.

More Information

The PRS file extension was used by Harvard Graphics for Windows versions 1-3. Current Harvard Graphics programs include "Pro Presentations," "Advanced Presentations," "Easy Presentations," and "Harvard Graphics 98."

PRS Archive

Developer N/A
Popularity
2.7  |  3 Votes
 

A PRS file may also be an archive used to store a SEGA Saturn game, such as Sonic Adventure, Sakura Taisem, or Dark Legend. It stores the contents of a game compressed with PRS compression, which is based on LZ77 compression.

More Information

You will most likely only encounter a PRS archive if you enjoy emulating old SEGA Saturn video games. You can extract the contents of a PRS archive using PRSDec Auto, which is a PRS decompression utility developed by Nemesis. This utility is especially useful for gamers looking to modify the files that make up the game to alter gameplay.

NOTE: PRS archives may also be used to store various PC games released in the 1990s.

Batch & Print Pro Printer Settings File

Developer Traction Software
Popularity
2.5  |  2 Votes
 

Stores custom printer settings saved within Batch & Print Pro, a print management program; includes print setup options, such as printer settings, paper size, and print tray selection.

More Information

PRS files are created by selecting "Save Printer Settings As..." from the File menu in Batch & Print Pro. They can be opened by selecting "Load Printer Settings..."
