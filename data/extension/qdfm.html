What is a QDFM file?

A QDFM file is a financial data file created by Intuit Quicken 2007, personal finance software for the Mac. It contains a user's financial data, including purchase history and bank accounts.

More Information

Intuit originally released Quicken in 1983 and has used various formats throughout different versions of Quicken for storing financial data. For example, versions 3 and 4 of Quicken used the .QDT format, versions 5 and 6 used the .QDB format, and versions 98 and later use the .QDF format.

While Windows versions of Quicken use the QDF format, Intuit developed the QDFM format for the Mac versions of Quicken. Then, Intuit replaced the format with .QUICKEN2017 after releasing Quicken 2017,

NOTE: Intuit sold Quicken to H.I.G. Capital in 2016, then H.I.G. sold Quicken to Aquiline Capital Partners in 2021. Quicken Inc. now develops Quicken.

Common QDFM Filenames

Quicken Data.qdfm - Default filename given to QDFM files.

How to open a QDFM file

You can only open a QDFM file with Quicken 2007 or later for Mac. To open the QDFM file, select File → New, then select "Start from a Quicken 2007 data file" on the "Getting Started" page.

How to convert a QDFM file

Quicken can convert QDFM files to various file types, including the following formats:

  • .CSV - Comma-Separated Values File
  • .QXF - Quicken Essentials for Mac Exchange File
  • .QIF - Quicken Interchange Format File
