SolidWorks Assembly

What is an SLDASM file?

An SLDASM file is a 3D assembly created by SolidWorks CAD software. It typically contains a model of an engine, vehicle, tool, or other manufactured item. SLDASM files are comprised of SolidWorks part (.SLDPRT) files, which users combine to form assemblies.

More Information

Product designers and engineers use Dassault Systemes SolidWorks to design 3D product prototypes. For example, an engineer might use SolidWorks to create many different prototypes of a new car engine. In SolidWorks, these prototypes are referred to as assemblies.

To create an assembly, SolidWorks users create or import parts, saved as SLDPRT files, and then combine them. After creating an assembly, users can save the assembly as an SLDASM file. They can also use SolidWorks tools to troubleshoot their assembly's construction and render the assembly as a 2D drawing (.SLDDRW file).

How to open an SLDASM file

You can open an SLDASM file in Dassault Systemes SolidWorks, Dassault Systemes SolidWorks eDrawings Viewer, or Autodesk Fusion 360. Note that if you open an SLDASM file in SolidWorks eDrawings Viewer, you can only view the file's contents, not edit them.

How to convert an SLDASM file

You can use Dassault Systemes SolidWorks or Autodesk Fusion 360 to convert SLDASM files to other formats. For example, SolidWorks can convert SLDASM files to the following formats (and many others):

  • .SLDPRT - SolidWorks Part
  • .DWG - AutoCAD Drawing
  • .STL - Stereolithography File
  • .STEP - STEP 3D Model
  • .IGS - IGES Drawing
  • .DXF - Drawing Exchange Format
  • .WRL - VRML World
  • .PSD - Adobe Photoshop Document
  • .PNG - Portable Network Graphic
  • .PDF - Portable Document Format
