Sisulizer Package File

Popularity

What is an SLE file?

Compressed package file saved by Sisulizer, a visual software localization tool; contains the exchange project file and other files needed by the translator; similar to a .SLP file, but is compressed using .ZIP compression.

Steganos Safe Encrypted Drive

Developer Steganos
Popularity
2.0  |  5 Votes
 

An SLE file contains data encrypted by Steganos Safe encryption software to securely store data. It contains a "Safe," which is a drive that stores data encrypted with 384-bit AES-XEX encryption. SLE files can only be decrypted by Steganos Safe software.

More Information

To create an SLE file with Steganos Safe, select Safe → Create..., then follow the prompts to specify the location, maximum file size, and password for the file. To adjust the settings of the Safe, select Safe → Settings....

To open an SLE file with Steganos Safe, click Safe → Import, then enter the password to open the file. After opening the file, the drive stored in the file opens and becomes accessible, allowing files to be added or removed. Once the SLE file is closed, the drive is encrypted again and no longer accessible.
