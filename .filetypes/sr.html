Sun Raster Graphic

Developer Sun Microsystems
Popularity
3.8  |  9 Votes
 

What is an SR file?

An SR file is a Sun Raster Graphic. It is saved in a proprietary image format used by SunOS (now Solaris) workstations. However, you can open an SR file in many common image viewers and editors.

More Information

While Sun Raster Graphics primarily use the .SUN and .RAS extensions, some are saved as SR files. You can identify a Sun Raster Graphic by opening the file in a hex editor and examining its header. A Sun Raster Graphic's header always includes the magic number 59a66a95.

SR files may contain full-color, paletted, or grayscale images, saved with an image depth of 1, 4, 8, 24, or 32 bits. The data an SR file contains may be compressed, using run-length encoding, or uncompressed.

How to open an SR file

You can open an SR file in many image viewers and editors. Some examples include:

  • qView (multiplatform)
  • XnViewMP (multiplatform)
  • ImageMagick (multiplatform)
  • GIMP (multiplatform)

How to convert an SR file

You can use ImageMagick or XnViewMP to convert SR files to other image formats. For example, XnViewMP can convert SR files to the following formats (and many others):

  • .JPG - JPEG Image
  • .GIF - Graphical Interchange Format
  • .PNG - Portable Network Graphic
  • .TIFF - Tagged Image File Format
  • .BMP - Bitmap Image
  • .PCX - Paintbrush Bitmap Image
  • .TGA - Targa Graphic
  • .JP2 - JPEG 2000 Core Image
  • .JXR - JPEG XR Image
  • .WEBP - WebP Image

Sega Genesis Collection Archive

Developer Sega Games
Popularity
1.3  |  3 Votes
 

An SR file may also be a compressed archive included in a Sega Genesis Collection video game. It contains files used by a game included in the collection. SR files are compressed using zlib compression.

More Information

Various Sega Genesis Collections include video games whose ROMs and assets are saved within SR archives. An archive that contains a game ROM uses the game's name as its filename, while archives that contain game assets use the game's name and a suffix (such as _FW or _FFE) as their filenames.

How to open an SR file

Because SR files are so uncommon, you may have trouble extracting the files they contain. However, some gamers have been able to extract ROMs and assets from SR files using offzip and QuickBMS.
