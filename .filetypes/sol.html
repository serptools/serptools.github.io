Flash Local Shared Object File

Developer Adobe Systems
Popularity
3.9  |  46 Votes
 

What is a SOL file?

A SOL file contains user-specific data and settings used by Macromedia Flash Player 6 and later versions, including Adobe Flash Player. It is similar to a website cookie but used specifically to store Flash Player-related data. For example, when a person plays a Flash-based game in a web browser, their game save information is often stored in a SOL file.

How to open a SOL file

You can open a SOL file in SolVE: the Local Shared Object Viewer/Editor (Windows, Mac), Sol Editor (Windows), or .minerva (web). .minerva is compatible with Google Chrome (version 38 or later) and Mozilla Firefox (version 32 or later).

Solidity Script

Developer Ethereum Foundation
Popularity
4.0  |  20 Votes
 

A SOL file may also be a script written in the Solidity scripting language, which is similar to C++ and JavaScript. It contains Solidity source code, which is used to create smart contracts for blockchain transactions. Solidity is utilized by several blockchain platforms, such as Ethereum, Tendermint, and Counterparty.

More Information

Smart contracts are executable programs built in the Solidity scripting language. They allow users to send value (money) to others using a digital currency, such as Ethereum. The value is called “Ether” and is similar to Bitcoin, another cryptocurrency. SOL files used to complete Ethereum transactions are processed by the Ethereum Virtual Machine (EVM). They record information such as the sender, receiver, and amount transferred. When the contract is completed, the transaction information is added to the blockchain. Smart contracts may be created for a variety of purposes, such as blind auctions, crowdfunding, voting, and multi-signature wallets.

SOL files are supported by Dapp, which is a Solidity package manager, build tool, and deployment assistant. They are also supported by various programs with the Solidity plugin installed, such as Visual Studio, Visual Studio Code, Vim, and Sublime Text.

NOTE: Smart contracts are also known as decentralized apps (dapps) on Ethereum.

Sierra On-Line Audio File

Developer Sierra Entertainment
Popularity
3.5  |  2 Votes
 
NEW: Open your .SOL file online with File Helper.View Online

A SOL file may also be an audio file used by one of a number of 1990s Sierra Entertainment video games. It contains music, sound effects, or dialogue that is played within a game. SOL files cannot be played by most popular media players, so if you want to play the files outside the game in which they are used, you must convert them to a standard audio format.

More Information

Prior to 2002, Sierra Entertainment was known as Sierra On-Line, which explains why SOL files are known as Sierra On-Line audio files and use the .sol extension. Sierra games that use SOL files include:

  • The Dagger of Amon Ra
  • The Island of Dr. Brain
  • King's Quest V - VII
  • Leisure Suit Larry 5 - 7
  • Mixed-Up Mother Goose
  • Pepper's Adventures in Time
  • Phantasmagoria
  • Space Quest IV - VI
  • Torin's Passage

Sierra Entertainment released all of these games between 1991 and 1996. In the 2000s, many of these games were re-released for purchase online via digital distribution platforms like GOG.com.

How to open a SOL file

Because SOL files are so uncommon, most popular media players cannot play them. However, you can convert SOL files to a usable audio format using FFmpeg and then play the converted file using a media player such as VLC media player.
