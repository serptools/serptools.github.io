CompeGPS Land Track File

Developer CompeGPS TEAM
Popularity
3.8  |  11 Votes
 

What is a TRK file?

File created by CompeGPS Land, a program used to to interface with CompeGPS GPS devices; contains a series of locations that are overlaid on a map; used to plan routes and load them into CompeGPS devices.

Garmin Track File

Developer Garmin
Popularity
2.3  |  3 Votes
 

File created by Garmin GPS devices, including the PCX5, PCX5AVD, and PC150 models; contains a series of locations, or route points, that are logged in real-time while a user navigates with a GPS unit; includes software version, coordinate system, latitude, longitude, date, and altitude.

More Information

TRK files are used to record traveling logs. They are stored in an ASCII text format.

Finale Track-to-Staff Mapping File

Developer MakeMusic
Popularity
3.0  |  1 Vote
 

A TRK file may also be a track-to-staff mapping file created by MakeMusic Finale, a popular music notation program. It contains staff configuration data that Finale references when you open a .MID (MIDI) file. Finale uses this data to translate the audio data your MID file contains into notation.

More Information

Composers use Finale to create, edit, play back, and share musical scores. One way composers can begin creating scores is by importing existing music saved in MID files.

To import a MID file, a composer:

  1. Selects File → Open...
  2. Navigates to and opens their MID file.

Finale then shows the Import MIDI File Options screen. On this screen, composers can select Set Track-to-Staff List. Doing so allows them to create or load a track-to-staff mapping file, which defines how Finale will transcribe the audio data the MID file contains. These mapping settings include the top-to-bottom order of the resulting Finale staves, staff transpositions, clefs, and the distance between staves.

How to open a TRK file

You can open a TRK file with MakeMusic Finale (Windows, Mac). To do so, you must first import a .MID file. Then, on the Import MIDI File Options screen, select Set Track-to-Staff List. When the Track/Channel Mapping to Staves window appears, click Load... to navigate to and open your TRK file.

Pacific Fighters Offline Track File

Developer Ubisoft Entertainment
Popularity
2.0  |  1 Vote
 

File created by Pacific Fighters, a flight simulator game; contains a recording of a flight mission in "offline" mode; created by clicking the Save Track button when a mission is complete; can be played back in Pacific Fighter.

More Information

Tracks recorded in "online" mode are stored in .NTRK files. Unlike NTRK files, TRK files only store initial positions and player inputs rather than all object positions during a mission.

NOTE: TRK files do not contain video data. They only store the initial positions and the player control inputs.
