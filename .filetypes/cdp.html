Trainz Simulator Content Dispatcher Pack

Developer N3V Games
Popularity
4.1  |  52 Votes
 

What is a CDP file?

File used by Trainz Simulator railroad simulation games; contains game add-on content, such as train routes, locomotive designs, and scenery; can be imported using the Content Dispatcher and Content Manager Plus tools within Trainz Simulator products; installed in the World\Dispatcher\Downloads folder of Trainz installations.

More Information

CDP files are developed by third party content providers, such as Auran, and are not part of the original game content. Several tools exist for managing CDP packages.

Sony CD Architect Project

Developer Sony
Popularity
3.3  |  4 Votes
 

A CDP file may also be a project created by Sony CD Architect, an audio CD mastering and burning program. It contains project information, such as the location of source files, cuts, insertion points, edits, and project properties. CDP files do not store the actual audio, only the information necessary for creating the CD after the editing is completed.

More Information

CDP files allow you to make edits to your CD without modifying the actual source files. This means that you can experiment with the settings and edits and not have to worry about altering the audio.

When you first run CD Architect, a new project is automatically created but it is not until you choose to save the project that an actual CDP file is created to store the contents of the project. Choose File → Save to save the project.

CDP files are only supported by CD Architect. To open a CDP file, select File → Open..

A CDP project is automatically saved every 5 minutes by CD Architect in the .AUTOSAVE.CDP file. However, the file is deleted when CD Architect is closed. You can turn off the autosave function by unchecking the Enable autosave checkbox.

NOTE: Sony CD Architect is discontinued.

ConceptDraw PROJECT File

Developer CS Odessa
Popularity
2.3  |  6 Votes
 

A CDP file may also be a project file created by ConceptDraw PROJECT, a project management application. It stores project schedules, resources, priorities, and tracking information. CS Odessa replaced CDP files with .CDPZ files and .CDPX files (saved in the XML standard).

More Information

ConceptDraw PROJECT is a project management tool that provides various features for planning and implementing projects, such as building renovations or mobile app development. While ConceptDraw PROJECT no longer supports CDP files, users of older versions used CDP projects for displaying project "dashboards," which summarize vital performance indicators.

How to open a CDP file

You can open CDP files with older versions of CS Odessa ConceptDraw PROJECT, such as version 5. However, newer versions of ConceptDraw PROJECT, such as version 13, no longer support the format.
