Flash Lite Sound Bundle

Developer Adobe Systems
Popularity
3.4  |  14 Votes
 

What is an FLS file?

Collection of sound files stored in a single bundle; used by Flash Lite, a version of the Flash development platform designed for mobile devices; may include multiple sound files, but only supports one sound file per file format.

More Information

FLS files are created using Flash Lite Sound Bundler (FlashLiteBundler.exe), which is only available for Windows systems. However, they can be imported into Flash documents using Adobe Flash, which is available for both Mac and Windows.

Lexmark Firmware Update

Developer Lexmark
Popularity
2.0  |  1 Vote
 

An FLS file may also be a firmware update file used by Lexmark printers and multifunction products (MFPs). It contains data that fixes a firmware bug, allows a device to operate more efficiently, or adds compatibility with additional devices. Typically, users install an FLS file by placing it on a USB drive, connecting the drive to their printer or MFP, and then opening the file on their printer or MFP.

More Information

Lexmark is a printing and imaging company that specializes in providing printers and MFP (which can print, scan, and copy) to businesses. Periodically, Lexmark releases firmware updates for printers and MFPs, which are saved as FLS files.

Businesses that use Lexmark printers and MFPs can download FLS files from Lexmark's Drivers & Downloads webpage. While businesses that use a small number of Lexmark devices typically install FLS files using a USB drive, large businesses that have many Lexmark printers often use Markvision Enterprise, a Lexmark printer management application, to deploy FLS files to all their applicable printers or MFPs.

How to open an FLS file

For instructions on how to install an FLS file on your Lexmark printer or MFS, refer to your device's documentation or the applicable page on Lexmark's support website.
