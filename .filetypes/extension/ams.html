Velvet Studio Advanced Module System

Developer Velvet Development
Popularity
3.2  |  5 Votes
 

What is an AMS file?

An AMS file is an Advanced Module System used by Velvet Development Velvet Studio, a music tracker developed for MS-DOS and Microsoft Windows. It contains a song created in Velvet Studio. AMS files are comprised of audio samples (.ASE files) and instruments (.AIS files) used to play those audio samples.

More Information

In 1996, developers Patrik Oscarsson and David Broman released the first version of Velvet Studio (previously known as Extreme Tracker). This music tracker allows users to create synthesized music, by keying in notes or importing audio samples and playing them using one or more synthesized instruments.

Songs created using Velvet Studio are saved as AMS files. Often, users stored their AMS files in the C:/​SOUND/​MODULES/​AMS directory. While Velvet Studio is now legacy software, users can still open their AMS files in Velvet Studio and some other, current music trackers.

How to open an AMS file

You can open an AMS file with:

  • Velvet Development Velvet Studio (Windows)
  • OpenMPT (Windows)
  • Vitamin/CAIG ZXTune (multiplatform)

How to convert an AMS file

OpenMPT can convert the song an AMS file contains to many other formats, including:

  • .WAV - WAVE Audio
  • .FLAC - Free Lossless Audio Codec
  • .AU - Sun Audio
  • .PCM - Pulse Code Modulation
  • .OPUS - Opus Audio
  • .OGG - Ogg Vorbis Audio
  • .MP3 - MP3 Audio
  • .MID - MIDI Audio
  • .IT - Impulse Tracker Module

Extreme Tracker Module

Developer Patrik Oscarsson and David Broman
Popularity
2.2  |  6 Votes
 

An AMS file may also be a module created by Extreme Tracker, an early version of the Velvet Studio music tracker. It contains a song that a user created in Extreme Tracker. This song consists of notes played using one or more synthesized instruments.

More Information

In 1994, Patrik Oscarsson and David Broman began work on the music tracker that would become Velvet Studio. In its early iterations, this tracker was called Extreme Tracker.

Just like Velvet Studio, Extreme Tracker saved the songs that users created as AMS files. Because Extreme Tracker and Velvet Studio AMS files are saved in roughly the same file format, many music trackers and players that can open Velvet Studio AMS files can also open Extreme Tracker AMS files.

NOTE: Extreme Tracker was also known as Extreme's Tracker.

How to open an AMS file

You can open an AMS file with OpenMPT (Windows) or Vitamin/CAIG ZXTune (multiplatform).

How to convert an AMS file

Using OpenMPT, you can convert the song an AMS file contains to many other formats, including:

  • .WAV - WAVE Audio
  • .FLAC - Free Lossless Audio Codec
  • .AU - Sun Audio
  • .PCM - Pulse Code Modulation
  • .OPUS - Opus Audio
  • .OGG - Ogg Vorbis Audio
  • .MP3 - MP3 Audio
  • .MID - MIDI Audio
  • .IT - Impulse Tracker Module
