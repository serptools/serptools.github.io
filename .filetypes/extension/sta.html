Adobe Photoshop Match Color Image Statistics File

Developer Adobe Systems
Popularity
4.2  |  6 Votes
 

What is an STA file?

Data file used by Adobe Photoshop, an application used primarily for editing raster graphics; stores values used to compute a color match and adjust an image; can be used for matching colors between multiple images, layers, or selections.

ABAQUS Status File

Developer ABAQUS
Popularity
3.1  |  9 Votes
 

Stores the status of increment summaries written by ABAQUS using the analysis option; ABAQUS is data analysis program used for finite element analysis (FEA); examples of uses for ABAQUS include static and dynamic stress calculation, heat transfer and thermal stress, mass diffusion, and fracture mechanics analysis.

More Information

ABAQUS is available in the following versions:

  1. ABAQUS/Standard - for traditional implicit finite element analyses
  2. ABAQUS/Explicit - for transient dynamics and quasi-static analyses
  3. ABAQUS/CAE - for modeling and visualizing environments

MAME Saved State File

Developer The MAME Team
Popularity
3.5  |  4 Votes
 

File created by a MAME arcade video game emulation program; contains a "frozen" state of the emulated arcade machine at a particular time during gameplay; used for saving and loading games.

More Information

Saved state files differ from in-game saved games, which only stores the current in-game progress, not the entire state of the game running on the machine.

Vectorworks Design Template

Developer Nemetschek
Popularity
3.5  |  2 Votes
 

An STA file may also be a template of a building, interior, or landscape design created by Nemetschek Vectorworks, a CAD design suite. It may contain a 3D model and 2D renderings of the model, along with layer information and project metadata. STA templates are similar to .VWX design files, but Vectorworks users save designs as templates when replicating designs with similar content and appearance.

More Information

Various professional users, including architects, landscape designers, and live event designers, use Vectorworks software to create 3D models and 2D drawings of buildings, landscapes, and live event setups. The Vectorworks suite consists of the following programs (known collectively as the Vectorworks Design Suite):

  • Architect: Used to model and draw building projects.
  • Landmark: Used to model and draw landscape projects.
  • Spotlight: Used to model and draw live event setups (staging, lighting, interior design, etc.).
  • Fundamentals: Used to create CAD drawings and models for design and fabrication.

These programs are also available as standalone apps. Additionally, they all save templates as STA files (File → Save As Template).

NOTE: Saving an STA file in the "Templates" folder in your user folder allows you to access the template when you select File → New → Use Document Template.

How to open an STA file

You can open STA templates with various Nemetschek Vectorworks apps. If you only want to view your STA template, you can open it with Nemetschek Vectorworks Viewer. This free VWX viewer is available for Windows and macOS.

If you need to edit or convert your STA file, you can open it with the appropriate Vectorworks program. For example, if you have an STA file created by Vectorworks Landmark, you can open that file with Vectorworks Landmark (select File → Open).

How to convert an STA file

Vectorworks design programs can convert STA files to various file types, including the following formats:

  • .VWX - Vectorworks Design
  • .DXF - Drawing Exchange Format
  • .DWG - AutoCAD Drawing
  • .FBX - Autodesk FBX Interchange File
  • .C4D - Cinema 4D Model
  • .OBJ - Wavefront 3D Object
  • .STP - STEP 3D Model
  • .STL - Stereolithography File
  • .KML - Keyhole Markup Language
  • .JPG - JPEG Image
  • .PDF - Portable Document Format
