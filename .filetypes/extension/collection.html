Adobe Bridge Collection File

Developer Adobe Systems
Popularity
2.9  |  8 Votes
 

What is a COLLECTION file?

A COLLECTION file contains references to one or more files organized in Adobe Bridge, a file management application for Adobe Creative Cloud (CC) applications. Bridge users create a collection to sort digital assets, such as images and videos, without moving the files from their actual locations on the computer.

More Information

Bridge integrates with other Adobe applications, such as Photoshop and After Effects to help users organize their many digital assets. One of Bridge's features is the ability to place assets in collections, making them easier to sort and find.

How to create a COLLECTION file

To create a collection, open the Collections panel by selecting Window and checking the Collections panel. Then, select the file(s) you want to add to the collection and click the "New Smart Collection" icon in the lower-right corner of the Collections panel. After creating the collection, you can add or remove files to and from it.

The COLLECTION file is automatically saved in the following directory based on the computer platform:

  • Windows - [user]\​Application Data\​Adobe\​Bridge\​Collections\​
  • macOS - [user]/​Library/​Application Support/​Adobe/​Bridge/​Collections/​

How to open a COLLECTION file

You can open a COLLECTION file with Adobe Bridge (multiplatform) by double-clicking the file (Bridge must be installed on your computer).

You can also place the COLLECTION file in the "Collections" folder in the following directory:

  • Windows - [user]\​Application Data\​Adobe\​Bridge\​
  • macOS - [user]/​Library/​Application Support/​Adobe/​Bridge/​

Then, display the Collections panel by selecting it from the Window dropdown list and click the name of the collection in the panel.
