What is a WORLDPACK file?

A WORLDPACK file is a custom world pack created and used by the video game Cats Are Liquid: A Better Place. It typically contains several A Better Place worlds, which are comprised of one or more rooms. Players create WORLDPACK files using A Better Place’s level editor.

More Information

Cats Are Liquid: A Better Place is a 2D platformer in which players play as a cat who goes on an adventure through several worlds. Players can use A Better Place’s level editor to create their own worlds and rooms, for custom gameplay.

Each set of A Better Place worlds and rooms is saved in a WORLDPACK file. Typically, the worlds a WORLDPACK file contains are related to each other somehow (for example, they’re used to tell an A Better Place side story).

After a player finishes creating a world pack, they can use the level editor to share that WORLDPACK file with others, by emailing it, messaging it, or uploading it to the Internet. WORLDPACK files are cross-platform compatible. For example, a world pack created in Android will work with the Windows version of Cats Are Liquid: A Better Place.

How to open a WORLDPACK file

You can open a WORLDPACK file with Cats Are Liquid: A Better Place (multiplatform) to import the worlds it contains. If you are playing A Better Place on an Android or iOS device, you can tap the file to import the worlds it contains. If you are playing A Better Place on PC, you must place your WORLDPACK file in the appropriate game directory (often the maps directory) to use it.

You can also open a WORLDPACK file with Microsoft Notepad (Windows), Apple TextEdit (Mac), or any other text editor, if you want to see the plain text data it contains. Keep in mind, however, that editing a WORLDPACK file may cause it to function incorrectly.
