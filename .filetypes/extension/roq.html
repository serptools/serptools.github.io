.ROQ File Extension

RoQ Full Motion Video

Developer id Software
Popularity
3.0  |  14 Votes
 

What is a ROQ file?

A ROQ file is a video saved in the RoQ full motion video format. It likely contains a cinematic played in a video game. Many games developed using id Software's id Tech 3 and 4 engines, including Quake III and Doom 3, use ROQ files.

More Information

Developer Graeme Devine created the RoQ format for the video game The 11th Hour. When Devine later joined id Software, he brought the RoQ format with him. Thus many id Software games, and other games developed using id Software's id Tech 3 and id Tech 4 engines, include cinematics and other videos saved as ROQ files.

All ROQ files have a framerate of 30 frames per second and may include a 22050 Hz mono or stereo audio track. While RoQ videos can have a resolution of up to 65,520 x 65,520 pixels, they typically have a resolution of 512 x 512 or less, as larger videos can cause issues when played back in games.

Games that use ROQ files include:

  • The 11th Hour
  • American McGee's Alice
  • Clandestiny
  • Doom 3
  • Quake III
  • Return to Castle Wolfenstein
  • Star Wars: Jedi Knight II - Jedi Outcast

How to open a ROQ file

You can open a ROQ file with VideoLAN VLC media player, to play the video it contains. Advanced users can also open a ROQ file with FFmpeg, for more extensive playback, editing, and conversion options.

How to convert a ROQ file

You can use VideoLAN VLC media player to convert ROQ files to the following video formats:

  • .WEBM - WebM Video
  • .TS - Video Transport Stream
  • .ASF - Advanced Systems Format

Advanced users can use FFmpeg to convert ROQ files to additional formats.

How to create a ROQ file

Those looking to convert a video to the RoQ format can do so with Quake Video Maker. However, the to-be converted video must be saved in the .AVI format. (If your video is not saved in the AVI format, you can use FFmpeg to convert it.) For complete instructions, click here.
