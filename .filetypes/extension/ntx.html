theWord Encrypted New Testament Text Module

Developer theWord Bible Software
Popularity
3.4  |  8 Votes
 

What is an NTX file?

File used by theWord Bible software; contains a copy of the New Testament and incorporates encryption, unlike the plain text formatting used by theWord's .NT format; helps distribution by protecting digital copies of the New Testament.

More Information

NTX files are often downloaded as ".ntx.exe" files that install the New Testament copy when opened.

Clipper Index File

Developer Clipper

KA3D N-Gage Texture

Developer Rovio Entertainment
Popularity
1.0  |  1 Vote
 

An NTX file may also be an image used in a video game developed using Rovio Entertainment's KA3D engine. It contains a texture that is applied to an in-game item. Games developed for the Nokia N-Gage, such as Bounce, Bounce Boing Voyage, and Bounce Touch, use NTX files.

More Information

The Nokia N-Gage is a discontinued mobile device that combined the features of a mobile phone and a handheld video game console. It was first released in 2003 and discontinued in 2006.

Rovio Entertainment, the publisher of Angry Birds, developed several games for the N-Gage, including the Bounce series. These games were developed using Rovio's proprietary KA3D engine. Textures created for these games were saved as NTX files. Most gamers will never encounter NTX files, except those who attempt to explore or modify Rovio N-Gage games' contents.

How to open an NTX file

There is no known way to open NTX files outside the games they are used in.
