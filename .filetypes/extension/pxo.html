Pixelorama Project

Developer Orama Interactive
Popularity
2.3  |  4 Votes
 

What is a PXO file?

A PXO file is a project created by Orama Interactive Pixelorama, an open-source sprite editor. It contains project information, such as pencil strokes, lines, shapes, gradients, and shading that comprise the graphic, along with metadata, such as the canvas size and creator. Users typically open and edit PXO files with Pixelorama and export them as .PNG images.

More Information

Pixelorama allows you to create sprites, tiles, and animations for expressive purposes or as graphical resources in a game or app. The software is available for Windows, macOS, and Linux and can run in a web browser.

When you save your project to preserve your editing progress, Pixelorama saves it as a PXO file by default. Once saved, you can close and re-open the project to continue editing.

NOTE: Pixelorama compresses PXO files with ZSTD compression if the creator chooses the "Use ZSTD Compression" option when saving the PXO project.

How to open a PXO file

You can open a PXO file with Orama Interactive Pixelorama in Windows, macOS, Linux, and a web browser. To do so, select File → Open.

How to convert a PXO file

Pixelorama can convert PXO files to the following formats when you select File → Export:

  • .PNG - Portable Network Graphic
  • .GIF - Graphical Interchange Format
  • .APNG - Animated Portable Network Graphic

LG Mobile Phone Executable File

Developer LG Electronics
Popularity
3.0  |  2 Votes
 

A PXO file may also be an executable file that can be run on one of several old LG mobile phones. It may contain a game or other program, as well as the program's constituent files.

More Information

Several models of LG mobile phones, including the GS290, BL40, GD510, and GM360, use executables saved as PXO files. These files contain programs that can be run only on LG phones. For example, the game Music World, which was available only on old LG phones, was saved as a PXO file.

How to open a PXO file

If you can transfer your PXO file to an LG mobile phone that supports the PXO format, you may be able to run the program the file contains. Those interested in decompressing a PXO file and examining its contents can use LGUI (Windows) to do so.
