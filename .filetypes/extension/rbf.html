LEGO MINDSTORMS EV3 Robot Brick File

Developer The LEGO Group
Popularity
4.0  |  8 Votes
 

What is an RBF file?

An RBF file is an executable file used by the EV3 Intelligent Brick, which is a device used to control LEGO MINDSTORMS EV3 robots. It stores an EV3 Basic program, which is used to make a robot perform a function. RBF files are compiled by the built-in compiler in EV3 Explorer and executed directly on the EV3 Intelligent Brick.

More Information

EV3 Basic programs are used to make LEGO MINDSTORMS EV3 robots perform a variety of tasks, whether it be moving a certain direction or picking up an object. These programs can be run in PC mode and brick mode. In PC mode, the programs runs in Microsoft Small Basic, while relaying information to the EV3 Intelligent Brick in order to operate the robot.

However, in brick mode, the programs run on the EV3 Intelligent Brick, which is a portable device that features a Linux-based operating system and is used to interface with the robots. RBF files are used to run programs on the EV3 Intelligent Brick.

RBF files are compiled from .SB files, which are files that store Small Basic program code, by EV3 Explorer, which is a Windows program used to view and organize files stored on the EV3 Intelligent Brick. EV3 Explorer can also be used to download RBF, .RSF, and .RGF files to the EV3 Intelligent Brick.

Windows Installer Rollback File

Developer Microsoft
Popularity
3.4  |  14 Votes
 

Archive of files created during a software installation process run by Windows Installer; contains files that can be restored if the installation does not successfully complete; if the installation is successful, the rollback file is automatically deleted.

More Information

The RBF file is saved along with a corresponding .RBS script file and stored in an archive named "Config.Msi."

Microsoft Windows Installer
Included with OS

Stellar Insta Backup Image

Developer Stellar Data Recovery
Popularity
3.0  |  3 Votes
 

Backup image created by Insta Backup Gold, a Windows program used to backup and restore files; contains the backup data of a file, folder, volume, or email database.

More Information

There are a number of settings that Insta Backup Gold provides for creating an RBF backup file. You can protect your backup with a password, select the compression level of your backup, and schedule backups to be created at a later time and at a custom frequency. Also, you can choose if you want a full, incremental, or a differential backup.

NOTE: Stellar Insta Backup Gold has been discontinued.

Redatam Binary File

Developer CELADE
Popularity
2.5  |  4 Votes
 

File created by Retadam (Retrieval of Data for Small Areas by Microcomputer), a mapping software used for analyzing local and regional census and survey data; saves data as structured records in a compressed binary format; used for analyzing geographical areas from the size of a few city blocks to a whole country.

More Information

RBF files may store census data such as gender, age, and household information. The data is stored in the form of "entities" (a logical hierarchy of objects) and "variables" (information on entities), which are further defined by the Retadam documentation.

NOTE: The Retadam software is supported by the Economic Commission for Latin America and the Caribbean (CELADE).
