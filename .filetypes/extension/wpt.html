WordPerfect Template

Developer Corel
Popularity
3.3  |  10 Votes
 

What is a WPT file?

A WPT file is a text document template created by Corel WordPerfect, a word processor for Windows. It contains a pre-formatted document that may include text, drawn objects, charts, tables, and images. The primary purpose of WPT files is to allow users to quickly create multiple WordPerfect (.WPD) documents with similar content and appearance.

More Information

WordPerfect is available with the Corel WordPerfect office suite, which is similar to Microsoft Office. You can use the WordPerfect word processor in home, academic, and business environments to author various documents, such as reports, letters, invitations, resumes, flyers, and newsletters.

While the proprietary Corel WPD document format is the main file type associated with WordPerfect, WPT templates are a helpful format when you want to re-use a document to create other documents with a similar layout or content. Examples of commonly used templates include company letterheads, business memos, legal forms, and mailing envelopes.

How to open a WPT file

You can open WPT files with Corel WordPerfect in Windows. The application is included with the Corel WordPerfect Office suite, along with the Quattro Pro and Presentations programs.

To open a WPT file with WordPerfect, select File → Open.... You can also select File → New Template... to open WPT files packaged with WordPerfect.

Kingsoft Writer Template

Developer Kingsoft
Popularity
2.8  |  6 Votes
 

A WPT file may also be a document template created by Kingsoft Writer, a word processor that is part of the Kingsoft WPS Office suite. It stores the page layout formatting and styles for a document and can be used for creating multiple Kingsoft Writer documents (.WPS files) with the same layout and text.

More Information

Writer comes bundled with several WPT templates and allows you to download many more directly through the application. They can be used for resumes, letterheads, weekly planners, newsletters, job descriptions, certificates, and other documents that are often replicated. WPT templates can also be converted to other spreadsheet files such as .DOT, .DOTX, .DOC and .DOCX files.

The WPS Office suite is similar to the Microsoft Office suite but is available for free. WPS Office offers four primary tools that are accessible in one single module:

  • Writer - Word processor similar to Microsoft Word
  • Spreadsheets - Spreadsheet program similar to Microsoft Excel
  • Presentation - Presentation program similar to Microsoft PowerPoint
  • PDF Editor - Create a PDF file from scratch or from an existing file or open, edit, save, compress, or convert a PDF file

Writer features rich text editing and document formatting capabilities and comes with several tools including spell check, word count, thesaurus, and add-ins. Besides the WPS and WPT files, Writer also supports DOCX, DOC, DOTX, RTF, and .PDF files.

Works Template

Popularity

Microsoft Works includes many pre-built templates that users can select from in the program. Users can also make custom templates and store them in WPT files.

Cube 2 Waypoint File

Developer Wouter van Oortmerssen
Popularity
2.0  |  4 Votes
 

Game data file used by games developed with the Cube 2 first-person shooter (FPS) engine; stores locations, or waypoints, within a game map; used for marking safe locations and pathways that artificial intelligence players, or "bots," can move to.

More Information

Waypoints can be linked in series with other waypoints so that bots follow pre-defined paths within a map. To create a WPT file, users can enter the Cube 2: Sauerbraten waypoint mode and walk through a game map, marking locations as they move.

NOTE: Cube 2 is a free cross-platform game engine that was released in 2004.
