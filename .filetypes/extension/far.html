The Sims Archive File

Developer Electronic Arts
Popularity
3.4  |  17 Votes
 

What is a FAR file?

A FAR file is an archive that contains game data used by The Sims, a life-simulation game. It stores a collection of objects used within the game. FAR files are referenced by The Sims during gameplay and can be unpacked and edited with various third-party programs.

More Information

Examples of FAR files used by The Sims include Animation.far, Textures.far, and Objects.far. The Sims also uses .IFF files to store object data.

FARO Zone File

Developer FARO Technologies
Popularity
3.8  |  6 Votes
 

A FAR file may also be a data file created by FARO Zone 2D or FARO Zone 3D, which are forensics investigation and security programs used to recreate a scene with animations and diagrams. It may contain 2D or 3D diagrams, models, vehicle crash simulations, bullet trajectories, blood spattering analyses, and other measurement data.

More Information

FAR files are primarily used for forensic investigations to recreate scenes that a user can fly or walk through and gain insight about an incident. They are also used for security planning purposes.

A FAR file can be viewed and edited using FARO Zone 2D and FARO Zone 3D. The file can also be viewed using FARO Zone Viewer, which is a free program that only allows users to view FAR files.

Farandole Composer Module

Developer Farandole
Popularity
3.3  |  3 Votes
 

A FAR file may also be an audio module created by Farandole Composer, a music tracker for MS-DOS. It contains up to 16 channels of synthesized audio. FAR files most often contain music meant to be played in video games and computer demos.

More Information

Farandole Composer was a DOS program that allowed users to create synthesized music. It was released in 1994 and used primarily by those interested in creating computer demos (intricate multimedia presentations, which combine animations, visual effects, and audio).

The music that users created in Composer was saved as FAR files. Each FAR file contains a record of a song's channels, notes, instruments, and other data.

How to open a FAR file

You can open a FAR file and play the music it contains with VideoLAN VLC media player (multiplatform), Schism Tracker (multiplatform), or Vitamin/CAIG ZXTune (multiplatform).

How to convert a FAR file

You can use VideoLAN VLC media player (multiplatform) to convert FAR files to the following formats:

  • .MP3 - MP3 Audio
  • .OGG - Ogg Vorbis Audio
  • .FLAC - Free Lossless Audio Codec
  • .WAV - WAVE Audio

Fuchsia Archive

Developer Google
Popularity
4.0  |  2 Votes
 

A FAR file may also be an archive used by the Google Fuchsia operating system. It contains a directory tree and the files referenced in that tree. FAR files can be opened only by Google Fuchsia.

More Information

Google Fuchsia is an open-source operating system developed by Google. Unlike Google's Chrome OS and Android operating systems, Fuchsia is not based on the Linux kernel. Instead, Fuchsia uses a unique kernel named Zircon.

Fuchsia can read files from archives saved in the Fuchsia archive (FAR) format. These archives use the .far file extension. Most notably, FAR files may contain and be used to run Android apps on Fuchsia, using the Android Runtime for Fuchsia. In this way, FAR files can be used similarly to Android's .APK files.

What chunks does a FAR archive contain?

Each FAR archive contains the following chunks of data:

     
  • Index chunk: Lists the other chunks the archive contains
  • Directory chunk: Contains entries that represent the archive's files
  • Directory names chunk: Used by the directory chunk to name the content chunks
  • Content chunks: The files the archive contains

How to open a FAR file

FAR files can be opened by the Google Fuchsia operating system. You can run Fuchsia on Linux and macOS PCs using a Fuchsia emulator. For more instructions, click one of the Google Fuchsia links found below.
