S Memo File

Developer Samsung
Popularity
3.5  |  2 Votes
 

What is an SFM file?

An SFM file is a memo created using Samsung S Memo, a memo application installed on some Samsung smartphones. It may contain text, images, and/or audio. S Memo has been discontinued and replaced by Samsung Notes.

More Information

Samsung S Memo is an application that some Samsung smartphone users use to create memos. S Memo allows users to type memos using the keypad, draw memos using the Samsung S Pen, and embed recorded audio. The memos users create in S Memo are saved as SFM files.

In 2018, Samsung discontinued S Memo in favor of Samsung Notes. As of 2021, Samsung Notes saves users' notes as .SDOCX files. However, users who backed up their SFM files using their Samsung account can restore those memos in Samsung Notes.

How to open an SFM file

You can open an SFM file in Samsung S Memo (Android). If you no longer have access to Samsung S Memo, but you do have access to Samsung Notes (Android), you may be able to restore your SFM memos in Samsung Notes, provided you are using the same Samsung account you used to create and back up your SFM memos.

To attempt to restore your SFM memos in Samsung notes:

  1. Open Samsung Notes.
  2. Open Notes' menu and select Settings → Import memo data from Samsung account.
