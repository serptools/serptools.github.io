Street Atlas USA Map File

Developer DeLorme
Popularity
3.9  |  41 Votes
 

What is a SAF file?

An SAF file is a map created by DeLorme Street Atlas USA, a custom mapping and GPS navigation program for Windows. It contains a detailed digital map of a specific area, along with associated data such as the map center coordinates, zoom level, and user-defined settings. It may also include custom routes, points of interest, and user-added layers for planning or navigation purposes.

More Information

​DeLorme released the first version of Street Atlas USA in 1991. Over the years, DeLorme has updated and enhanced Street Atlas USA, incorporating various features, including GPS navigation and route planning. The software remained popular among travelers, businesses, and mapping enthusiasts until its discontinuation following Garmin's acquisition of DeLorme in 2016.

SAF files saved by Street Atlas USA allowed users to reopen their saved maps with all their preferences and additions intact. The software automatically saved the files in the following directory:

C:\​DeLorme Docs\​Map Files

How to open a SAF file

You can open SAF files with DeLorme Street Atlas USA in Windows. However, the software is discontinued.

Secure Audio File

Developer InterTrust Technologies
Popularity
4.0  |  10 Votes
 

An SAF file is an audio file that includes digital rights management (DRM) information in a format developed by InterTrust Technologies, which restricts how the file can be used or shared. It contains compressed audio and embedded DRM rules, such as limits on playback devices or restrictions on burning the file to a CD. While SAF files are now obscure, they were commonly used in the early 2000s for downloading purchased songs through platforms like MusicMatch Jukebox, a popular music player and manager for Windows.

More Information

InterTrust Technologies developed the SAF format for securely distributing copyrighted music and preventing unauthorized copying. Because of these protections, SAF files usually require specific software, such as MusicMatch Jukebox and valid user credentials, to be opened or played. As the music industry shifted to DRM-free formats and streaming services, the use of the SAF format declined and is now largely obsolete.

How to open a SAF file

You can open SAF files with MusicMatch Jukebox in Windows. However, the software is discontinued.

SafeText Document

Developer SafeText
Popularity
3.4  |  14 Votes
 

An SAF file is a text document encrypted and password-protected for security by the now-discontinued Safe Text program. It stores a document with sensitive contents a user has secured, such as medical records, financial numbers, or a legal contract.

More Information

SafeText was an application in the 1990s designed to create encrypted, password-protected text files so that if a person gains access to the file, they won’t be able to view the information without the correct password. However, newer and more advanced encryption tools and methods developed, leading SafeText to be discontinued in the early 2000s and the SAF format to become obsolete.

How to open a SAF file

The only known program that can open SAF files is SafeText in Windows. The program is discontinued.

McAfee Fortress File

Developer McAfee
Popularity
3.4  |  12 Votes
 

An SAF file is a secure archive created by McAfee Fortress, a discontinued data protection program developed by McAfee. It stores one or more files in an encrypted and password-protected container. The purpose of the SAF format is to prevent unauthorized access to sensitive data by requiring the correct password and software to open the secured file.

More Information

McAfee developed Fortress in the late 1990s and early 2000s. It was designed primarily for home and small business users looking to protect sensitive files and folders on personal computers through password protection and encryption.

The software created secure containers (SAF files) that could only be opened with the correct credentials within the Fortress environment. However, McAfee discontinued the software in the early 2000s, and the SAF file format is now largely obsolete.

How to open a SAF file

You can open SAF files with McAfee Fortress in Windows. However, the software is discontinued.

Cricket Shot Animation File

Developer EA Sports
Popularity
1.0  |  1 Vote
 

An SAF file is a game data file used by EA Sports Cricket 07 or another EA Sports Cricket video game. It contains plain text information that defines how the game animates a cricket shot. Some modders edit and create custom copies of Cricket's SAF files while modding the game.

More Information

Cricket 07 is the last game in EA Sports's Cricket series of games. For this reason, it remains popular among cricket enthusiasts. Cricket 07 players have produced many mods and patches for the game, some of which depend on the game's SAF files.

For example, a modder who is trying to add a new shot to Cricket 07 may need to copy an existing SAF file and associate it with the shot, so the shot has an in-game animation. You can find Cricket players discussing SAF files and Cricket mods and patches in the forums on PlanetCricket.org.

How to open a SAF file

You can use BigGUI (Windows) to extract SAF files from EA Sports Cricket video games. You can then open and edit the files with any text editor, such as Microsoft Notepad (Windows) or Apple TextEdit (Mac).
