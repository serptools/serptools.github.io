MrSID World File

Developer LizardTech
Popularity
3.1  |  14 Votes
 

What is an SDW file?

GIS file created for MrSID .SID raster images; contains georeferencing information that maps the raster image data in the SID file to real-world coordinates; contains the units of measurement for the image pixels as well as the coordinates of the upper-left hand pixel.

More Information

SDW files are saved in a plain text format and may be viewed with a text editor.

StarOffice Writer Text Document

Developer N/A
Popularity
2.4  |  5 Votes
 

Text document created with StarOffice Writer 5.0 or earlier (also known as "StarWriter"); the word processor included with Sun Microsystem's StarOffice suite; may also be opened with the Apache OpenOffice Writer program.

More Information

StarWriter documents created with StarOffice 6 or later use the .SXW extension.

NOTE: Apache OpenOffice was formerly known as OpenOffice.org.
