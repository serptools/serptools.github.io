.RWD File Extension

2 file types use the .rwd file extension.
  • 1.NRG Raw Wind Data File
  • 2.Microsoft Flight Simulator Rewards File

NRG Raw Wind Data File

Developer NRG Systems
Popularity
4.3  |  9 Votes
 

What is an RWD file?

Data file created by NRG Systems' Symphonie data loggers, which capture wind speed and direction information; can be uploaded to a PC and processed using Symphonie Data Retriever, a free program provided by NRG Systems.

Microsoft Flight Simulator Rewards File

Developer Microsoft
Popularity
3.0  |  3 Votes
 

Game file created by Microsoft Flight Simulator, a series of flight simulator programs; contains rewards data from completing a certain achievements in the game, such as completing a first solo flight or earning the best course time.

More Information

Many different rewards are available, including certificates, trophies, medals, badges, special items, and post cards.

NOTE: RWD files are saved to the \​Rewards\​ directory of the game installation.
